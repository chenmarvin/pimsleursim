# 🇯🇵 Japanese for Taiwanese Learners — JLPT N5, Lesson 8

**Theme:** 「どんな人ですか」(What kind of person/thing is it?)

**Objectives**
- Describe people and things with i-adjectives and na-adjectives
- Talk about the weather in past tense

> Note: This lesson formally introduces 〜ます／〜ません, the polite present-tense verb forms (some ます-form verbs already appeared as vocabulary chunks in Lesson 6, but this is the first time the conjugation rule itself is explained). The weather words 暑い／寒い also let us preview a simple past-tense adjective sentence like 昨日は寒かったです — the full past-tense adjective conjugation rule (〜かった) is taught formally in a later lesson, so treat it here as a recognized set phrase, not yet a pattern to produce freely.

### Word-by-word details

**1. 大きい（おおきい）— 大的**
- 例文: 公園は大きいです。
- 翻譯: 公園很大。
- Common mistake: Adding な before a noun because it "feels" like it should need one (❌大きいな公園) — 大きい is an i-adjective, so it modifies a noun directly: 大きい公園.
- Memory tip: 大 alone already means "big" in Chinese; the い ending simply marks it as a Japanese i-adjective.

**2. 小さい（ちいさい）— 小的**
- 例文: 財布は小さいです。
- 翻譯: 錢包很小。
- Common mistake: Same false-な trap as 大きい (❌小さいな声) — treat it exactly like 大きい: i-adjective, no な before a noun.
- Memory tip: 小 = small in Chinese too; memorize 大きい・小さい together as an opposite pair.

**3. 新しい（あたらしい）— 新的**
- 例文: 新しい先生です。
- 翻譯: 是新來的老師。
- Common mistake: Confusing the kun-reading あたら.しい (used in this everyday adjective) with the on-reading シン (used in compounds like 新聞, "newspaper") — same kanji, two very different-sounding readings.
- Memory tip: 新 is identical to Chinese 新 (new); only the okurigana しい is new to memorize.

**4. 古い（ふるい）— 舊的（物品）**
- 例文: この財布は古いです。
- 翻譯: 這個錢包很舊。
- Common mistake: Using 古い to describe a person's age (❌おじいさんは古いです) — 古い only describes objects/things, never people; use 年上 or age words for people instead.
- Memory tip: 古 looks exactly like Chinese 古 (ancient); just remember it's for things, not people.

**5. いい — 好的**
- 例文: この学校はいいです。
- 翻譯: 這所學校很好。
- Common mistake: Trying to build negative/past forms directly from いい (❌いいくない, ❌いいかった) — いい is irregular; its negative and past forms actually come from the older form よい (よくない, よかった), not from いい itself.
- Memory tip: Memorize いい as a special two-character word that breaks the usual i-adjective conjugation pattern — one of the first true irregulars learners meet.

**6. 悪い（わるい）— 不好的、壞的**
- 例文: 今日は天気（てんき）が悪いです。
- 翻譯: 今天天氣不好。
- Common mistake: Assuming 悪い mainly means morally "evil," matching the strong sense of Chinese 惡 — in everyday Japanese it usually just means "bad/poor quality" (天気が悪い, 頭が悪い); the "evil" sense is much rarer in casual speech.
- Memory tip: Pair it with いい as its direct opposite: いい⇔悪い.

**7. 高い（たかい）— 貴的、高的**
- 例文: あの建物は高いです。
- 翻譯: 那棟建築物很高。
- Common mistake: Assuming 高い only means "tall," like Chinese 高 — Japanese uses this same word for "expensive" too; a sentence like これは高いです about a product almost always means "this is expensive," not "this is tall."
- Memory tip: 高 is identical to Chinese 高; just remember Japanese uses this ONE word for both "tall" and "expensive," unlike Chinese, which uses 貴 for "expensive."

**8. 安い（やすい）— 便宜的**
- 例文: このレストランは安いです。
- 翻譯: 這家餐廳很便宜。
- Common mistake: Confusing 安い (cheap, about price) with 安全（あんぜん, safe) — in everyday shopping contexts, 安 signals "cheap," not "safety."
- Memory tip: Pair 高い・安い as a price opposite-set: expensive⇔cheap.

**9. 暑い（あつい）— 熱的（天氣）**
- 例文: 台湾の夏（なつ）は暑いです。
- 翻譯: 台灣的夏天很熱。
- Common mistake: Using 暑い to describe hot food or drinks — 暑い is only for weather/air temperature; hot objects/food use 熱い（あつい, same reading, different kanji!).
- Memory tip: 暑 has 日 (sun) at the top — a visual reminder that this "hot" is about weather/sun, not objects.

**10. 寒い（さむい）— 冷的（天氣）**
- 例文: 今日はとても寒いです。
- 翻譯: 今天非常冷。
- Common mistake: Using 寒い for cold objects/drinks — like 暑い／熱い, weather-cold is 寒い, but object-cold is a completely different word, 冷たい（つめたい).
- Memory tip: Pair 暑い・寒い as the two weather-temperature opposites. Preview: 昨日は寒かったです ("it was cold yesterday") shows the かった past-tense ending used for talking about yesterday's weather — the full rule is taught in a later lesson, but recognize this set phrase now.

**11. 忙しい（いそがしい）— 忙碌的**
- 例文: 兄は忙しいです。
- 翻譯: 哥哥很忙。
- Common mistake: Confusing 忙しい (busy) with 忘れる（わすれる, to forget) just because both start with a similar sound — there is no kanji or meaning relation, purely a coincidental sound similarity.
- Memory tip: 忙 = 忄(heart) + 亡 (lost) — literally "losing your heart/mind," a vivid image for being busy.

**12. 暇な（ひまな）— 空閒的**
- 例文: 日曜日は暇です。
- 翻譯: 星期天很空閒。
- Common mistake: Forgetting to drop な before です for na-adjectives (❌暇なです) — な only appears directly before a noun (暇な時間), never before です.
- Memory tip: Pair 忙しい・暇な as opposites: busy⇔free. Note that 忙しい is an i-adjective but 暇な is a na-adjective — a classic pair for memorizing both adjective types at once.

**13. きれいな — 漂亮的、乾淨的**
- 例文: この公園はきれいです。
- 翻譯: 這座公園很乾淨。
- Common mistake: Forgetting な before a noun because きれい ends in い and looks like an i-adjective (❌きれい人) — きれい is actually a na-adjective in disguise; the correct form is きれいな人.
- Memory tip: Memorize きれい as one of the "fake i-adjectives" — it looks like it ends in い but grammatically behaves like a na-adjective.

**14. 静かな（しずかな）— 安靜的**
- 例文: 図書館は静かです。
- 翻譯: 圖書館很安靜。
- Common mistake: Treating 静か like an i-adjective and adding い (❌静かい) — it is a na-adjective; never add い to it.
- Memory tip: 静 resembles Chinese 靜 (quiet) — the meaning transfers directly; just remember the な grammar category.

**15. にぎやかな — 熱鬧的**
- 例文: あの町はにぎやかです。
- 翻譯: 那個城鎮很熱鬧。
- Common mistake: Assuming にぎやか is the opposite of 忙しい (busy) — it is actually the opposite of 静か (quiet), describing a lively/bustling atmosphere, not a busy schedule.
- Memory tip: Pair 静かな・にぎやかな as opposites: quiet⇔lively — both na-adjectives, both often describing places (town, street, room).

**16. 有名な（ゆうめいな）— 有名的**
- 例文: あの先生は有名です。
- 翻譯: 那位老師很有名。
- Common mistake: Forgetting な before a noun (the correct phrase is 有名な人, not ❌有名人 pronounced the same way) — note that 有名人（ゆうめいじん）does exist as a fixed compound noun meaning "celebrity," a special exception worth flagging separately.
- Memory tip: 有 (have) + 名 (name) = "having a name" = famous — the same logic as Chinese 有名.

**17. 元気な（げんきな）— 健康的、有精神的**
- 例文: 元気ですか。
- 翻譯: 你好嗎？（你精神好嗎？）
- Common mistake: Treating 元気 as equivalent to the more literary, limited Chinese 元氣 (mostly seen in set phrases like 恢復元氣) — in Japanese, 元気 is an everyday word used constantly as a greeting response ("I'm fine/well"), not a rare or poetic term.
- Memory tip: Memorize 元気です as a fixed greeting-response chunk, just like です alone can answer a simple question.

**18. 好きな（すきな）— 喜歡的**
- 例文: 私は日本が好きです。
- 翻譯: 我喜歡日本。
- Common mistake: Marking the liked thing with を instead of が (❌日本を好きです) — 好き is a na-adjective describing a feeling, and the thing liked takes が, not を (this reuses Lesson 7's 〜は〜が好きです pattern exactly).
- Memory tip: 好きな is simply the adjective form of Lesson 7's 好きです pattern — same が, same logic.

**19. 嫌いな（きらいな）— 討厭的**
- 例文: 私は納豆（なっとう）が嫌いです。
- 翻譯: 我不喜歡納豆。
- Common mistake: Using 嫌いです too bluntly about people, which can sound harsh — Japanese speakers often soften dislike statements (e.g. あまり好きじゃないです) rather than saying 嫌いです directly, especially about people.
- Memory tip: Pair 好きな・嫌いな as opposites: like⇔dislike, both na-adjectives, both using が for the object.

**20. 便利な（べんりな）— 方便的**
- 例文: この駅は便利です。
- 翻譯: 這個車站很方便。
- Common mistake: Confusing 便利 (convenient, describing things/places) with 上手（じょうず, skilled, describing a person's ability — preview vocabulary) — 便利 is never used to praise a person's skill.
- Memory tip: 便 (convenience) + 利 (benefit) = "beneficial convenience" = convenient.

**21. 不便な（ふべんな）— 不方便的**
- 例文: この村は不便です。
- 翻譯: 這個村莊很不方便。
- Common mistake: Trying to negate 便利 by attaching ません like a verb (❌便利ませんでした) — 不便 is simply the ready-made opposite word; na-adjective negation itself uses じゃないです／じゃありません (taught in a later lesson), not ません, which this lesson's grammar shows is only for verbs.
- Memory tip: 不 (negation prefix) + 便 (convenience) = "not convenient" — a very transparent compound for Chinese speakers.

**22. 楽しい（たのしい）— 開心的、有趣的**
- 例文: 日本語の勉強は楽しいです。
- 翻譯: 學日文很開心（很有趣）。
- Common mistake: Confusing 楽しい (fun/enjoyable — how an activity feels) with 面白い (interesting, item #23) — they overlap, but 楽しい leans toward "an enjoyable experience," while 面白い leans toward "intellectually interesting."
- Memory tip: 楽 resembles Chinese 樂 (joy/music) — the same root meaning of happiness/enjoyment.

**23. 面白い（おもしろい）— 有趣的**
- 例文: この本（ほん）は面白いです。
- 翻譯: 這本書很有趣。
- Common mistake: Reading 面白い too literally as "white face" and connecting it to appearance — the whole word simply means "interesting/funny," unrelated to a literal reading of its two kanji.
- Memory tip: 面 (surface/scene) + 白い (bright/vivid) historically described a bright, vivid scene, which evolved into "interesting" — don't translate the parts literally.

**24. つまらない — 無聊的**
- 例文: この映画（えいが）はつまらないです。
- 翻譯: 這部電影很無聊。
- Common mistake: Assuming つまらない must relate to つまる (to be blocked/stuffed) because the root looks similar — つまらない is a standalone i-adjective meaning "boring," best learned purely by sound, not by breaking down a kanji (it has no kanji at all).
- Memory tip: Pair 面白い・つまらない as opposites: interesting⇔boring.

**25. 難しい（むずかしい）— 困難的**
- 例文: この漢字（かんじ）は難しいです。
- 翻譯: 這個漢字很難。
- Common mistake: Confusing 難しい (difficult) with 忙しい (busy, item #11) — both are i-adjectives with a similar rhythm and both loosely relate to "hard to deal with," but they describe completely different things (task difficulty vs. schedule busyness).
- Memory tip: 難 resembles the traditional Chinese 難 (difficult) — same meaning; just add しい for the Japanese adjective form.

## ② Kanji (5): 大, 小, 新, 古, 高

**大（big）**
- Meaning (Chinese): 大
- On-reading: ダイ、タイ
- Kun-reading: おお.きい
- Words: 大きい, 大学, 大丈夫（だいじょうぶ, bonus — "it's okay/alright"）
- Example sentence: 大学はとても大きいです。
- Stroke order tip: Horizontal stroke first, then the long diagonal from upper-left to lower-right, then the diagonal from upper-right to lower-left — like a person spreading their arms wide, a classic mnemonic for "big."
- Chinese difference: Visually identical to Chinese 大; the twist is that Japanese splits the on-reading into ダイ (大学) and タイ (大丈夫, 大切), depending on the compound — Chinese uses a single reading for 大, so which on-reading applies must be memorized word by word.

**小（small）**
- Meaning (Chinese): 小
- On-reading: ショウ
- Kun-reading: ちい.さい
- Words: 小さい, 小学生（しょうがくせい, bonus — "elementary school student"）
- Example sentence: この財布は小さいです。
- Stroke order tip: Write the short central hook stroke first, then the two small strokes on either side — only 3 strokes, one of the simplest kanji shapes overall.
- Chinese difference: Identical to Chinese 小; meaning and general usage overlap fully, a low-risk kanji.

**新（new）**
- Meaning (Chinese): 新
- On-reading: シン
- Kun-reading: あたら.しい
- Words: 新しい, 新聞（しんぶん, bonus — "newspaper"）
- Example sentence: 新しい先生です。
- Stroke order tip: Build the left-side 立＋木 (standing tree) component first, then add the right-side 斤 (axe) component last — 13 strokes total, written left to right in blocks.
- Chinese difference: Identical to Chinese 新; the only new element is the kun-reading あたら.しい used for the everyday adjective, versus the on-reading シン used in compounds like 新聞.

**古（old, thing）**
- Meaning (Chinese): 古
- On-reading: コ
- Kun-reading: ふる.い
- Words: 古い, 中古（ちゅうこ, bonus — "secondhand/used"）
- Example sentence: この建物は古いです。
- Stroke order tip: Write the top cross-like stroke first, then close the 口 (mouth) shape underneath — only 5 strokes.
- Chinese difference: Identical to Chinese 古 (ancient); meaning overlaps closely, but remember 古い in Japanese describes objects/buildings, not a person's age directly.

**高（tall, expensive）**
- Meaning (Chinese): 高
- On-reading: コウ
- Kun-reading: たか.い
- Words: 高い, 高校（こうこう, bonus — "high school"）
- Example sentence: あのデパートは高いです。
- Stroke order tip: Build it top to bottom in three stacked sections — the small 亠 top, the 口 box in the middle, then the wider base — like a tall tower stacked in layers.
- Chinese difference: Identical shape to Chinese 高, but usage differs: Japanese 高い covers BOTH "tall" (height) and "expensive" (price), while Chinese separates these into 高 (tall) and 貴 (expensive) — a genuine false-friend risk in the "expensive" sense.

## ③ Grammar (2 points)

### Grammar 1 — 動詞のます形（肯定・現在）(polite present affirmative verbs)

**Meaning:** The polite, present/future affirmative form of a verb, used in standard polite conversation (with です／ます-style speech, as opposed to casual dictionary-form speech).

**Structure:**
- **Group 1 (u-verbs):** Change the final -u sound of the dictionary form to its -i sound (the "i-stem"), then add ます.
  - 飲む（のむ）→ 飲みます／買う（かう）→ 買います／話す（はなす）→ 話します／書く（かく）→ 書きます／行く（いく）→ 行きます
- **Group 2 (ru-verbs):** Drop the final る and add ます.
  - 食べる（たべる）→ 食べます／見る（みる）→ 見ます／起きる（おきる）→ 起きます／寝る（ねる）→ 寝ます
- **Irregular verbs:** Memorized individually.
  - する → します／来る（くる）→ 来ます（きます）— note the reading itself changes from く to き.

**Examples:**
1. 毎日、日本語を勉強します。(I study Japanese every day.)
2. 明日、友達に会います。(I will meet my friend tomorrow.)
3. 田中さんは七時に起きます。(Tanaka gets up at seven o'clock.)

**Common mistakes (Chinese speakers):**
- ❌ Using the dictionary form (食べる, 飲む) in polite conversation where ます-form is expected — Chinese verbs never conjugate, so learners often default to the plain/dictionary form even in polite contexts; ます-form is required for standard polite speech.
- ❌ Trying to derive します／来ます from する／来る using the regular drop-u or drop-る rules — these two verbs are irregular and must simply be memorized as fixed forms.
- ❌ Misclassifying verbs that end in -iru/-eru but are secretly Group 1: 帰る（かえる, to return home) looks like a Group 2 ru-verb but is actually Group 1, so its polite form is 帰ります (from 帰り + ます), not a Group-2-style "帰ます."

**Exercises (translate into Japanese):**
1. I get up at seven o'clock.
2. I study Japanese every day.
3. My friend comes to Taiwan.
4. I read a book at night.

### Grammar 2 — 動詞のません形（否定・現在）(polite present negative verbs)

**Meaning:** The polite, present/future negative form of a verb — "do not / will not [verb]."

**Structure:** Take the ます-stem (the same stem used in Grammar 1) and attach ません instead of ます.
- 食べます → 食べません／飲みます → 飲みません／します → しません／来ます → 来ません（きません）

**Examples:**
1. 今日は勉強しません。(I'm not studying today.)
2. 妹はあまり食べません。(My younger sister doesn't eat much.)
3. 山田さんは今日、会社に来ません。(Yamada isn't coming to the office today.)

**Common mistakes (Chinese speakers):**
- ❌ Attaching ません directly to the dictionary form (❌食べるません, ❌飲むません) — you must first form the ます-stem, then attach ません: 食べ＋ません, 飲み＋ません.
- ❌ Mixing the ません system with the plain-form ない negative (❌食べないません) — ません completely replaces ます; it is not added on top of the separate plain-form ない pattern (that plain form is taught later).
- ❌ Forgetting the irregular reading change in 来ません（きません）and pronouncing it with the raw kanji reading — 来る always changes its reading to き in the ます／ません forms.

**Exercises (translate into Japanese):**
1. I don't work today.
2. My younger brother doesn't study.
3. My friend isn't coming today.
4. I don't clean on Sundays.

**Grammar Answer Key:**
- Grammar 1: ① 七時に起きます。 ② 毎日、日本語を勉強します。 ③ 友達が台湾に来ます。 ④ 夜、本（ほん）を読みます。
- Grammar 2: ① 今日は働きません。 ② 弟は勉強しません。 ③ 友達は今日、来ません。 ④ 日曜日は掃除しません。

## ④ Reading

田中さんは私の友達です。田中さんはいつも元気ですが、忙しいです。日曜日は暇です。田中さんは毎日七時に起きます。田中さんの家（いえ）は静かで、きれいです。（「静かで」は形容詞をつなぐ形で、後の課で詳しく学びます。）田中さんは本（ほん）が好きですが、寒い天気（てんき）は嫌いです。暑い天気が好きです。

**Comprehension questions:**
1. 田中さんはどんな人ですか。
2. 田中さんは何時に起きますか。
3. 田中さんはどんな天気が好きですか。

**Answer key:**
1. 元気な人です。忙しいですが、日曜日は暇です。
2. 七時に起きます。
3. 暑い天気が好きです。（寒い天気は嫌いです。）

## 🇹🇼 Chinese Notes

- **元気**: In Chinese, 元氣 is a fairly literary or limited word (e.g. 恢復元氣). In Japanese, 元気 is an everyday word meaning "well/healthy/energetic," used constantly as a simple greeting response (元気です) — don't treat it as rare or formal.
- **高い covers both "tall" and "expensive"**: Chinese keeps these separate (高 = tall, 貴 = expensive), but Japanese uses 高い for both. Context — usually whether you're discussing a building/person versus a price — tells you which meaning is intended.
- **な-adjectives have no equivalent category in Chinese**: Words like きれいな, 静かな, 元気な, 好きな behave grammatically like adjectives but need な before a noun and drop it before です — a distinct grammatical class (形容動詞) that Chinese, which doesn't conjugate or mark adjectives this way, has no direct counterpart for. Chinese speakers should treat na-adjectives as "noun-like adjectives" requiring their own memorized attachment rule, not simply as vocabulary to memorize by meaning alone.

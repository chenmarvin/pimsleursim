# 🇯🇵 Japanese for Taiwanese Learners — JLPT N5, Lesson 4

**Theme:** 「今日は何日ですか」(What's today's date?)

**Objectives**
- Say today's date and day of the week
- Talk about this/next/last week, month, and year
- Say where things are with 〜があります

> Note: This lesson's seed vocabulary/kanji set (time expressions and days of the week) has 25 words and 15 kanji, since Japanese time vocabulary is taught as a connected block (今/来/先 + 週/月/年, plus the seven day-of-week kanji). All items below are covered in full, per the curated N5 syllabus.

### Word-by-word details

**1. 今（いま）— 現在**
- 例文: 今、何時ですか。
- 翻譯: 現在幾點？
- Common mistake: Confusing 今 (now, this exact moment) with 今日 (today, the whole day) — 今 is much narrower in scope.
- Memory tip: 今 alone = "now"; adding 日/週/月/年 broadens the time frame to day/week/month/year.

**2. 今日（きょう）— 今天**
- 例文: 今日は月曜日です。
- 翻譯: 今天是星期一。
- Common mistake: Reading 今日 with its component readings いま+ひ instead of the special fused reading きょう — this is an irregular reading that must be memorized as a whole word.
- Memory tip: きょう is one of a small set of irregular date readings (today/tomorrow/yesterday) — memorize all three together.

**3. 明日（あした）— 明天**
- 例文: 明日は休みです。
- 翻譯: 明天放假。
- Common mistake: Guessing a reading based on the kanji 明 (bright) + 日 (day) — the actual reading あした is irregular and unrelated to either kanji's normal readings.
- Memory tip: Group with 今日／昨日 as the three essential irregular date words.

**4. 昨日（きのう）— 昨天**
- 例文: 昨日は忙しかったです。（過去形は後の課で詳しく）
- 翻譯: 昨天很忙。
- Common mistake: Confusing 昨日 (きのう, yesterday) with 去年 (きょねん, last year) — different time units, easy to mix up when memorizing quickly.
- Memory tip: Complete the trio: 昨日（きのう）・今日（きょう）・明日（あした） — yesterday, today, tomorrow.

**5. 今週（こんしゅう）— 這星期**
- 例文: 今週は忙しいです。
- 翻譯: 這星期很忙。
- Common mistake: Mixing up the readings こん (this) vs らい (next) vs せん (last) when attached to 週/月 — these prefixes are consistent across 週/月, so learn the pattern once and reuse it.
- Memory tip: 今 (now) + 週 (week) = "this week," following the regular こん〜 pattern.

**6. 来週（らいしゅう）— 下星期**
- 例文: 来週、日本へ行きます。
- 翻譯: 下星期要去日本。
- Common mistake: Reading 来 as く (from 来る, to come) instead of らい in this compound — 来週 uses the on-reading らい, a different reading from the verb's kun-reading.
- Memory tip: 来 (come/future) + 週 (week) = "the week that's coming" = next week.

**7. 先週（せんしゅう）— 上星期**
- 例文: 先週は休みでした。
- 翻譯: 上星期休假了。
- Common mistake: Confusing 先週 (last week) with 先生 (teacher, Lesson 1) — both start with 先, but the reading differs (せんしゅう vs せんせい) and meanings are unrelated.
- Memory tip: 先 (previous/ahead) + 週 (week) = "the week before" = last week.

**8. 今月（こんげつ）— 這個月**
- 例文: 今月は忙しいです。
- 翻譯: 這個月很忙。
- Common mistake: Reading 月 as つき (its kun-reading, meaning "moon/month" standalone) instead of げつ in this compound — 月 shifts reading depending on whether it's standalone or part of a time-word compound.
- Memory tip: Same こん〜 pattern as 今週 — just swap 週 for 月.

**9. 来月（らいげつ）— 下個月**
- 例文: 来月、誕生日です。
- 翻譯: 下個月是我的生日。
- Common mistake: Same らい〜 pattern reminder as 来週 — 来 always reads らい in these "next ___" time compounds.
- Memory tip: Reuse the 来週→来月 pattern; only the second kanji changes.

**10. 先月（せんげつ）— 上個月**
- 例文: 先月、旅行しました。（旅行 = travel, glossed for context）
- 翻譯: 上個月去旅行了。
- Common mistake: Same せん〜 pattern reminder as 先週 — don't confuse with 先生.
- Memory tip: Reuse the 先週→先月 pattern; only the second kanji changes.

**11. 今年（ことし）— 今年**
- 例文: 今年は忙しいです。
- 翻譯: 今年很忙。
- Common mistake: Reading 今年 with the expected こんねん (following the こん〜 pattern) instead of the irregular ことし — this is the one exception that breaks the otherwise regular こん／らい／せん pattern.
- Memory tip: Memorize ことし as the single big exception in this whole vocabulary set.

**12. 来年（らいねん）— 明年**
- 例文: 来年、大学生になります。（〜になります = become, glossed）
- 翻譯: 明年要成為大學生。
- Common mistake: None major — 来年 correctly follows the regular らい〜 pattern (unlike 今年's irregularity).
- Memory tip: 来 (next) + 年 (year) = "next year," regular pattern restored here.

**13. 去年（きょねん）— 去年**
- 例文: 去年、日本へ行きました。
- 翻譯: 去年去了日本。
- Common mistake: Expecting せんねん (following the せん〜 "last" pattern) but the actual word is 去年 with a totally different kanji 去 (leave/past) — another exception to memorize alongside 今年.
- Memory tip: Memorize 今年・来年・去年 as a set with two regular members (来年) and two irregular-looking ones (今年, 去年) that use different kanji roots than 週/月.

**14. 月曜日（げつようび）— 星期一**
- 例文: 月曜日から働きます。（働く from Lesson 6, preview）
- 翻譯: 星期一開始上班。
- Common mistake: Dropping 日 at the end (❌ 月曜) in formal writing — 曜日 is the standard full form, though 月曜 alone is acceptable in very casual speech.
- Memory tip: 月 (moon) + 曜 (day-of-week marker) + 日 (day) = Monday, matching the Chinese logic of 星期一 built from a weekday system.

**15. 火曜日（かようび）— 星期二**
- 例文: 火曜日に病院へ行きます。（病院 preview, Lesson 7）
- 翻譯: 星期二要去醫院。
- Common mistake: Confusing 火 (かようび, Tuesday) with 火（ひ, fire) used elsewhere — same kanji, different reading depending on context (day-of-week compound uses on-reading か).
- Memory tip: The seven day-of-week kanji (月火水木金土日) match the five elements + sun/moon — a fun mnemonic set shared conceptually with Chinese astrology.

**16. 水曜日（すいようび）— 星期三**
- 例文: 水曜日は暇です。（暇 from Lesson 8, preview）
- 翻譯: 星期三有空。
- Common mistake: Misreading 水 as みず (its common kun-reading) instead of すい in this compound.
- Memory tip: 水 (water) — third day, third element in the classical five-elements sequence.

**17. 木曜日（もくようび）— 星期四**
- 例文: 木曜日に友達と会います。（会う from Lesson 6, preview）
- 翻譯: 星期四要跟朋友見面。
- Common mistake: Misreading 木 as き (its common kun-reading) instead of もく in this compound.
- Memory tip: 木 (tree/wood) — fourth day, fourth element.

**18. 金曜日（きんようび）— 星期五**
- 例文: 金曜日は忙しいです。
- 翻譯: 星期五很忙。
- Common mistake: Confusing 金 (きんようび, Friday) with 金 meaning "money/gold" (also きん) — same reading here, but be sure context (day list vs. currency) is clear.
- Memory tip: 金 (gold) — fifth day, fifth element; also the most-anticipated day of the work week.

**19. 土曜日（どようび）— 星期六**
- 例文: 土曜日はいつも休みです。
- 翻譯: 星期六總是休假。
- Common mistake: Misreading 土 as つち (its common kun-reading, "soil") instead of ど in this compound.
- Memory tip: 土 (earth/soil) — sixth day, completing the five elements (火水木金土) sandwiched between 月 (moon) and 日 (sun).

**20. 日曜日（にちようび）— 星期日**
- 例文: 日曜日に映画を見ます。（映画・見る preview）
- 翻譯: 星期日要看電影。
- Common mistake: Confusing にち (day-of-week reading) with に or ひ, other readings of 日 depending on the word.
- Memory tip: 日 (sun) — completes the week alongside 月 (moon); together 日曜日・月曜日 bookend the seven-day cycle.

**21. 何曜日（なんようび）— 星期幾**
- 例文: 今日は何曜日ですか。
- 翻譯: 今天星期幾？
- Common mistake: Using 何日 (なんにち, what date) when asking about the day of the WEEK instead — 何曜日 and 何日 are easily confused since both start with 何 and relate to dates.
- Memory tip: 何曜日 asks specifically "which weekday," while 何日 (preview) asks "which date/day-of-month."

**22. 誕生日（たんじょうび）— 生日**
- 例文: 誕生日はいつですか。（いつ = when, glossed）
- 翻譯: 你的生日是什麼時候？
- Common mistake: Forgetting the final 日 or mispronouncing the compound as separate words instead of one fused reading たんじょうび.
- Memory tip: 誕生 (birth) + 日 (day) = "the day of birth" = birthday.

**23. 休み（やすみ）— 休假、放假**
- 例文: 明日は休みです。
- 翻譯: 明天放假。
- Common mistake: Confusing the noun 休み (a day off/vacation) with the verb 休む (Lesson 6, to rest) — related root but different parts of speech and usage.
- Memory tip: 休み as a noun answers "is it a day off?" (休みです), while 休む as a verb describes the action of resting.

**24. 予定（よてい）— 預定、計畫**
- 例文: 今週の予定はありますか。
- 翻譯: 這星期有安排嗎？
- Common mistake: Using 予定 interchangeably with 約束（やくそく, "promise/appointment," a later word) — 予定 is a general schedule/plan, not a fixed promise between two people.
- Memory tip: 予 (in advance) + 定 (decide/fix) = "decided in advance" = a schedule/plan.

**25. 時間（じかん）— 時間**
- 例文: 時間がありません。
- 翻譯: 沒有時間。
- Common mistake: Confusing 時間 (time, duration, an abstract noun) with 時（じ, "o'clock," used for specific clock times, preview) — 時間 measures duration/availability, while 時 marks a specific point in time.
- Memory tip: 時 (time/hour) + 間 (interval/between) = "the space of time" = time/duration.

## ② Kanji (15): 今, 月, 年, 週, 時, 間, 先, 来, 半, 何, 火, 水, 木, 金, 土

**今** — Meaning: 今 (now). On: コン、キン. Kun: いま. Words: 今日, 今週, 今月. Example: 今、忙しいです。 Stroke order: top diagonal stroke, then the enclosing stroke, then the final short stroke inside. Chinese difference: Identical to Chinese 今; meaning and basic usage transfer directly.

**月** — Meaning: 月 (month, moon). On: ゲツ、ガツ. Kun: つき. Words: 今月, 一月（いちがつ, preview）, 月曜日. Example: 空に月があります。（preview sentence using kun-reading） Stroke order: outer curved frame first (like a bent rectangle), then the two internal horizontal strokes. Chinese difference: Identical to Chinese 月; note Japanese uses two different on-readings (ゲツ in 月曜日, ガツ in month names like 一月) depending on the compound — Mandarin uses a single reading yuè throughout.

**年** — Meaning: 年 (year). On: ネン. Kun: とし. Words: 今年, 来年, 去年. Example: 今年は忙しいです。 Stroke order: top horizontal stroke, then the vertical stroke down through the middle crossing two more horizontal strokes, finishing with the long bottom horizontal stroke (similar structure to 生). Chinese difference: Identical to Chinese 年; meaning and usage overlap closely.

**週** — Meaning: 週 (week). On: シュウ. Kun: none in common use. Words: 今週, 来週, 先週. Example: 今週は忙しいです。 Stroke order: the enclosing "walking" radical (辶) is written last in most references, but stroke practice apps typically have learners draw the inner 周 component first, then close with 辶. Chinese difference: Same character as Traditional Chinese 週 (used in Taiwan); no meaning shift, an easy one-to-one match.

**時** — Meaning: 時 (time, hour). On: ジ. Kun: とき. Words: 時間, 何時（preview, "what time"）, 時々（preview, "sometimes"）. Example: 今、何時ですか。 Stroke order: 日 (sun) radical first on the left, then the 寺 (temple) component on the right, written top to bottom. Chinese difference: Identical to Traditional Chinese 時; meanings overlap closely, including the "hour/o'clock" sense.

**間** — Meaning: 間 (interval, between). On: カン、ケン. Kun: あいだ. Words: 時間, 人間（にんげん, preview, "human"）, 週間（preview, "period of a week"）. Example: 時間がありません。 Stroke order: outer 門 (gate) radical first, then 日 (sun) written inside it. Chinese difference: Identical to Chinese 間/间-related forms; visually matches Traditional Chinese 間 used in Taiwan exactly.

**先** — Meaning: 先 (previous, ahead). On: セン. Kun: さき. Words: 先週, 先月, 先生（Lesson 1）. Example: 先週は忙しかったです。 Stroke order: top component (a stylized 生-like shape) first, then the two legs at the bottom. Chinese difference: Identical to Chinese 先; meaning ("before/ahead") transfers directly, and this same kanji already appeared in 先生 (Lesson 1) with the same core meaning of "coming before."

**来** — Meaning: 来 (come). On: ライ. Kun: く.る、きた.る. Words: 来週, 来月, 来年. Example: 来週、日本へ来ます。 Stroke order: top short stroke, then the crossing horizontal and vertical strokes, then the two diagonal strokes at the bottom. Chinese difference: This is the Japanese simplified form of Traditional Chinese 來 (used in Taiwan) — visually closer to Simplified Chinese 来. Same pattern as noted for 学 in Lesson 1: expect some Japanese kanji forms to resemble Simplified rather than Traditional Chinese.

**半** — Meaning: 半 (half). On: ハン. Kun: なか.ば. Words: 半分（Lesson 2）, 一時半（preview, "half past one"）. Example: 今、三時半です。 Stroke order: top two short strokes, then the vertical stroke through the center, crossed by the horizontal stroke, finishing with the bottom-left and bottom-right strokes. Chinese difference: Identical to Chinese 半; meaning overlaps fully, including its use in time expressions ("half past").

**何** — Meaning: 何 (what). On: カ. Kun: なに、なん. Words: 何曜日, 何時（preview）, 何人（preview）. Example: 今日は何曜日ですか。 Stroke order: person radical (亻) on the left first, then the 可 component on the right. Chinese difference: Identical to Chinese 何 (a more literary/formal "what" in Mandarin), but in Japanese 何 is the everyday, most common word for "what" — not a formal/literary register as it can feel in Chinese.

**火** — Meaning: 火 (fire). On: カ. Kun: ひ. Words: 火曜日, 花火（preview, "fireworks"）. Example: 火曜日に会いましょう。（preview grammar） Stroke order: center dot-like short stroke, then the left stroke, then the right stroke, forming a flame shape. Chinese difference: Identical to Chinese 火; simple, low-risk kanji shared visually and semantically.

**水** — Meaning: 水 (water). On: スイ. Kun: みず. Words: 水曜日, 水 (みず, "water," Lesson 9 vocab). Example: 水を飲みます。（preview grammar, Lesson 6） Stroke order: center vertical hook stroke first, then the left and right strokes flowing outward like droplets. Chinese difference: Identical to Chinese 水; straightforward with no meaning shift.

**木** — Meaning: 木 (tree, wood). On: ボク、モク. Kun: き. Words: 木曜日, 木 (き, "tree"). Example: 公園に木があります。（preview, "There is a tree in the park"） Stroke order: horizontal stroke first, then the vertical stroke down through the center, finishing with the two diagonal strokes at the bottom. Chinese difference: Identical to Chinese 木; no meaning shift, low risk.

**金** — Meaning: 金 (money, gold, Friday). On: キン、コン. Kun: かね. Words: 金曜日, お金 (Lesson 2). Example: 金曜日は楽しいです。（preview, Lesson 14） Stroke order: top triangular "roof" stroke, then the vertical stroke, then the two horizontal strokes, finishing with the two small dots at the bottom. Chinese difference: Identical to Chinese 金; meaning overlaps for both "gold/money" and as a component in day-of-week vocabulary, unlike Chinese which doesn't use 金 to name a weekday.

**土** — Meaning: 土 (soil, earth). On: ド、ト. Kun: つち. Words: 土曜日, 土地（preview, "land"）. Example: 土曜日は休みです。 Stroke order: horizontal stroke first, then the vertical stroke crossing through the center, finishing with the bottom horizontal stroke (similar to 十, but with an extra base line). Chinese difference: Identical to Chinese 土; simple, low-risk kanji, though note it names a weekday in Japanese with no equivalent naming convention in Chinese.

## ③ Grammar (2 points)

### Grammar 1 — ここ・そこ・あそこ・どこ (here/there/over there/where)
**Meaning:** Location pronouns for places, following the same near-speaker/near-listener/far-from-both/question pattern as これ・それ・あれ・どれ (Lesson 3).
**Structure:** ここ／そこ／あそこ／どこ + は／に + Noun/です
**Examples:**
1. ここは学校です。(Here is a school.)
2. 図書館はどこですか。（preview, Lesson 7） (Where is the library?)
3. あそこに友達がいます。(My friend is over there.)

**Common mistakes (Chinese speakers):**
- ❌ Using これ／それ／あれ (object pronouns from Lesson 3) to talk about a place instead of ここ／そこ／あそこ — Japanese strictly separates "thing" pronouns from "place" pronouns, unlike Chinese 這裡/那裡 which are clearly place-marked but sometimes conflated by beginners transferring これ/それ patterns.
- ❌ Forgetting に before a verb of existence (あります／います) when stating something is located somewhere — see Grammar 2 below.

**Exercises (translate into Japanese):**
1. Here is my school.
2. Where is your house?
3. My family is over there.

### Grammar 2 — 〜がいます (there is/are — for animate/living things)
**Meaning:** States the existence of a living being (person or animal) at a location.
**Structure:** Location + に + Person/Animal + が + います
**Examples:**
1. 教室に学生がいます。(There is a student in the classroom.)
2. 家に父がいます。(My father is at home.)
3. 公園に犬がいます。（犬 = dog, glossed for context, preview） (There is a dog in the park.)

**Common mistakes (Chinese speakers):**
- ❌ Using あります (for inanimate things, formally taught in Lesson 6) instead of います for people/animals — this animate/inanimate distinction does not exist in Chinese 有, which covers both cases with one verb.
- ❌ Dropping に after the location noun (❌ 教室学生がいます) → ⭕ 教室に学生がいます — に is required to mark the location.

**Exercises (translate into Japanese):**
1. There is a teacher in the classroom.
2. My mother is at home.
3. There is a child over there.

## 🇹🇼 Chinese Notes

- **今年（ことし）**: Unlike the regular pattern seen in 今週／今月 (both read with こん), 今年 breaks the pattern with the irregular reading ことし. Chinese 今年 has no such irregularity — it's a good reminder that Japanese on/kun-reading choice isn't always predictable from the pattern alone.
- **何（なに／なん）**: In Chinese, 何 is a fairly formal/literary word for "what" (何謂, 何時 in classical-flavored writing). In Japanese, 何 is the ordinary, everyday word for "what," used constantly in casual speech — don't assume it carries the same formal register as it does in Mandarin.
- **週（しゅう）**: Visually identical between Japanese and Traditional Chinese used in Taiwan (週), unlike 来 or 学, which follow Simplified-Chinese-like forms. Not every Japanese kanji diverges from Traditional Chinese — it's worth noticing which do and which don't as you build kanji intuition.

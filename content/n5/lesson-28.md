# 🇯🇵 Japanese for Taiwanese Learners — JLPT N5, Lesson 28

**Theme:** 「仕事は何ですか」(What's your job?)

**Objectives**
- Talk about jobs and the workplace
- Explain things with 〜という〜
- Talk about doing things in advance with 〜ておきます

*(Note: this lesson's actual grammar points, per the syllabus, are 〜くれます／〜と思います／〜でしょう — see ③ Grammar below. 〜という〜 and 〜ておきます are covered in Lesson 33.)*

### Word-by-word details

**1. 公務員（こうむいん）— 公務員**
- 例文: 兄は公務員です。
- 翻譯: 哥哥是公務員。
- Common mistake: Confusing with 会社員 (かいしゃいん, "office worker," Lesson 1) — 公務員 specifically means a government/civil-service employee, not any office worker.
- Memory tip: 公（public）＋務（duty）＋員（member）＝ "a member serving public duty."

**2. 警察官（けいさつかん）— 警察**
- 例文: 彼は警察官です。
- 翻譯: 他是警察。
- Common mistake: Dropping 官 and just saying 警察 to refer to a person — 警察 alone usually refers to the police *organization*, while 警察官 refers to an individual officer.
- Memory tip: 警察（police）＋官（official）＝ "police official."

**3. 料理人（りょうりにん）— 廚師**
- 例文: 父は料理人です。
- 翻譯: 爸爸是廚師。
- Common mistake: Using シェフ (a katakana loanword also common for "chef") interchangeably without noticing 料理人 is the more neutral native term, usable for any style of cooking.
- Memory tip: 料理（cooking, Lesson 29）＋人（person）＝ "a cooking person."

**4. エンジニア — 工程師**
- 例文: 彼女はエンジニアです。
- 翻譯: 她是工程師。
- Common mistake: Trying to translate "engineer" using kanji (技師 exists but is less common in casual speech) — エンジニア is the standard modern loanword.
- Memory tip: Katakana loanword from English "engineer."

**5. 主婦（しゅふ）— 家庭主婦**
- 例文: 私の母は主婦です。
- 翻譯: 我媽媽是家庭主婦。
- Common mistake: Confusing with 奥さん (おくさん, "someone else's wife," Lesson 3) — 主婦 describes an occupation/role, not a relationship term.
- Memory tip: 主（main）＋婦（woman）＝ "the main woman [of the household]."

**6. 給料（きゅうりょう）— 薪水**
- 例文: 今月の給料はいくらですか。
- 翻譯: 這個月的薪水是多少？
- Common mistake: Confusing with 料金 (りょうきん, "fee," not yet studied) — 給料 specifically means wages/salary paid to an employee.
- Memory tip: 給（provide）＋料（payment）＝ "provided payment" = salary.

**7. 出勤する（しゅっきんする）— 上班**
- 例文: 毎朝9時に出勤します。
- 翻譯: 每天早上九點上班。
- Common mistake: Confusing with 出発する (しゅっぱつする, "to depart," Lesson 17) — 出勤する specifically means going to one's workplace, not departing in general.
- Memory tip: 出（exit/go out）＋勤（work duty）＝ "go out to work duty."

**8. 退勤する（たいきんする）— 下班**
- 例文: 6時に退勤します。
- 翻譯: 六點下班。
- Common mistake: Using 帰る (かえる, "to return home," Lesson 6) as if it meant "clock out" — 退勤する specifically marks the end of the work shift, while 帰る just means going home (which may happen later).
- Memory tip: Pairs directly as the opposite of 出勤する (#7): 退（withdraw）＋勤（work duty）.

**9. 名刺（めいし）— 名片**
- 例文: 名刺をいただけますか。
- 翻譯: 可以給我您的名片嗎？
- Common mistake: Confusing with 名前 (なまえ, "name," Lesson 1) due to the shared 名 kanji — 名刺 is the physical business card, not the name itself.
- Memory tip: 名（name）＋刺（card, historically "to pierce/present"）＝ "name card."

**10. 紹介する（しょうかいする）— 介紹**
- 例文: 友達を紹介します。
- 翻譯: 介紹朋友。
- Common mistake: Using 案内する (Lesson 26, "to guide through a place") when introducing a *person* is meant — 紹介する is for people, 案内する is for places.
- Memory tip: 紹（connect）＋介（mediate）＝ "connect and mediate" = introduce.

**11. 就職する（しゅうしょくする）— 就業、找到工作**
- 例文: 来年、就職します。
- 翻譯: 明年要開始工作了。
- Common mistake: Confusing with 卒業する (そつぎょうする, "to graduate," Lesson 16) — the two often happen close together but mean different things.
- Memory tip: 就（take up）＋職（job）＝ "take up a job" = get employed.

**12. 辞める（やめる）— 辭職**
- 例文: 会社を辞めます。
- 翻譯: 要辭職了。
- Common mistake: Confusing with 止める (also やめる, "to stop/quit" a general activity, Lesson 32) — same pronunciation and often overlapping kanji use; context (a job) signals the 辞める meaning.
- Memory tip: 辞（resign/decline）＋める＝ "to resign."

**13. 昇進する（しょうしんする）— 升職**
- 例文: 部長に昇進しました。
- 翻譯: 升職成部長了。
- Common mistake: Confusing with 出勤する/退勤する due to similar-sounding しん/きん endings — 昇進する is specifically about a promotion in rank.
- Memory tip: 昇（rise）＋進（advance）＝ "rise and advance" = promotion.

**14. 残業する（ざんぎょうする）— 加班**
- 例文: 今日は残業します。
- 翻譯: 今天要加班。
- Common mistake: Confusing with 授業 (じゅぎょう, "class," Lesson 16) — different kanji (残 vs 授) and completely different meaning despite the shared ぎょう sound.
- Memory tip: 残（remaining）＋業（work）＝ "remaining work" = overtime.

**15. 部長（ぶちょう）— 部長、經理**
- 例文: 部長はとても忙しいです。
- 翻譯: 部長非常忙。
- Common mistake: Assuming 部長 always equals an English "department head" exactly — Japanese company job titles (部長, 課長, 社長) form a specific hierarchy that doesn't map one-to-one onto Western/Chinese titles.
- Memory tip: 部（department）＋長（chief）＝ "department chief."

**16. 同僚（どうりょう）— 同事**
- 例文: 同僚と昼ご飯を食べます。
- 翻譯: 和同事吃午餐。
- Common mistake: Confusing with 友達 (Lesson 1, "friend") — 同僚 specifically means a coworker, a professional relationship, not necessarily a friendship.
- Memory tip: 同（same）＋僚（colleague）＝ "same-rank colleague."

**17. 面接（めんせつ）— 面試**
- 例文: 明日、面接があります。
- 翻譯: 明天有面試。
- Common mistake: Confusing with 試験 (しけん, "exam," Lesson 16) — 面接 is specifically a face-to-face interview, not a written test.
- Memory tip: 面（face）＋接（meet/contact）＝ "meeting face to face."

**18. 履歴書（りれきしょ）— 履歷表**
- 例文: 履歴書を書きます。
- 翻譯: 寫履歷表。
- Common mistake: Confusing with 教科書 (きょうかしょ, "textbook," not yet studied) due to the shared 書 ending — 履歴書 specifically means a personal resume/CV.
- Memory tip: 履歴（personal history）＋書（document）＝ "personal history document" = resume.

**19. 契約（けいやく）— 合約**
- 例文: 会社と契約しました。
- 翻譯: 和公司簽了合約。
- Common mistake: Confusing with 約束 (やくそく, "promise/appointment," Lesson 21) — 契約 is a formal, often written, legal agreement, while 約束 is a casual personal promise.
- Memory tip: 契（bond/pledge）＋約（arrange）＝ "a pledged arrangement" = contract.

**20. 社長（しゃちょう）— 總經理、社長**
- 例文: 社長に会いました。
- 翻譯: 見到了社長。
- Common mistake: Overusing さん after 社長 unnecessarily in address (社長さん is common but slightly redundant/casual) — simply 社長 alone, used as a title, is already respectful in formal contexts.
- Memory tip: 社（company）＋長（chief）＝ "company chief" — the highest position in a Japanese company.

**21. 部下（ぶか）— 下屬**
- 例文: 彼には部下が5人います。
- 翻譯: 他有五個下屬。
- Common mistake: Confusing with 部長 (#15) due to the shared 部 kanji — 部下 means "subordinate," the opposite hierarchy direction from 部長 ("department chief").
- Memory tip: 部（department）＋下（below）＝ "below in the department" = subordinate.

**22. 事務所（じむしょ）— 辦事處、事務所**
- 例文: 事務所で働いています。
- 翻譯: 在事務所工作。
- Common mistake: Using 会社 (かいしゃ, "company," Lesson 7) when 事務所 (a specific office/workplace location, sometimes for smaller firms like law or design offices) is meant.
- Memory tip: 事務（office work）＋所（place）＝ "office-work place."

**23. 出張する（しゅっちょうする）— 出差**
- 例文: 来週、大阪へ出張します。
- 翻譯: 下週要去大阪出差。
- Common mistake: Confusing with 旅行する (Lesson 26, "to travel") — 出張する is specifically work-related travel, not leisure travel.
- Memory tip: 出（go out）＋張（extend/expand）＝ "go out to extend [business]" = business trip.

**24. 休暇（きゅうか）— 休假**
- 例文: 来月、休暇を取ります。
- 翻譯: 下個月要請假。
- Common mistake: Confusing with 休み (やすみ, "day off," Lesson 4) — 休暇 is a more formal word, typically used for planned, longer leave from work.
- Memory tip: 休（rest）＋暇（free time）＝ "resting free time" = leave/vacation.

**25. 定年（ていねん）— 退休年齡**
- 例文: 父は来年、定年です。
- 翻譯: 爸爸明年就要退休了。
- Common mistake: Confusing 定年 (the fixed age of mandatory retirement) with just "retiring" generally (退職する) — 定年 specifically refers to reaching that fixed age.
- Memory tip: 定（fixed）＋年（year/age）＝ "the fixed age" = retirement age.

## ② Kanji (5): 仕, 事, 働, 出, 休

**仕 — Meaning: 仕 (serve, work)**
- On-reading: シ
- Kun-reading: —
- Words: 仕事
- Example sentence: 私の仕事は先生です。
- Stroke order: 5 strokes — イ (person radical, 2 strokes), then 士 (3 strokes) on the right.
- Chinese difference: Same character as Chinese 仕 (as in 仕途, "official career"), but modern Chinese rarely uses 仕 alone in everyday speech, while Japanese 仕事 (work/job) is one of the most common everyday nouns.

**事 — Meaning: 事 (matter, affair)**
- On-reading: ジ
- Kun-reading: こと
- Words: 仕事, 用事 (Lesson 21)
- Example sentence: これは大切な仕事です。
- Stroke order: 8 strokes — stacked horizontal strokes first, finishing with one long vertical hook stroke through the center.
- Chinese difference: Same character as Chinese 事, meaning "matter/affair" in both, but Japanese also uses こと as an abstract grammatical nominalizer (as in 〜ことができます, Lesson 34) — a function Chinese 事 does not have.

**働 — Meaning: 働 (work)**
- On-reading: ドウ
- Kun-reading: はたら.く
- Words: 働く (Lesson 6), 仕事
- Example sentence: 会社で働いています。
- Stroke order: 13 strokes — イ (person, 2 strokes) on the left, then 動 (motion, 11 strokes) on the right.
- Chinese difference: 働 is a kanji invented in Japan (国字) — it does not exist in standard Chinese hanzi at all (Chinese uses 工作 or 劳动 for "to work"). This is a rare and useful exception to remember: not every kanji has a Chinese ancestor.

**出 — Meaning: 出 (exit, leave)**
- On-reading: シュツ
- Kun-reading: で.る、だ.す
- Words: 出勤する, 出張する
- Example sentence: 朝9時に家を出ます。
- Stroke order: 5 strokes — the left 山-like stroke set first, then a repeated, slightly offset stroke set to the right.
- Chinese difference: Same meaning as Chinese 出 ("exit/go out") and usage is quite similar; Japanese distinguishes the intransitive/transitive pair 出る (to exit) vs. 出す (to take something out) through different readings, a distinction Chinese doesn't mark morphologically.

**休 — Meaning: 休 (rest)**
- On-reading: キュウ
- Kun-reading: やす.む
- Words: 休む (Lesson 6), 休暇, 休み (Lesson 4)
- Example sentence: 今日は仕事を休みます。
- Stroke order: 6 strokes — イ (person, 2 strokes), then 木 (tree, 4 strokes) — "a person resting against a tree."
- Chinese difference: Same character and meaning as Chinese 休 (rest); the vivid "person + tree" image works identically as a memory aid in both languages.

## ③ Grammar

### Grammar 1 — 〜くれます (someone gives to me / my in-group)
**Meaning:** Someone else gives something to the speaker (or the speaker's in-group), viewed from the receiver's perspective.
**Structure:** 人が（私に）物をくれます
**Examples:**
1. 部長がボーナスをくれました。(My boss gave me a bonus.)
2. 同僚が仕事を手伝ってくれました。(My coworker helped me with work.)
3. 友達が名刺をくれました。(My friend gave me a business card.)

**Common mistakes (Chinese speakers):**
- Confusing くれます with あげます (Lesson 27) — くれます is used specifically when the giver gives *to the speaker* (or the speaker's in-group), while あげます is used when the speaker or a third party gives to someone else. Learners often default to あげます in every giving sentence.
- Forgetting that 〜てくれます can attach to a verb (as in 手伝ってくれました) to mean "did [the favor of] doing X for me," not just "gave me an object."

**Exercises (translate into Japanese):**
1. My coworker gave me a present.
2. My department manager taught me [how to do the job].

### Grammar 2 — 〜と思います (I think that...)
**Meaning:** Expresses the speaker's own opinion or thought.
**Structure:** [plain-form clause] ＋ と思います
**Examples:**
1. 彼はいい社長だと思います。(I think he is a good president.)
2. この仕事は面白いと思います。(I think this job is interesting.)
3. 明日、会議があると思います。(I think there will be a meeting tomorrow.)

**Common mistakes (Chinese speakers):**
- Attaching と思います directly after a polite-form verb (❌ 面白いですと思います) instead of the plain form — と思います always follows the plain/dictionary style clause.
- Forgetting だ before と思います after a noun or na-adjective (❌ 社長と思います → ⭕ 社長だと思います).

**Exercises (translate into Japanese):**
1. I think this company is good.
2. I think he will quit his job.

### Grammar 3 — 〜でしょう (probably)
**Meaning:** Expresses the speaker's guess or inference about something.
**Structure:** [plain-form clause] ＋ でしょう
**Examples:**
1. 明日は雨でしょう。(It will probably rain tomorrow.)
2. 彼は忙しいでしょう。(He is probably busy.)
3. 部長はもう帰ったでしょう。(The manager has probably already gone home.)

**Common mistakes (Chinese speakers):**
- Confusing でしょう (a speaker's own guess, "probably") with a plain yes/no question — でしょう said with rising intonation can also mean "...right?", seeking the listener's agreement, a nuance learners often miss.
- Overusing でしょう where a simple です would do — でしょう specifically signals uncertainty/inference, not a plain statement of fact.

**Exercises (translate into Japanese):**
1. It will probably be hot tomorrow.
2. He is probably a coworker of yours.

## ④ Reading

田中さんの仕事はエンジニアです。毎日会社へ行きます。今日は残業しました。同僚が仕事を手伝ってくれました。田中さんは「同僚はとてもいい人だ」と思いました。部長から電話がありました。「明日は大切な会議があるでしょう」と部長は言いました。田中さんはまた忙しいと思います。

**Comprehension questions:**
1. 田中さんの仕事は何ですか。
2. 今日、誰が田中さんを手伝いましたか。
3. 明日、会社で何がありますか。

**Answer key:**
1. エンジニアです。(He is an engineer.)
2. 同僚が手伝いました。(A coworker helped him.)
3. 大切な会議があります。(There is an important meeting.)

## 🇹🇼 Chinese Notes

- **働**: This kanji was invented in Japan (国字) and has no equivalent in standard Chinese hanzi — Chinese uses entirely different characters (工作/劳动) for "to work." Don't try to map it onto any Chinese character you already know.
- **仕事**: Chinese speakers might expect separate words for "job" (工作) and "task/matter" (事情), but Japanese 仕事 comfortably covers both meanings in one word — a broader single-word usage than the Chinese split.
- **主婦**: This transfers fairly well from Chinese 家庭主婦, but be careful not to confuse it with 主人（ご主人, Lesson 3）which means "husband," a related-looking but different word.

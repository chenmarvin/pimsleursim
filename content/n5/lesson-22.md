### Word-by-word details

**1. 貸す（かす）— 借出**
- 例文: 友達に本を貸します。
- 翻譯: 借書給朋友。
- Common mistake: Confusing 貸す (I lend, the object leaves me) with 借りる (I borrow, the object comes to me) — Chinese 借 covers both directions, but Japanese requires two completely different verbs.
- Memory tip: Think of loanwords like 貸款 (a loan) — money going *out* from you.

**2. 借りる（かりる）— 借入**
- 例文: 図書館で本を借ります。
- 翻譯: 在圖書館借書。
- Common mistake: Using 借りる when you actually mean to lend something — 借りる is always "I receive," never "I give."
- Memory tip: Pair 貸す／借りる together as opposite-direction verbs, the same way you memorized 行く／来る.

**3. 返す（かえす）— 歸還**
- 例文: 本を図書館に返します。
- 翻譯: 把書還給圖書館。
- Common mistake: Confusing 返す (to return an *object*, takes を) with 帰る (to return *home*, L6, never takes を) — related idea, completely different verbs.
- Memory tip: 返 contains 反 (turn back/reverse) — "reversing" something back to its owner.

**4. 手伝う（てつだう）— 幫忙**
- 例文: 母を手伝います。
- 翻譯: 幫忙媽媽。
- Common mistake: Confusing 手伝う (helping with everyday tasks/chores) with 助ける (#21, rescuing someone from real danger) — 手伝う is for routine help.
- Memory tip: 手 (hand) + 伝う (convey/pass along) = "lending a hand."

**5. 教える（おしえる）— 教**
- 例文: 先生は日本語を教えます。
- 翻譯: 老師教日文。
- Common mistake: Expecting 教える to mean only "teach in a classroom" — it equally means "to tell/inform" casually, e.g. 道を教えてください (please tell me the way).
- Memory tip: Same 教 kanji you'll formally learn below — also seen in 教室 (classroom, L16).

**6. 習う（ならう）— 學習**
- 例文: ピアノを習います。
- 翻譯: 學鋼琴。
- Common mistake: Confusing 習う (learning a skill *from an instructor*) with 勉強する (studying in general, L6, including self-study) — 習う implies being taught.
- Memory tip: Same 習 kanji as 復習する (review, L16) — the idea of practicing under guidance.

**7. 頼む（たのむ）— 拜託**
- 例文: 友達に手伝いを頼みます。
- 翻譯: 拜託朋友幫忙。
- Common mistake: Thinking 頼む only means "to order (food)" — that's one common use, but its core meaning is broader: asking someone for a favor or request.
- Memory tip: Shares the 頼 kanji with 頼る (#20) — both about depending on someone.

**8. お願いする（おねがいする）— 拜託**
- 例文: 先生にお願いします。
- 翻譯: 拜託老師。
- Common mistake: Treating お願いします and 頼む as equally blunt — お願いします is the softer, more universally polite version; 頼む can sound slightly more direct.
- Memory tip: You already know よろしくお願いします (Lesson 1) — this is that same 願う root used as a standalone verb.

**9. どうぞ（どうぞ）— 請、請用**
- 例文: どうぞ、こちらへ。
- 翻譯: 請往這邊。
- Common mistake: Confusing どうぞ (offering something outward, "here you go") with ください (#11, requesting something for yourself) — they point in opposite directions.
- Memory tip: どうぞ = your hand extending outward to offer; ください = your hand reaching in to request.

**10. どうも（どうも）— 謝謝、有點**
- 例文: どうも、失礼します。
- 翻譯: 謝謝，先告辭了。
- Common mistake: Overusing どうも as a full formal "thank you" in writing — it's a flexible, somewhat casual word (thanks / somewhat / very, depending on context), not a fixed replacement for ありがとうございます.
- Memory tip: A flexible, all-purpose filler — let context decide its exact meaning.

**11. ください（ください）— 請給我**
- 例文: 水をください。
- 翻譯: 請給我水。
- Common mistake: Dropping the object particle を before ください (fine in casual speech, but formally 水をください is the complete pattern) — many beginners skip it entirely by habit.
- Memory tip: Contrast with どうぞ (#9) — ください is what *you* say when requesting; どうぞ is what someone says when offering to *you*.

**12. 見せる（みせる）— 給看**
- 例文: 写真を見せてください。
- 翻譯: 請給我看照片。
- Common mistake: Confusing 見せる (to show *someone else*, transitive, involves another person) with 見る (to look/see, L6, doesn't require another person) — 見せる always implies showing to someone.
- Memory tip: 見せる = 見る + せる (a causative-like ending) = "make someone see" = show.

**13. 持つ（もつ）— 拿、持有**
- 例文: 傘を持っています。
- 翻譯: 拿著雨傘。
- Common mistake: Using 持つ where simple existence（〜があります, L6）would do — 持つ specifically implies actively holding or owning, not just something existing nearby.
- Memory tip: Same 持 kanji appears in 気持ち (feeling) — literally "holding a feeling."

**14. 持ってくる（もってくる）— 帶來**
- 例文: 明日、本を持ってきます。
- 翻譯: 明天把書帶來。
- Common mistake: Using 持ってくる for bringing a *person* — Japanese distinguishes bringing objects (持ってくる) from bringing people (連れてくる, a later-level word).
- Memory tip: 持つ (hold) + 来る (come) = "hold it and come" = bring.

**15. 思い出す（おもいだす）— 想起來**
- 例文: 名前を思い出しました。
- 翻譯: 想起名字了。
- Common mistake: Confusing 思い出す (a sudden, spontaneous recall of something forgotten) with 覚える (deliberate memorization) — different mental processes.
- Memory tip: 思う (think) + 出す (put out) = "a thought comes out" = recall.

**16. 決める（きめる）— 決定**
- 例文: 晩ご飯を決めます。
- 翻譯: 決定晚餐吃什麼。
- Common mistake: Forgetting 決める always needs an object marked with を ("I decide X") — don't drop the object the way a vaguer Chinese sentence might.
- Memory tip: Picture 氵(water) breaking through and settling on one course — a decisive resolution.

**17. 落とす（おとす）— 掉落**
- 例文: 財布を落としました。
- 翻譯: 錢包掉了。
- Common mistake: Forgetting 落とす always takes an object with を (you dropped *something*) — it describes your own action, not an event that just happens on its own.
- Memory tip: 落 (fall) + とす (a causative-like ending) = "make it fall" = drop.

**18. 渡す（わたす）— 交給**
- 例文: 手紙を渡します。
- 翻譯: 把信交給對方。
- Common mistake: Confusing 渡す (physically handing something directly to a person) with あげる (L19, the general "give" verb) — 渡す emphasizes the physical hand-to-hand act.
- Memory tip: The same 渡 appears in 渡る (to cross a street) — both involve something moving across to the other side.

**19. 交換する（こうかんする）— 交換**
- 例文: 電話番号を交換します。
- 翻譯: 交換電話號碼。
- Common mistake: Using 交換する for a simple one-way "give" — 交換する specifically implies a mutual swap where both sides give something.
- Memory tip: 交 (cross) + 換 (change/replace) = "cross-change" = exchange.

**20. 頼る（たよる）— 依靠**
- 例文: 家族に頼ります。
- 翻譯: 依靠家人。
- Common mistake: Confusing 頼る (ongoing reliance/dependence on someone) with 頼む (#7, a specific one-time request) — different scope of meaning.
- Memory tip: Shares the 頼 kanji with 頼む — both about leaning on someone's support.

**21. 助ける（たすける）— 幫助、救**
- 例文: 子供を助けます。
- 翻譯: 救孩子。
- Common mistake: Using 助ける for small everyday favors — it carries a stronger "rescue/save from real trouble" nuance compared to the lighter, routine 手伝う (#4).
- Memory tip: 助 contains 力 (strength/power) — think "rescue," a serious kind of help.

**22. 受け取る（うけとる）— 收到**
- 例文: プレゼントを受け取ります。
- 翻譯: 收到禮物。
- Common mistake: Treating 受け取る and もらう (L19) as fully interchangeable — 受け取る is more formal/procedural (mail, packages, payments), while もらう is the everyday general "receive."
- Memory tip: 受ける (receive/undergo) + 取る (take) = "receive by taking it."

**23. 配る（くばる）— 分發**
- 例文: 先生が本を配ります。
- 翻譯: 老師發書。
- Common mistake: Using 配る for giving something to just one person — 配る always implies distributing to a group of people.
- Memory tip: Shares the 配 root with 心配 (worry, later lesson) — literally "distributing" attention or items.

**24. 分ける（わける）— 分、分享**
- 例文: お菓子を分けます。
- 翻譯: 分點心。
- Common mistake: Confusing 分ける (わける, to divide/share) with 分かる (わかる, to understand) — same 分 kanji, completely different reading and meaning.
- Memory tip: Same 分 kanji as 半分 (half, L2) — the idea of "splitting" runs through both.

**25. 許す（ゆるす）— 原諒、允許**
- 例文: 遅刻を許してください。
- 翻譯: 請原諒我遲到。
- Common mistake: Only remembering the "forgive" meaning and missing its equally common "permit/allow" sense — 許す as a full verb covers granting permission too, not just pardoning a mistake.
- Memory tip: Shares a root with 許可 (permission, a higher-level word) — memorize as "the permit verb."

## ② Kanji (3): 教, 習, 持

**教 — Meaning: 教 (teach)**
- On-reading: キョウ
- Kun-reading: おし.える
- Words: 教える, 教室 (L16), 教科書 (preview きょうかしょ)
- Example sentence: 先生が日本語を教えます。
- Stroke order: 11 strokes — left component (耂 over 子, resembling 孝) first, then the right-side 攵 radical last.
- Chinese difference: Same character as Chinese 教 (jiào); the "teach" meaning matches closely, and both languages extend it to "religion" (宗教 in both). The adjustment is the native kun-reading おしえる, which has no equivalent single-syllable shift in Mandarin.

**習 — Meaning: 習 (learn, practice)**
- On-reading: シュウ
- Kun-reading: なら.う
- Words: 習う, 復習する (L16), 予習する (L16)
- Example sentence: ピアノを習います。
- Stroke order: 11 strokes — top 羽 (feathers, evoking a young bird practicing to fly) first, then 白 at the bottom.
- Chinese difference: Matches the Traditional Chinese form 習 used in Taiwan (not the Simplified 习) — this kanji transfers cleanly for Taiwanese learners, unlike some others that follow Simplified Chinese shapes.

**持 — Meaning: 持 (hold, have)**
- On-reading: ジ
- Kun-reading: も.つ
- Words: 持つ, 持ってくる, 気持ち (preview きもち)
- Example sentence: 傘を持っています。
- Stroke order: 9 strokes — left radical 扌 (hand, 3 strokes) first, then the right side 寺 (temple component); characters with 扌 always draw the hand radical first.
- Chinese difference: Same character as Chinese 持 (chí), and the "hold/maintain" meaning overlaps well (持續／持有 in Chinese, 持続 in Japanese). The real hurdle for Chinese speakers isn't meaning but conjugation: 持つ is a godan verb that becomes 持って in the te-form — Chinese verbs never conjugate at all.

## ③ Grammar (3 points)

### Grammar 1 — 〜てください (please do)
**Meaning:** A polite request or instruction asking someone to do something.
**Structure:** Verb て-form + ください
**Examples:**
1. ここに名前を書いてください。(Please write your name here.)
2. 早く来てください。(Please come early.)
3. 写真を見せてください。(Please show me the photo.)

**Common mistakes (Chinese speakers):**
- ❌ Attaching ください to the dictionary form (❌ 書くください) instead of the て-form (⭕ 書いてください) — ください always attaches to the て-form, never the plain form.
- ❌ Assuming てください is equally polite in every situation — it's a direct instruction, fine for teachers, service staff, or close relationships, but can sound too blunt toward a superior (more deferential forms exist at higher levels).

**Exercises (translate into Japanese):**
1. Please lend me the book.
2. Please teach me Japanese.
3. Please return this by tomorrow.

### Grammar 2 — 〜ています（進行, ongoing action）
**Meaning:** Describes an action currently in progress ("is doing").
**Structure:** Verb て-form + います
**Examples:**
1. 今、本を読んでいます。(I am reading a book right now.)
2. 友達と話しています。(I am talking with a friend.)
3. 今、ご飯を食べています。(I am eating a meal right now.)

**Common mistakes (Chinese speakers):**
- ❌ Using ています for every present-tense sentence the way Chinese loosely uses 在/正在 — some verbs use ています to describe a resulting *state* instead of an ongoing action (see Grammar 3); not every verb behaves the same way.
- ❌ Forgetting to convert to the て-form first (❌ 読むています instead of ⭕ 読んでいます).

**Exercises (translate into Japanese):**
1. I am studying Japanese now.
2. She is writing a letter now.
3. We are eating breakfast now.

### Grammar 3 — 〜ています（状態, resulting state）
**Meaning:** With verbs describing a change of state (結婚する, 住む, 持つ, etc.), ています describes the resulting *ongoing state* after the change happened — not an action still in progress.
**Structure:** Verb て-form + います (same form as Grammar 2, different meaning class)
**Examples:**
1. 傘を持っています。(I have/am holding an umbrella — a current state.)
2. 田中さんは結婚しています。(Mr. Tanaka is married — a state, not "in the middle of marrying.")
3. 台湾に住んでいます。(I live in Taiwan — a state.)

**Common mistakes (Chinese speakers):**
- ❌ Reading 結婚しています as "in the process of getting married" — for verbs like 結婚する, 住む, 持つ, ています expresses the state *after* the change, i.e. "already married," "already living there."
- ❌ Using plain 結婚します to mean "I am married" — that incorrectly implies "I will get married (in the future)"; the current married state requires ています.

**Exercises (translate into Japanese):**
1. I live in Taipei.
2. He is married.
3. I have (am holding) a book.

## 🇹🇼 Chinese Notes

- **貸す vs. 借りる**: Chinese 借 covers both directions — "我跟你借" (I borrow from you) and "我借給你" (I lend to you) both use the same character. Japanese has no single equivalent: 貸す is strictly "to lend" (giving something out) and 借りる is strictly "to borrow" (receiving something). Chinese speakers very commonly default to 借りる for both meanings and need to consciously check the direction each time.
- **伝 (in 手伝う)**: This is the shinjitai (post-war simplified) form of Traditional Chinese 傳 (chuán), which Taiwanese learners are used to writing in full. Seeing 伝 for the first time can look unfamiliar — treat it the same way you treat 学 as a simplified relative of Traditional 學.
- **どうも**: A flexible, context-dependent word meaning "thanks," "somewhat," or "very" depending on what follows — there is no single fixed Chinese word that maps onto it, so resist the urge to memorize one direct translation.

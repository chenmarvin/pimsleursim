# 🇯🇵 Japanese for Taiwanese Learners — JLPT N5, Lesson 3

**Objectives**

### Word-by-word details

**1. 家族（かぞく）— 家人**
- 例文: 私の家族は四人です。
- 翻譯: 我家有四個人。
- Common mistake: Using 家族 to refer to a single family member (❌ 私の家族は父です meaning "my family is my father") — 家族 means the whole family unit, not one person.
- Memory tip: 家 (house) + 族 (group/clan) = "the group of the house" = family.

**2. 父（ちち）— 爸爸（自己的）**
- 例文: 父は会社員です。
- 翻譯: 我爸爸是公司職員。
- Common mistake: Using 父 when talking TO your own father, or when referring to someone else's father — 父 is only for talking ABOUT your own father to others; use お父さん for someone else's father or when addressing your own.
- Memory tip: 父 alone (plain, no お) = humble reference to your own father, matching です-level politeness toward outsiders.

**3. 母（はは）— 媽媽（自己的）**
- 例文: 母は先生です。
- 翻譯: 我媽媽是老師。
- Common mistake: Same as 父 — using 母 to refer to someone else's mother is impolite; use お母さん instead.
- Memory tip: Pair 父・母 together as the "humble own-family" set, contrasted with お父さん・お母さん for others.

**4. 兄（あに）— 哥哥（自己的）**
- 例文: 兄は医者です。
- 翻譯: 我哥哥是醫生。
- Common mistake: Calling your own older brother 兄さん directly to his face when speaking humbly about him to a third party — use plain 兄 when talking about him to others, お兄さん when addressing him or referring to someone else's brother.
- Memory tip: Shorter/plainer word = "my own," longer お〜さん word = "someone else's or addressing directly."

**5. 姉（あね）— 姊姊（自己的）**
- 例文: 姉は大学生です。
- 翻譯: 我姊姊是大學生。
- Common mistake: Confusing 姉（あね, own older sister) with 姉さん・お姉さん (someone else's, or direct address) — same distinction pattern as 兄.
- Memory tip: Group 兄・姉 as "my older siblings," both irregular readings that must be memorized.

**6. 弟（おとうと）— 弟弟**
- 例文: 弟は学生です。
- 翻譯: 我弟弟是學生。
- Common mistake: Unlike 兄/姉, 弟 and 妹 (younger siblings) do NOT get an お〜さん honorific form when referring to someone else's younger sibling — you still just say 弟さん／妹さん, adding さん only, not お〜さん.
- Memory tip: Younger siblings are addressed more plainly in Japanese family-title logic — respect concentrates on elders.

**7. 妹（いもうと）— 妹妹**
- 例文: 妹はかわいいです。
- 翻譯: 我妹妹很可愛。
- Common mistake: Forgetting the reading いもうと and guessing an on-reChinese-based reading — this is a native kun-reading word with no relation to Chinese mèi sound.
- Memory tip: Pair 弟・妹 as "my younger siblings" — both use さん (not お〜さん) for someone else's.

**8. 両親（りょうしん）— 父母**
- 例文: 両親は台湾にいます。
- 翻譯: 我父母在台灣。
- Common mistake: Using 父母 (direct Chinese-order calque) instead of the correct Japanese word 両親 — 父母 as a compound is rarely used in everyday Japanese speech.
- Memory tip: 両 (both) + 親 (parent) = "both parents."

**9. 子供（こども）— 小孩**
- 例文: あの子供はかわいいです。
- 翻譯: 那個小孩很可愛。
- Common mistake: Using 子供 to mean "son/daughter" possessively without の (❌ 私子供 for "my child") — always needs 私の子供.
- Memory tip: 子 (child) + 供 (accompany/offer) = historically "children as a group"; now just means "child/children."

**10. 赤ちゃん（あかちゃん）— 嬰兒**
- 例文: 赤ちゃんは元気です。
- 翻譯: 嬰兒很健康。
- Common mistake: Confusing 赤ちゃん (baby) with 赤い (red, Lesson 11) just because they share the kanji 赤 — historically linked (babies were called "little red ones") but the meanings are distinct today.
- Memory tip: 赤 (red) + ちゃん (cute diminutive suffix) = "little red one" = baby.

**11. おじいさん — 爺爺、外公**
- 例文: おじいさんは元気です。
- 翻譯: 爺爺很健康。
- Common mistake: Confusing おじいさん (grandfather, long い) with おじさん (uncle, short じ) — a single vowel length changes the meaning entirely; a very common N5 listening trap.
- Memory tip: The longer word (おじいさん) refers to the "bigger" relation — grandfather is one generation further than uncle (おじさん).

**12. おばあさん — 奶奶、外婆**
- 例文: おばあさんの家は大きいです。
- 翻譯: 奶奶家很大。
- Common mistake: Same vowel-length trap as above — おばあさん (grandmother) vs. おばさん (aunt); mixing these up is one of the most common N5 listening mistakes for all learners, not just Chinese speakers.
- Memory tip: Long vowel = older generation (grandmother); short = same generation as your parents (aunt).

**13. おじさん — 叔叔、伯伯**
- 例文: おじさんは会社員です。
- 翻譯: 叔叔是公司職員。
- Common mistake: See おじいさん note — do not lengthen the じ sound, or it becomes "grandfather" instead of "uncle."
- Memory tip: Short word, same generation as parents = uncle.

**14. おばさん — 阿姨、姑姑**
- 例文: おばさんは料理が上手です。（料理は後の課で詳しく）
- 翻譯: 阿姨很會做菜。
- Common mistake: Same vowel-length trap as おばあさん — keep it short for "aunt."
- Memory tip: Short word, same generation as parents = aunt (parallels おじさん = uncle).

**15. 兄弟（きょうだい）— 兄弟姊妹**
- 例文: 兄弟がいますか。
- 翻譯: 你有兄弟姊妹嗎？
- Common mistake: Assuming 兄弟 refers only to brothers (based on the kanji 兄+弟) — in modern Japanese it commonly means "siblings" in general, including sisters, when asking casually.
- Memory tip: 兄 (older brother) + 弟 (younger brother) as a compound broadened in meaning to cover all siblings.

**16. 一人っ子（ひとりっこ）— 獨生子女**
- 例文: 私は一人っ子です。
- 翻譯: 我是獨生子女。
- Common mistake: Trying to build this word from 一人 + 子 without the small っ (a Japanese sound-doubling convention) — omitting it changes the pronunciation and sounds unnatural.
- Memory tip: 一人 (one person) + っ子 (child suffix) = "an only child."

**17. 結婚する（けっこんする）— 結婚**
- 例文: 兄は結婚しています。
- 翻譯: 我哥哥已經結婚了。
- Common mistake: Using 結婚です instead of 結婚しています to describe someone's current marital status — the ongoing state "is married" needs the ている form, not です alone (this ている grammar point comes formally later, but recognize the pattern now).
- Memory tip: 結 (tie/connect) + 婚 (marriage) = "to tie the marriage knot."

**18. 独身（どくしん）— 單身**
- 例文: 姉は独身です。
- 翻譯: 我姊姊單身。
- Common mistake: Confusing 独身 (unmarried status) with 一人 (alone/one person) — 独身 specifically describes marital status, not just being physically alone.
- Memory tip: 独 (alone) + 身 (body/self) = "alone in oneself" = single/unmarried.

**19. 家庭（かてい）— 家庭**
- 例文: 幸せな家庭です。（幸せ = happy, extra word for context）
- 翻譯: 是幸福的家庭。
- Common mistake: Confusing 家庭 (household/home life, an abstract concept) with 家族 (the family members themselves) — 家庭 describes the home environment, not the people directly.
- Memory tip: 家 (house) + 庭 (garden/yard) = the whole "home" including its environment, not just the people.

**20. 奥さん（おくさん）— 太太（別人的）**
- 例文: 田中さんの奥さんはきれいです。
- 翻譯: 田中先生的太太很漂亮。
- Common mistake: Using 奥さん to refer to your OWN wife when speaking to others — 奥さん is respectful language reserved for someone else's wife; you'd use 妻（つま）for your own.
- Memory tip: 奥 (inner/deep part of the house) + さん = a respectful term picturing the wife as "mistress of the household."

**21. ご主人（ごしゅじん）— 先生（別人的丈夫）**
- 例文: 山田さんのご主人は医者です。
- 翻譯: 山田太太的先生是醫生。
- Common mistake: Using ご主人 for your own husband when speaking to others (should use 夫/おっと) — the ご〜 honorific prefix marks it as reserved for someone else's husband, mirroring 奥さん.
- Memory tip: ご (honorific) + 主人 (master/head of household) = respectful term for someone else's husband.

**22. 年上（としうえ）— 年長**
- 例文: 兄は私より年上です。
- 翻譯: 哥哥比我年長。
- Common mistake: Word order errors copying English/Chinese comparative structure — Japanese needs 〜より (than) placed directly before 年上, not after the whole sentence.
- Memory tip: 年 (age/year) + 上 (above/up) = "above in age" = older.

**23. 年下（としした）— 年幼**
- 例文: 妹は私より年下です。
- 翻譯: 妹妹比我年幼。
- Common mistake: Confusing 年下 (younger in age) with 年上 (older in age) since they share the first kanji — remember 上=up=older, 下=down=younger.
- Memory tip: 年 (age) + 下 (below/down) = "below in age" = younger.

**24. 似ている（にている）— 相似**
- 例文: 私は母に似ています。
- 翻譯: 我長得像媽媽。
- Common mistake: Using を instead of に with 似ている — this verb takes the particle に (母に似ている), not を, unlike many other transitive-feeling verbs.
- Memory tip: Memorize the fixed phrase "Aに似ている" as one chunk — "resembles A."

**25. かわいい — 可愛**
- 例文: 赤ちゃんはかわいいです。
- 翻譯: 嬰兒很可愛。
- Common mistake: Treating かわいい like a na-adjective and adding な before a noun (❌ かわいいな赤ちゃん) — it's an i-adjective, so it attaches directly (かわいい赤ちゃん), no な needed.
- Memory tip: Ends in い like other i-adjectives (大きい、新しい) — a reliable pattern to identify this adjective class.

## ② Kanji (6): 父, 母, 子, 男, 女, 友

**父** — Meaning: 父 (father). On: フ. Kun: ちち. Words: 父, お父さん, 父親（ちちおや）. Example: 父は先生です。 Stroke order: two short diagonal strokes crossing near the top, then two more diagonal strokes spreading downward like legs. Chinese difference: Identical to Chinese 父; meaning and general usage overlap closely, but Japanese distinguishes plain 父 (own father, humble) from お父さん (address form/someone else's father) — Chinese 父 doesn't carry this same politeness split.

**母** — Meaning: 母 (mother). On: ボ. Kun: はは. Words: 母, お母さん, 母親（ははおや）. Example: 母は元気です。 Stroke order: outer curved frame first, then the two dots inside. Chinese difference: Identical to Chinese 母; same politeness-split note as 父 applies — 母 (plain) vs. お母さん (respectful/address form).

**子** — Meaning: 子 (child). On: シ. Kun: こ. Words: 子供, 一人っ子, 男の子（おとこのこ）. Example: あの子はかわいいです。 Stroke order: top hook stroke first, then the horizontal stroke crossing it, then the final upward-curving stroke. Chinese difference: Identical to Chinese 子; Japanese frequently uses 子 alone (こ) to mean "child/kid" colloquially, more often than standalone 子 is used this way in Mandarin.

**男** — Meaning: 男 (male). On: ダン、ナン. Kun: おとこ. Words: 男の子（おとこのこ）, 男性（preview）, 男の人. Example: あの男の人は先生です。 Stroke order: write 田 (field) on top first, then 力 (strength) beneath it. Chinese difference: Identical to Chinese 男, and the "field + strength" composition logic is the same in both languages — a straightforward, low-risk kanji.

**女** — Meaning: 女 (female). On: ジョ、ニョ. Kun: おんな. Words: 女の子（おんなのこ）, 女性（preview）, 女の人. Example: あの女の人は医者です。 Stroke order: the crossing diagonal strokes first, then the horizontal stroke through the middle. Chinese difference: Identical to Chinese 女; meaning fully overlaps, low risk of confusion.

**友** — Meaning: 友 (friend). On: ユウ. Kun: とも. Words: 友達（ともだち）, 友人（ゆうじん, preview）. Example: 友達と話します。（話す from Lesson 6, preview） Stroke order: top-left diagonal stroke, then the horizontal stroke, then the final hooked stroke on the right. Chinese difference: Identical to Chinese 友; both languages use 友 as the root of "friend" compounds, so intuition transfers directly.

## ③ Grammar (2 points)

### Grammar 1 — これ・それ・あれ・どれ (this/that/that over there/which one)
**Meaning:** Pronouns for pointing at objects without naming them, based on distance from speaker/listener: これ (near speaker), それ (near listener), あれ (far from both), どれ (which one, question).
**Structure:** これ／それ／あれ／どれ + は + Noun/です
**Examples:**
1. これは私の財布です。(This is my wallet.)
2. それはあなたの本ですか。(Is that your book?)
3. あれは兄の車です。（車 is preview vocab） (That over there is my brother's car.)

**Common mistakes (Chinese speakers):**
- ❌ Using これ／それ／あれ as if they directly modify a noun like Chinese 這／那 can loosely do (❌ これ人) — these words stand alone as full pronouns; to modify a noun directly, use この／その／あの (Grammar 2 below) instead.
- ❌ Confusing それ (near listener) and あれ (far from both) — Chinese 那 covers both cases, but Japanese requires picking the correct one based on distance from the listener specifically.

**Exercises (translate into Japanese):**
1. This is my father's wallet.
2. Is that (near you) your mother's phone number?
3. Which one is your child?

### Grammar 2 — この・その・あの・どの＋Noun (this/that/which + noun)
**Meaning:** Demonstrative adjectives that must directly precede and modify a noun (cannot stand alone).
**Structure:** この／その／あの／どの + Noun
**Examples:**
1. この子は私の妹です。(This child is my younger sister.)
2. その人は誰ですか。（誰 = who, preview） (Who is that person [near you]?)
3. あの家族は大きいです。(That family [over there] is big.)

**Common mistakes (Chinese speakers):**
- ❌ Using これ／それ／あれ directly before a noun (❌ これ子は妹です) → ⭕ この子は妹です — これ etc. cannot modify a noun directly; only この／その／あの／どの can.
- ❌ Forgetting どの requires a noun immediately after it, unlike どれ which stands alone (どの人ですか vs. どれですか).

**Exercises (translate into Japanese):**
1. This child is cute.
2. That person (near you) is my older brother.
3. Which family (of the choices) is yours?

## 🇹🇼 Chinese Notes

- **父／母 vs. お父さん／お母さん**: Chinese 爸爸/媽媽 works the same way regardless of whose father/mother you're discussing. Japanese requires a politeness split: plain 父／母 only when referring to your OWN parent while speaking to an outsider, and お父さん／お母さん both when addressing your own parent directly AND when referring to someone else's parent. This dual-purpose honorific form has no direct one-to-one match in Chinese.
- **奥さん／ご主人**: These are honorific terms used only for someone ELSE's spouse — never your own. A Chinese speaker used to referring to "我太太/我先生" the same way regardless of audience should be careful: for your own spouse, Japanese uses 妻（つま）/夫（おっと）instead.
- **兄弟（きょうだい）**: Literally built from "older brother + younger brother," but in casual modern usage it broadly means "siblings" (including sisters) when asking 兄弟がいますか — a broader sense than the literal kanji might suggest to a Chinese reader used to 兄弟 meaning specifically "brothers."

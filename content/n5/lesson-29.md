### Word-by-word details

**1. 食堂（しょくどう）— 食堂、餐廳**
- 例文: 会社の食堂で昼ご飯を食べます。
- 翻譯: 在公司的食堂吃午餐。
- Common mistake: Using 食堂 and レストラン (Lesson 7) interchangeably — 食堂 usually implies a simple, casual cafeteria-style eatery, while レストラン suggests a proper sit-down restaurant.
- Memory tip: 食（eat）＋堂（hall）＝ "eating hall."

**2. メニュー — 菜單**
- 例文: メニューを見せてください。
- 翻譯: 請給我看菜單。
- Common mistake: Confusing with 献立（こんだて）, a more formal/native word for "menu/meal plan" not covered in this course — メニュー (the loanword) is the standard everyday choice.
- Memory tip: Katakana loanword from English "menu."

**3. 注文する（ちゅうもんする）— 點餐**
- 例文: ラーメンを注文します。
- 翻譯: 點拉麵。
- Common mistake: Using 予約する (Lesson 26, "to reserve") when 注文する (to order food/goods) is meant — different actions entirely.
- Memory tip: 注（pour in/request）＋文（statement）＝ "make a request statement" = to order.

**4. 席（せき）— 座位**
- 例文: 窓側の席をお願いします。
- 翻譯: 麻煩給我靠窗的座位。
- Common mistake: Confusing with 石（いし, "stone," Lesson 24) due to a similar-sounding start — different kanji, unrelated meaning.
- Memory tip: Pairs with 満席 (#22, "fully booked" = "seats full").

**5. お勘定（おかんじょう）— 帳單、結帳**
- 例文: お勘定をお願いします。
- 翻譯: 麻煩結帳。
- Common mistake: Using お会計（おかいけい, a synonym not covered here) or literally translating "bill" — お勘定 is the standard spoken phrase for asking to pay at a restaurant.
- Memory tip: A fixed phrase to memorize as a whole: お勘定をお願いします = "check, please."

**6. 領収書（りょうしゅうしょ）— 收據**
- 例文: 領収書をください。
- 翻譯: 請給我收據。
- Common mistake: Confusing with レシート (Lesson 11, a simple store receipt) — 領収書 is a more formal receipt, often needed for business expense reports, sometimes requiring a name to be written on it.
- Memory tip: 領収（receive）＋書（document）＝ "a document proving receipt [of payment]."

**7. いただきます — （用餐前）我開動了**
- 例文: 「いただきます」と言ってから食べます。
- 翻譯: 說完「我開動了」之後開始吃。
- Common mistake: Skipping this phrase entirely because Chinese dining customs have no exact equivalent — in Japan it's considered essential etiquette before any meal, not optional politeness.
- Memory tip: Literally "I humbly receive [this food]" — always said before the first bite.

**8. ごちそうさま — （用餐後）謝謝款待**
- 例文: 食べ終わったら「ごちそうさま」と言います。
- 翻譯: 吃完之後說「謝謝款待」。
- Common mistake: Forgetting to pair this with いただきます (#7) — they form a matched before/after set that Taiwanese learners often only learn half of.
- Memory tip: 馳走（ちそう, a feast/treat）＋さま（polite suffix）＝ "thank you for the treat."

**9. 辛い（からい）— 辣的**
- 例文: このカレーは辛いです。
- 翻譯: 這個咖哩很辣。
- Common mistake: Confusing with 甘い (#10, "sweet") when speaking quickly — opposite tastes, easy to swap under pressure.
- Memory tip: Pairs as an opposite with 甘い (#10) — learn the pair together.

**10. 甘い（あまい）— 甜的**
- 例文: このデザートは甘いです。
- 翻譯: 這個甜點很甜。
- Common mistake: Using 甘い to describe temperature — 甘い is exclusively about taste (sweetness), unrelated to 冷たい／熱い (#11/#12).
- Memory tip: Pairs as an opposite with 辛い (#9).

**11. 冷たい（つめたい）— 冰的、冷的（觸感）**
- 例文: 冷たい水をください。
- 翻譯: 請給我冰水。
- Common mistake: Confusing with 寒い (さむい, "cold weather/air temperature," Lesson 8) — 冷たい is for touchable/drinkable things (water, hands, food), while 寒い is for ambient air temperature.
- Memory tip: Pairs as an opposite with 熱い (#12).

**12. 熱い（あつい）— 燙的（觸感）**
- 例文: 熱いお茶を飲みます。
- 翻譯: 喝熱茶。
- Common mistake: Confusing with 暑い (あつい, "hot weather," Lesson 8) — same pronunciation, different kanji; 熱い is for touchable things (drinks, objects), 暑い is for air temperature.
- Memory tip: Pairs as an opposite with 冷たい (#11); same sound as 暑い but different kanji/context.

**13. 飲み物（のみもの）— 飲料**
- 例文: お飲み物は何にしますか。
- 翻譯: 飲料要點什麼？
- Common mistake: Confusing with 食べ物 (#14) — easy word-pair to mix up quickly in conversation.
- Memory tip: 飲む（drink, Lesson 6）＋物（thing）＝ "drinking thing."

**14. 食べ物（たべもの）— 食物**
- 例文: 好きな食べ物は何ですか。
- 翻譯: 你喜歡的食物是什麼？
- Common mistake: See #13 — easy to swap 飲み物 and 食べ物 under time pressure.
- Memory tip: 食べる（eat, Lesson 6）＋物（thing）＝ "eating thing."

**15. デザート — 甜點**
- 例文: デザートは何がありますか。
- 翻譯: 有什麼甜點？
- Common mistake: Confusing with お菓子 (Lesson 9, "snack/sweets" in general) — デザート specifically refers to a course served after a meal.
- Memory tip: Katakana loanword from English "dessert."

**16. セット — 套餐**
- 例文: Aセットをお願いします。
- 翻譯: 麻煩給我A套餐。
- Common mistake: Confusing with 定食 (#20) — セット often refers to a Western-style combo (like a burger + drink), while 定食 specifically implies a traditional Japanese set meal.
- Memory tip: Katakana loanword from English "set."

**17. カウンター — 櫃檯**
- 例文: カウンター席に座ります。
- 翻譯: 坐在櫃檯座位。
- Common mistake: Confusing with レジ (a cash register/checkout counter, not covered in this course) — カウンター in a restaurant context usually refers to bar-style seating facing the kitchen.
- Memory tip: Katakana loanword from English "counter."

**18. おすすめ — 推薦**
- 例文: おすすめは何ですか。
- 翻譯: 有什麼推薦的嗎？
- Common mistake: Treating おすすめ only as a noun — it's frequently used as a standalone question (おすすめは？) when ordering, functioning almost like "what's good here?"
- Memory tip: From the verb 勧める (すすめる, "to recommend") — お + noun form = "the recommendation."

**19. 持ち帰り（もちかえり）— 外帶**
- 例文: 持ち帰りでお願いします。
- 翻譯: 麻煩幫我外帶。
- Common mistake: Using テイクアウト (a common loanword alternative) exclusively and not recognizing 持ち帰り as an equally standard, often more traditional option at Japanese-style restaurants.
- Memory tip: 持つ（hold, Lesson 22）＋帰る（return home, Lesson 6）＝ "hold and go home with" = takeout.

**20. 定食（ていしょく）— 定食（日式套餐）**
- 例文: 焼き魚定食を注文しました。
- 翻譯: 點了烤魚定食。
- Common mistake: Confusing with セット (#16) — see #16's note; 定食 specifically implies a Japanese-style fixed set meal (rice, soup, main dish, pickles).
- Memory tip: 定（fixed）＋食（meal）＝ "a fixed meal."

**21. 割り勘（わりかん）— 各付各的、平分帳單**
- 例文: 今日は割り勘にしましょう。
- 翻譯: 今天各付各的吧。
- Common mistake: Assuming this is always the default in Japan the way it might be among friends elsewhere — treating (おごる, not covered here) also happens, so 割り勘 is a specific choice, not a universal default.
- Memory tip: 割る（divide）＋勘（account/reckoning）＝ "divide the account."

**22. 満席（まんせき）— 客滿**
- 例文: すみません、今、満席です。
- 翻譯: 不好意思，現在客滿了。
- Common mistake: Confusing with 混雑する (#25, "to be crowded" in general) — 満席 specifically means every seat is taken (no seats available at all), a stronger statement than "crowded."
- Memory tip: 満（full）＋席（seat）＝ "seats full."

**23. 味（あじ）— 味道**
- 例文: この料理は味がいいです。
- 翻譯: 這道菜味道很好。
- Common mistake: Confusing with 味噌（みそ, "miso," not covered in this course) due to the shared 味 kanji — different words, unrelated in this context.
- Memory tip: A single kanji, standalone noun meaning "taste/flavor."

**24. 量（りょう）— 份量**
- 例文: この店は量が多いです。
- 翻譯: 這家店的份量很多。
- Common mistake: Confusing with 料 (りょう, "fee/materials," seen in 料理) — different kanji, easy to visually confuse since both are read りょう.
- Memory tip: A single kanji, standalone noun meaning "amount/quantity."

**25. 混雑する（こんざつする）— 擁擠、混雑**
- 例文: 週末はいつも混雑しています。
- 翻譯: 週末總是很擁擠。
- Common mistake: Confusing with 混んでいる (Lesson 17, "crowded," a more casual everyday phrase) — 混雑する is the more formal/written equivalent; both describe the same idea of crowdedness.
- Memory tip: 混（mix/confuse）＋雑（mixed/miscellaneous）＝ "mixed together" = crowded.

## ② Kanji (3): 料, 理, 好

**料 — Meaning: 料 (fee, materials)**
- On-reading: リョウ
- Kun-reading: —
- Words: 料理, 給料 (Lesson 28)
- Example sentence: この料理はおいしいです。
- Stroke order: 10 strokes — 米 (rice, 6 strokes) on the left, then 斗 (dipper, 4 strokes) on the right.
- Chinese difference: Same character as Chinese 料 (as in 材料, 料理); the meaning largely overlaps, and 料理 as "cuisine/cooking" matches Chinese 料理 exactly — a reassuring direct match.

**理 — Meaning: 理 (reason, logic)**
- On-reading: リ
- Kun-reading: —
- Words: 料理, 理由 (Lesson 23)
- Example sentence: 日本料理が好きです。
- Stroke order: 11 strokes — 王 (jade radical, 4 strokes) on the left, then 里 (village, 7 strokes) on the right.
- Chinese difference: Same character and core meaning as Chinese 理 (reason/logic/manage) — very transparent for Taiwanese learners; 料理 as "cooking" matches Chinese usage directly as well.

**好 — Meaning: 好 (like, favorable)**
- On-reading: コウ
- Kun-reading: す.き
- Words: 好きな (Lesson 8), 料理人 + 好き combinations
- Example sentence: すき焼きが好きです。
- Stroke order: 6 strokes — 女 (woman, 3 strokes) on the left, 子 (child, 3 strokes) on the right.
- Chinese difference: Same character and core meaning as Chinese 好 (good/to like) — extremely transparent. Note that Japanese mainly uses 好き as a na-adjective ("likable/pleasing"), a different grammatical role than Chinese 好, which functions as a plain adjective ("good"); so 好きです literally structures as "[X] is pleasing [to me]," not "[I] like [X]."

## ③ Grammar

### Grammar 1 — 〜そうです（様態：appearance / "looks like"）
**Meaning:** Describes how something appears based on visual impression, before actually confirming it.
**Structure:** い-adjective(drop い) ＋ そうです ／ な-adjective(stem) ＋ そうです ／ V(ます-stem) ＋ そうです
**Examples:**
1. このラーメンはおいしそうです。(This ramen looks delicious.)
2. この料理は辛そうです。(This dish looks spicy.)
3. 雨が降りそうです。(It looks like it's going to rain.)

**Common mistakes (Chinese speakers):**
- Treating いい irregularly — the correct appearance form is よさそうです, not ❌ いいそうです.
- Attaching そうです to the full adjective with い still attached in the "looks like" sense (❌ おいしいそうです here would instead mean "I heard it's delicious," a different, hearsay meaning covered in Lesson 37) — for appearance, drop the final い first.

**Exercises (translate into Japanese):**
1. This cake looks sweet.
2. It looks like it will be crowded today.

### Grammar 2 — 〜たことがあります (have done ... before)
**Meaning:** Describes a past experience, something the speaker has done at least once.
**Structure:** V(た-form) ＋ ことがあります
**Examples:**
1. 日本料理を食べたことがあります。(I have eaten Japanese food before.)
2. この店で注文したことがあります。(I have ordered at this restaurant before.)
3. 一人でレストランに行ったことがありません。(I have never gone to a restaurant alone.)

**Common mistakes (Chinese speakers):**
- Using the dictionary form or ます-form instead of the た-form (❌ 食べることがあります, which instead means "there are times when I eat," a habitual meaning, not an experience).
- Misplacing the negative (❌ 食べたことがないあります → ⭕ 食べたことがありません／ないです).

**Exercises (translate into Japanese):**
1. I have tried natto before.
2. I have never ordered this dish.

### Grammar 3 — 〜ないでください (please don't do)
**Meaning:** A polite request asking someone not to do something.
**Structure:** V(ない-form) ＋ でください
**Examples:**
1. ここで写真を撮らないでください。(Please don't take photos here.)
2. 熱いものをすぐに食べないでください。(Please don't eat hot things right away.)
3. 大きい声で話さないでください。(Please don't speak in a loud voice.)

**Common mistakes (Chinese speakers):**
- Confusing 〜ないでください (a polite personal request) with 〜てはいけません (Lesson 23, a stronger rule/prohibition) — 〜ないでください is softer, appropriate for a restaurant table request, while 〜てはいけません sounds like an official rule.
- Misforming the ない-form of irregular verbs — する becomes しないでください, not ❌ するないでください.

**Exercises (translate into Japanese):**
1. Please don't take photos here.
2. Please don't eat this.

## ④ Reading

今日は友達とレストランへ行きました。メニューを見て、料理を注文しました。友達は「この料理は辛そうです」と言いました。私は前に辛い料理を食べたことがあります。でも、甘いものは苦手です。店員に「デザートを持ってこないでください」とお願いしました。料理はとても美味しかったです。

**Comprehension questions:**
1. 今日、どこへ行きましたか。
2. 店員に何をお願いしましたか。
3. 料理はどうでしたか。

**Answer key:**
1. レストランへ行きました。(They went to a restaurant.)
2. デザートを持ってこないでくださいとお願いしました。(They asked the waiter not to bring dessert.)
3. とても美味しかったです。(It was very delicious.)

## 🇹🇼 Chinese Notes

- **好き**: In Chinese, 好 functions as a plain adjective ("good"), but in Japanese 好き is a na-adjective meaning "likable/pleasing to someone." 私は寿司が好きです literally structures as "sushi is pleasing to me," not a direct word-for-word match with Chinese 我喜歡.
- **料理**: This one transfers cleanly from Chinese 料理 (cuisine/cooking) — a reassuring direct match, unlike many other words in this lesson.
- **いただきます／ごちそうさま**: These set phrases have no single-word Chinese equivalent. Taiwanese learners sometimes skip saying them or translate them awkwardly, but they are essential, near-mandatory etiquette phrases before and after any meal in Japan, unlike Chinese dining customs where no fixed phrase is expected.

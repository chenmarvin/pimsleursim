### Vocabulary in Depth

**1. 買い物（かいもの）— shopping**
- Example: 週末、デパートへ買い物に行きます。
- Translation: 週末我要去百貨公司購物。
- Common mistake: Learners often say 買い物します as if 買い物 could take an object directly, forgetting it's already a noun meaning "the act of shopping" — you go 買い物に行く, you don't "買い物 something."
- Memory tip: 買（buy）+ 物（thing）= literally "buying things." The kanji 買 looks like Chinese 買, an easy anchor.

**2. 店（みせ）— shop**
- Example: この店は九時に開きます。
- Translation: 這家店九點開門。
- Common mistake: Confusing 店（みせ, a physical shop）with 店員（てんいん, the clerk who works there) — 店に行きます, not 店員に行きます.
- Memory tip: Same character as Chinese 店 (shop) — meaning is nearly identical.

**3. 色（いろ）— color**
- Example: どんな色が好きですか。
- Translation: 你喜歡什麼顏色？
- Common mistake: Saying 何色 as なにいろ instead of なにいろ is actually correct, but learners often mispronounce it as なんしょく by analogy with 何色（なんしょく, used in more technical contexts) — for everyday speech なにいろ is standard.
- Memory tip: 色 alone (no い ending) is a plain noun — don't confuse it with the adjectives below that end in い.

**4. 赤い（あかい）— red**
- Example: 赤いかばんを買いました。
- Translation: 我買了紅色的包包。
- Common mistake: Forgetting 赤い is an i-adjective and trying to attach な, saying ❌ 赤いなかばん instead of ⭕ 赤いかばん.
- Memory tip: 赤 resembles Chinese 赤 (also meaning red/bare), so the meaning transfers directly — just remember the い ending when used before a noun.

**5. 青い（あおい）— blue**
- Example: 空は青いです。
- Translation: 天空是藍色的。
- Common mistake: 青い in Japanese also covers some shades Chinese speakers would call green (e.g. 青信号 "green traffic light"), so learners assume it always means exactly 藍色 — it's a slightly broader color term than in Chinese.
- Memory tip: Picture the sky (空) — 青い空 is a very common, easy-to-remember pairing.

**6. 白い（しろい）— white**
- Example: 白いシャツが欲しいです。
- Translation: 我想要一件白色的襯衫。
- Common mistake: Confusing 白い (white, adjective) with 白（はく, used in compounds like 白人) — as a standalone adjective before a noun it must keep the い.
- Memory tip: 白 is identical to Chinese 白 (white) — no surprises in meaning.

**7. 黒い（くろい）— black**
- Example: 黒い靴を試着したいです。
- Translation: 我想試穿黑色的鞋子。
- Common mistake: Mixing up 黒い (くろい, black) with 暗い (くらい, dark) because both start with く and relate to darkness — they describe different things (color vs. brightness).
- Memory tip: 黒 looks similar to Chinese 黑 — the top part differs slightly, but the meaning is identical.

**8. 黄色い（きいろい）— yellow**
- Example: 黄色いレインコートを買いました。
- Translation: 我買了一件黃色的雨衣。
- Common mistake: Unlike 赤い/青い/白い/黒い, this color word is built from a noun + い (黄色 + い), so learners sometimes drop the 色 and just say ❌ 黄い, which is not standard.
- Memory tip: Remember it as 黄色（yellow-color）+ い, unlike the other four basic colors which are pure i-adjectives.

**9. クレジットカード — credit card**
- Example: クレジットカードで支払います。
- Translation: 我要用信用卡付款。
- Common mistake: Using the particle を instead of で when saying "pay by/with credit card" — the means/instrument particle is で, not を.
- Memory tip: A long katakana loanword from English "credit card" — say it slowly syllable by syllable: ku-re-ji-tto-kaa-do.

**10. レシート — receipt**
- Example: レシートをもらいましたか。
- Translation: 你拿到收據了嗎？
- Common mistake: Confusing レシート (receipt/proof of payment) with 切符 (きっぷ, ticket) — different loanwords for different paper slips.
- Memory tip: From English "receipt" — pronounced re-shii-to.

**11. プレゼント — present**
- Example: これは友達へのプレゼントです。
- Translation: 這是給朋友的禮物。
- Common mistake: Using に instead of へ/に correctly is fine, but learners often forget the の linking プレゼント to its recipient: 友達へのプレゼント, not ❌ 友達へプレゼント when used as a noun phrase.
- Memory tip: From English "present" — same meaning, easy loanword.

**12. セール — sale**
- Example: 今、セールをしています。
- Translation: 現在正在特賣。
- Common mistake: Confusing セール (a sale/discount event) with 販売 (はんばい, to sell/selling as a business activity).
- Memory tip: From English "sale" — think of a "SALE" sign in a shop window.

**13. 店員（てんいん）— shop clerk**
- Example: 店員に聞いてください。
- Translation: 請問店員。
- Common mistake: Confusing 店員 (clerk) with お客さん (customer) — the two are opposite roles in the same sentence pattern.
- Memory tip: 店（shop）+ 員（member/staff）= "shop staff member."

**14. お客さん（おきゃくさん）— customer**
- Example: お客さん、こちらへどうぞ。
- Translation: 客人，這邊請。
- Common mistake: Dropping the honorific お and さん and just saying ❌ 客, which sounds blunt and is rarely used to address someone directly.
- Memory tip: 客 resembles Chinese 客 (guest); お...さん wraps it in politeness, common for addressing customers in service settings.

**15. サイズ — size**
- Example: このサイズは大きいです。
- Translation: 這個尺寸太大了。
- Common mistake: Using 大きさ (おおきさ, general size/magnitude) interchangeably with サイズ (specific clothing/product size) — they overlap but サイズ is the standard shopping term.
- Memory tip: From English "size" — pronounced sa-i-zu.

**16. 値段（ねだん）— price**
- Example: この靴の値段はいくらですか。
- Translation: 這雙鞋子的價格是多少？
- Common mistake: Using お金（money）where 値段（price of a specific item）is meant — 値段 asks about cost, not money itself.
- Memory tip: 値（value）+ 段（level/step）= "the value level," i.e. price.

**17. 割引（わりびき）— discount**
- Example: 学生は割引があります。
- Translation: 學生有折扣。
- Common mistake: Confusing 割引（わりびき, a discount amount/deal）with セール（a sale event) — 割引 can exist even without a formal セール.
- Memory tip: 割（to divide/split）+ 引（to subtract）= "splitting off and subtracting," i.e., cutting the price.

**18. 試着する（しちゃくする）— to try on**
- Example: このシャツを試着してもいいですか。
- Translation: 我可以試穿這件襯衫嗎？
- Common mistake: Using 着る（きる, simply "to wear"）when 試着する（specifically "to try on before buying"）is meant.
- Memory tip: 試（try/test）+ 着（wear）= "trial-wear."

**19. 包む（つつむ）— to wrap**
- Example: プレゼントを包んでください。
- Translation: 請幫我包裝禮物。
- Common mistake: Confusing the transitive 包む (to wrap something) with intransitive-feeling phrasing — remember it takes を: プレゼントを包む.
- Memory tip: Picture wrapping paper folding around (包) a box — the kanji itself looks like something enclosed.

**20. 袋（ふくろ）— bag**
- Example: 袋はいりますか。
- Translation: 需要袋子嗎？
- Common mistake: Confusing 袋（ふくろ, a shopping/plastic bag）with かばん（a bag/handbag you carry belongings in long-term).
- Memory tip: Common phrase at any Japanese register: 袋はいりますか — memorize it as a set phrase.

**21. 欲しい（ほしい）— want (a thing)**
- Example: 新しい財布が欲しいです。
- Translation: 我想要一個新錢包。
- Common mistake: Using を instead of が with 欲しい — the desired object takes が, not を: 財布が欲しい, not ❌ 財布を欲しい.
- Memory tip: 欲しい is only for wanting things (nouns); wanting to DO something uses 〜たいです instead (see Lesson 19 grammar).

**22. 売る（うる）— to sell**
- Example: この店は靴を売っています。
- Translation: 這家店在賣鞋子。
- Common mistake: Confusing 売る（to sell）with 買う（to buy）— visually and phonetically distinct but often swapped by beginners under time pressure.
- Memory tip: 売 and 買 look similar but are different characters; associate 売 with "out" (goods going out) and 買 with "in" (goods coming in, like a shell 貝 = money in old China).

**23. 選ぶ（えらぶ）— to choose**
- Example: どれを選びますか。
- Translation: 你要選哪一個？
- Common mistake: Using 決める（きめる, to decide）interchangeably with 選ぶ（to choose among options）— 選ぶ emphasizes picking from a set, 決める emphasizes finalizing.
- Memory tip: Pair with どれ (which one) from Lesson 3's これ・それ・あれ・どれ series: どれを選びますか.

**24. 支払う（しはらう）— to pay**
- Example: 現金で支払います。
- Translation: 我要用現金付款。
- Common mistake: Using お金をあげる (to give money, used for gifts) instead of 支払う when talking about paying for goods/services.
- Memory tip: 支（branch out, support）+ 払う（to pay/sweep away）— a slightly formal word for "to pay," common on receipts and signs.

**25. 現金（げんきん）— cash**
- Example: 現金かクレジットカードで支払ってください。
- Translation: 請用現金或信用卡付款。
- Common mistake: Confusing 現金（physical cash）with お金（money in general, including in a bank account).
- Memory tip: 現（present, current）+ 金（money）= "money in hand right now," i.e. cash.

## ② Kanji (3): 買, 店, 安

**買（buy）**
- Meaning (Chinese): 買
- On-reading: バイ
- Kun-reading: か.う
- Words: 買い物, 買う, 売買（ばいばい, buying and selling）
- Example sentence: デパートで新しい靴を買いました。
- Stroke order tip: Write the net-like 罒 top first left to right, then the 貝 (shell) body below — 12 strokes total; don't rush the bottom 貝 component, it has its own internal strokes.
- Chinese difference: Nearly identical to Chinese 買 (traditional), so recognition is easy; just remember the kun-reading か.う is used for the everyday verb "to buy," while バイ appears in compounds like 売買.

**店（shop）**
- Meaning (Chinese): 店
- On-reading: テン
- Kun-reading: みせ
- Words: 店, 店員, 書店（しょてん, bookstore）
- Example sentence: あの店はラーメンが有名です。
- Stroke order tip: Start with the 广 roof-like radical (the slanted stroke then the horizontal), then build 占 underneath, finishing with the bottom-right hook.
- Chinese difference: Identical shape and meaning to Chinese 店; only the kun-reading みせ is new — Chinese has no direct equivalent native reading concept.

**安（cheap, safe）**
- Meaning (Chinese): 安
- On-reading: アン
- Kun-reading: やす.い
- Words: 安い, 安心（あんしん, peace of mind）, 不安（ふあん, anxiety）
- Example sentence: このセールは安いです。
- Stroke order tip: The roof (宀) comes first, then 女 underneath — think "a woman under a roof is safe," a classic mnemonic.
- Chinese difference: In Chinese, 安 mainly means "safe/peaceful" (安全, 平安), but in Japanese 安い specifically and commonly means "cheap" in price — this is a false-friend trap: 安い does NOT primarily mean "safe" in everyday shopping contexts, even though 安全 (safety) still exists as a separate word.

**Kanji Review:** Can you read these? 買い物／店員／安い／買う

## ③ Grammar

### Grammar 1 — 〜を＋他動詞 (transitive verb + object marker を)

**Meaning/Structure:** Japanese transitive verbs (verbs that act on a direct object) mark that object with the particle を. Structure: [Object] を [Transitive Verb].

**Examples:**
- 靴を買います。(I buy shoes.)
- プレゼントを包みます。(I wrap the present.)
- 新しいシャツを選びました。(I chose a new shirt.)

**Common mistakes (Chinese speakers):**
- ❌ 靴が買います → ⭕ 靴を買います — Chinese doesn't mark objects with a particle, so learners often omit を entirely or substitute が (which marks subjects/desires, not direct objects of action verbs).
- ❌ 靴に買います → ⭕ 靴を買います — に is for direction/location/indirect objects, not direct objects of action verbs like 買う.

**Exercises (translate into Japanese):**
1. I bought a red bag.
2. She chose a new hat (帽子).
3. Please wrap this present.
4. He is selling old books.

### Grammar 2 — 〜へ行きます／来ます／帰ります (direction particle へ with motion verbs)

**Meaning/Structure:** へ marks the direction of movement toward a destination, used with motion verbs like 行く (go), 来る (come), 帰る (return). Structure: [Destination] へ [Motion Verb]. (に is also acceptable for destinations, but へ emphasizes the direction of travel.)

**Examples:**
- 店へ行きます。(I'm going to the shop.)
- 家へ帰ります。(I'm returning home.)
- 友達が私の家へ来ます。(My friend is coming to my house.)

**Common mistakes (Chinese speakers):**
- ❌ 店を行きます → ⭕ 店へ行きます — を marks direct objects of transitive verbs (Grammar 1 above), but 行く/来る/帰る are motion verbs that need へ or に for destinations, not を.
- Confusing へ (particle, pronounced え) with the hiragana へ used elsewhere — remember that as a particle, へ is always pronounced え, not へ.

**Exercises (translate into Japanese):**
1. I'm going to the department store.
2. My mother is coming to Taiwan.
3. What time are you (あなた) returning home?
4. Let's go to the bank together (一緒に).

## ④ Reading

夏休みに、デパートへ買い物に行きました。新しい靴が欲しかったので、たくさんの店を見ました。ある店で、白い靴と黒い靴がありました。店員さんに「これはいくらですか」と聞きました。値段は高かったですが、その店はセールをしていたので、少し安くなりました。私は黒い靴を選んで、クレジットカードで支払いました。店員さんはきれいな袋に靴を包んでくれました。とても嬉しかったです。

**Comprehension Questions:**
1. 何を買いに行きましたか。
2. どんな色の靴を選びましたか。
3. どうやって支払いましたか。

**Answer Key:**
1. 新しい靴を買いに行きました。(They went to buy new shoes.)
2. 黒い靴を選びました。(They chose black shoes.)
3. クレジットカードで支払いました。(They paid by credit card.)

## ⑤ Listening

*Read once slowly, then once at natural speed.*

A: いらっしゃいませ。何かお探しですか。
B: はい、赤いシャツが欲しいです。
A: こちらはいかがですか。サイズは何ですか。
B: Mサイズです。試着してもいいですか。
A: はい、どうぞ。
B: (試着した後) これをください。いくらですか。
A: 三千円です。今、セールで二千五百円になります。
B: じゃあ、これをお願いします。クレジットカードでいいですか。
A: はい、大丈夫です。

**Comprehension Questions:**
1. お客さんは何色のシャツが欲しかったですか。
2. シャツはセールでいくらになりましたか。

**Answer Key:**
1. 赤いシャツが欲しかったです。(The customer wanted a red shirt.)
2. 二千五百円になりました。(It became 2,500 yen on sale.)

## 🇹🇼 Chinese Notes

- **安い vs. 安**: In Chinese, 安 mostly means "safe/peaceful" (平安, 安全). In Japanese, 安い overwhelmingly means "cheap" in everyday shopping speech — don't assume it always relates to safety.
- **青い is broader than Chinese 藍/青**: Japanese 青い can describe some greens (e.g., traffic-light "green" is called 青 in Japanese), unlike Chinese where 藍 and 綠 are kept strictly separate.
- **買 vs 売 mix-ups**: Because 買 (buy) and 売 (sell) look visually similar and both relate to commerce, Chinese speakers — used to the very distinct 買/賣 — sometimes still swap them under time pressure in Japanese; double-check which character you're writing.

### Word-by-word details

**1. 前に（まえに）— 之前**
- 例文: 日本へ行く前に、日本語を勉強しました。
- 翻譯: 去日本之前，我學了日語。
- Common mistake: Confusing this "before doing X" use (verb dictionary form + 前に) with 前 alone meaning "in front of" (駅の前, Lesson 7) — same kanji, different job depending on what comes before it.
- Memory tip: 前 (front) + に (particle) — picture standing "in front of" a future event, i.e., the time before it happens.

**2. 一度（いちど）— 一次**
- 例文: 京都へ一度行ったことがあります。
- 翻譯: 我去過京都一次。
- Common mistake: Not realizing 一回 is a near-synonym — both work, but 一度 pairs most naturally with 〜たことがあります when describing life experience.
- Memory tip: 一 (one) + 度 (times/degree) = "one time."

**3. 二度（にど）— 兩次**
- 例文: 彼に二度会いました。
- 翻譯: 我見過他兩次。
- Common mistake: Writing ❌二つ度 by mixing the object counter 二つ (things) with the times-counter 二度 — they count different things.
- Memory tip: The set phrase 二度と〜ません ("never again") is built from this word: 二度と忘れません (I'll never forget again).

**4. 初めて（はじめて）— 第一次**
- 例文: 初めて日本へ行きました。
- 翻譯: 我第一次去了日本。
- Common mistake: Confusing with 初めまして (nice to meet you, Lesson 1) — 初めて is an adverb meaning "for the first time [doing something]," not a greeting.
- Memory tip: Both share 初め (beginning); 初めまして = "we're meeting for the first time," 初めて = "the first time [I did X]."

**5. 経験（けいけん）— 經驗**
- 例文: 私は日本で働いた経験があります。
- 翻譯: 我有在日本工作的經驗。
- Common mistake: Treating 経験 as a ready-made verb (❌経験ます) — it is a noun and needs する to become a verb: 経験する.
- Memory tip: 経 (pass through) + 験 (test/verify) = "what you've passed through and verified" — experience.

**6. 思い出（おもいで）— 回憶**
- 例文: これは楽しい思い出です。
- 翻譯: 這是一個開心的回憶。
- Common mistake: Mixing up 思い出 (a noun, "a memory/recollection") with 思う (to think, Lesson 23) — 思い出 is the *thing* you recall, not the act of thinking.
- Memory tip: 思い (thoughts) + 出す (to bring out) → 思い出 = "things brought out from your thoughts."

**7. 子供の時（こどものとき）— 小時候**
- 例文: 子供の時、よく公園で遊びました。
- 翻譯: 小時候，我常常在公園玩。
- Common mistake: Wrong word order copied from English/Chinese, like ❌時の子供 — Japanese fixes the order as 子供の時 (child's time = "when [I was] a child").
- Memory tip: 子供 (child) + の (possessive) + 時 (time) = "the time of [being a] child."

**8. 昔（むかし）— 從前**
- 例文: 昔、この町は小さかったです。
- 翻譯: 從前，這個城鎮很小。
- Common mistake: Using 昔 for "yesterday" or "last year" — 昔 implies a vague, distant past (years and years ago), not the recent past.
- Memory tip: Japanese fairy tales open with 昔々あるところに…("Once upon a time…") — a memorable fixed phrase.

**9. 当時（とうじ）— 當時**
- 例文: 当時、私は学生でした。
- 翻譯: 當時，我是學生。
- Common mistake: Using 当時 to mean "now" — it strictly points back to a specific past moment already established in context, unlike 今.
- Memory tip: 当 (that/appropriate) + 時 (time) = "that [particular] time being discussed."

**10. 以前（いぜん）— 以前**
- 例文: 以前、ここに銀行がありました。
- 翻譯: 以前這裡有一家銀行。
- Common mistake: Confusing 以前 (a general "formerly," often opening a whole sentence) with 前に (attached right after a specific verb, "before doing X").
- Memory tip: 以 (by means of) + 前 (before) = "from before" — a slightly formal way to say "previously."

**11. もう — 已經**
- 例文: もう晩ご飯を食べました。
- 翻譯: 我已經吃過晚飯了。
- Common mistake: Missing that もう flips meaning with a negative verb — もう+past = "already done," but もう+ません = "not anymore," e.g. もう食べません (I won't eat anymore).
- Memory tip: もう by itself, said with a sigh, also just means "ugh, enough already" in casual speech — same feeling of "that's done/enough."

**12. まだ — 還、尚未**
- 例文: まだ晩ご飯を食べていません。
- 翻譯: 我還沒吃晚飯。
- Common mistake: Pairing まだ with plain past tense to mean "not yet" (❌まだ食べました) instead of the correct まだ＋〜ていません.
- Memory tip: まだ + negative = "not yet"; まだ + affirmative = "still" (まだ勉強しています, "I'm still studying").

**13. すでに — 已經（書面語）**
- 例文: 会議はすでに終わりました。
- 翻譯: 會議已經結束了。
- Common mistake: Using すでに in casual chat with friends, where もう sounds far more natural — すでに reads as stiff/written style.
- Memory tip: Save すでに for writing, news, and reports; use もう when speaking casually.

**14. やっと — 終於**
- 例文: やっと宿題が終わりました。
- 翻譯: 作業終於做完了。
- Common mistake: Using やっと for an unwelcome outcome (❌やっと雨が降りました when annoyed) — やっと carries relief after waiting/effort, a positive nuance.
- Memory tip: やっと = the sigh of relief after a long wait finally pays off.

**15. ついに — 終於、最後**
- 例文: ついに日本へ行く日が来ました。
- 翻譯: 終於到了要去日本的日子。
- Common mistake: Treating やっと and ついに as fully interchangeable — ついに is more dramatic/formal and can head both good and bad news, while やっと leans positive only.
- Memory tip: ついに often opens headline-style sentences: ついに〜が発表されました (At last, ~ was announced).

**16. 結局（けっきょく）— 結果、最後**
- 例文: 結局、旅行はキャンセルしました。
- 翻譯: 結果，我們取消了旅行。
- Common mistake: Confusing 結局 ("in the end," summarizing how a whole story concluded) with 最後に ("lastly," just the final item in a sequence).
- Memory tip: 結 (tie up/conclude) + 局 (situation) = "how the situation concluded."

**17. 実は（じつは）— 其實**
- 例文: 実は、明日国へ帰ります。
- 翻譯: 其實，我明天要回國了。
- Common mistake: Placing 実は in the middle of a sentence the way English sometimes places "actually" — in Japanese it almost always opens the sentence to signal "here comes a confession/surprising fact."
- Memory tip: 実 (truth) + は (topic marker) = "as for the truth…"

**18. 実際に（じっさいに）— 實際上**
- 例文: 実際に、日本語は思ったより難しいです。
- 翻譯: 實際上，日語比我想的難。
- Common mistake: Mixing up 実際に (contrasts expectation vs. actual fact) with 実は (reveals a hidden fact) — they share 実 but do different jobs.
- Memory tip: 実際に = "in actual fact" (vs. what you assumed); 実は = "to tell the truth" (a secret you're revealing).

**19. 変わる（かわる）— 改變（自動詞）**
- 例文: 町の様子が変わりました。
- 翻譯: 城鎮的樣子改變了。
- Common mistake: Confusing intransitive 変わる (something changes on its own) with transitive 変える (Lesson 37, to change something) — 変わる never takes を + object.
- Memory tip: The わ sound feels passive/spontaneous — "わ〜, it changed by itself!"

**20. 育つ（そだつ）— 成長（自動詞）**
- 例文: 私は台北で育ちました。
- 翻譯: 我在台北長大。
- Common mistake: Confusing with 育てる (transitive, "to raise" someone/something) — 育つ describes the subject growing up themselves.
- Memory tip: 子供の時 + 育つ form a natural pair: "growing up as a child."

**21. 過ごす（すごす）— 度過（時間）**
- 例文: 週末は家族と過ごしました。
- 翻譯: 我和家人一起度過了週末。
- Common mistake: Using 使う (Lesson 6, "to use") for time by mistake (❌時間を使ごす) — 過ごす means "to spend/pass time," while 使う is for money or objects.
- Memory tip: Related to 過ぎる ("to pass") + す = "to make time pass [in a certain way]."

**22. 懐かしい（なつかしい）— 令人懷念的**
- 例文: この歌を聞くと懐かしいです。
- 翻譯: 聽到這首歌就覺得很懷念。
- Common mistake: Translating straight from Chinese 懷念 as an action verb ("to miss someone") — 懐かしい is an い-adjective describing a *feeling*, not an action; "to miss someone" is 会いたい/恋しい.
- Memory tip: 懐 (bosom/embrace) + かしい = feelings held close to your heart from the past.

**23. 記憶（きおく）— 記憶**
- 例文: 子供の時の記憶がほとんどありません。
- 翻譯: 我幾乎沒有小時候的記憶。
- Common mistake: Mixing up 記憶 (memory stored in the mind, more clinical, e.g. 記憶喪失 "memory loss") with 思い出 (a fond, emotional memory) — the two are not interchangeable.
- Memory tip: 記 (record) + 憶 (recall) = "recorded and recalled" — the brain's storage of memory.

**24. 体験する（たいけんする）— 體驗**
- 例文: 温泉を体験しました。
- 翻譯: 我體驗了溫泉。
- Common mistake: Using 経験する and 体験する interchangeably without nuance — 経験する suits accumulated life/work experience (仕事の経験), while 体験する suits a one-time hands-on activity (温泉体験、文化体験).
- Memory tip: 体 (body) + 験 (test/experience) = "experiencing with your own body" — hands-on.

**25. 振り返る（ふりかえる）— 回顧、回頭看**
- 例文: 昔のことを振り返ってみました。
- 翻譯: 我試著回顧了從前的事。
- Common mistake: Only knowing the literal "turn your head around" meaning and missing the very common figurative sense "to reflect on the past" — this lesson mainly uses the figurative one.
- Memory tip: 振り (swing) + 返る (return) = "swing [your head/thoughts] back" — look back.

## ② Kanji (3): 度, 昔, 実

**度 — Meaning: 度 (times, degree)**
- On-reading: ド
- Kun-reading: —
- Words: 一度, 二度, 今度 (preview, Lesson 37)
- Example sentence: 一度、日本へ行きました。
- Stroke order: 9 strokes — write the slanted "roof" stroke across the top-left first, then build the enclosed middle section, finishing with 又-like strokes at the bottom right.
- Chinese difference: The character matches Traditional Chinese 度 exactly, and "degree/extent" meanings overlap (温度, 角度). But Japanese also uses 度 as the everyday counter for "number of times" (一度、二度), where Mandarin normally uses 次 (一次、兩次) instead of 度 — a subtle usage gap even though the character itself is shared.

**昔 — Meaning: 昔 (long ago)**
- On-reading: セキ
- Kun-reading: むかし
- Words: 昔 (むかし, standalone), 昔話 (むかしばなし, "folk tale" — bonus word, not required vocab)
- Example sentence: 昔、この辺は静かでした。
- Stroke order: 8 strokes — write the top component (a crossed shape resembling 廿) first, left to right, then finish with 日 at the bottom.
- Chinese difference: Same character and core meaning as Chinese 昔 ("former times/past"), but Japanese layers a native kun-reading むかし onto it for everyday speech, alongside the Chinese-derived セキ used mainly in compounds like 今昔 — Mandarin has only the single reading "xī."

**実 — Meaning: 實 (truth, fruit)**
- On-reading: ジツ
- Kun-reading: み
- Words: 実は, 実際に, 果実 (かじつ, "fruit" — bonus word showing the み-related meaning)
- Example sentence: 実は、明日誕生日です。
- Stroke order: 8 strokes — write the roof-like 宀 shape first, then the 十-based lower body, keeping the central vertical stroke straight through the middle.
- Chinese difference: Japanese 実 is the shinjitai (simplified) form of Traditional Chinese 實, which Taiwanese learners still use — visually it looks much closer to Simplified Chinese 实. The core meaning ("truth," "reality," "fruit") is shared across all three forms.

## ③ Grammar (3 points)

### Grammar 1 — 〜たほうがいいです ("had better do")
**Meaning:** Gives advice or a recommendation to do something.
**Structure:** Verb-た form + ほうがいいです
**Examples:**
1. 早く寝たほうがいいです。(You'd better go to bed early.)
2. 薬を飲んだほうがいいです。(You'd better take medicine.)
3. 医者に相談したほうがいいです。(You'd better consult a doctor.)

**Common mistakes (Chinese speakers):**
- Forgetting the た form and using the dictionary form for concrete personal advice (寝るほうがいいです sounds more like a general statement) — for advice to a specific person right now, the た form is standard.
- ❌ Dropping です for politeness (寝たほうがいい alone is fine casually, but in polite/written contexts です is expected).

**Exercises (translate into Japanese):**
1. You'd better rest today.
2. You'd better call your teacher.
3. You'd better bring an umbrella.

### Grammar 2 — 〜ないほうがいいです ("had better not do")
**Meaning:** Advises against doing something.
**Structure:** Verb-ない form + ほうがいいです
**Examples:**
1. お酒を飲まないほうがいいです。(You'd better not drink alcohol.)
2. 無理をしないほうがいいです。(You'd better not overdo it.)
3. 夜遅く食べないほうがいいです。(You'd better not eat late at night.)

**Common mistakes (Chinese speakers):**
- ❌ Double-negating for emphasis (ないほうがよくないです) — ないほうがいいです is already the complete negative-advice form; no extra negative is needed.
- Confusing this with 〜てはいけません (Lesson 23, "must not") — ないほうがいい is a soft suggestion, while てはいけません is a strict prohibition/rule.

**Exercises (translate into Japanese):**
1. You'd better not go out today.
2. You'd better not forget your passport.
3. You'd better not be late.

### Grammar 3 — 〜かもしれません ("might be")
**Meaning:** Expresses a guess or possibility, weaker in certainty than 〜でしょう (Lesson 28).
**Structure:** Plain form (verb / adjective / noun) + かもしれません
**Examples:**
1. 明日は雨かもしれません。(It might rain tomorrow.)
2. 彼はもう帰ったかもしれません。(He might have already gone home.)
3. この漢字は難しいかもしれません。(This kanji might be difficult.)

**Common mistakes (Chinese speakers):**
- ❌ Adding です after it (かもしれませんです) — かもしれません is already a complete polite predicate on its own.
- Treating かもしれません and でしょう as equally confident — かもしれません signals low confidence (roughly 50% or less), while でしょう signals a fairly confident guess.

**Exercises (translate into Japanese):**
1. He might be busy.
2. It might already be too late.
3. This might be a mistake.

## ④ Reading

私は台北で生まれて、台北で育ちました。子供の時、家の近くに大きい公園がありました。よく友達とそこで遊びました。昔はこの町も静かでしたが、今はにぎやかになりました。実は、一度だけ日本へ旅行に行ったことがあります。とても楽しかったです。あの時の思い出は今でも懐かしいです。もう十年前のことですが、まだよく覚えています。いつか、また日本へ行ったほうがいいと思います。

**Answer key:**
1. 台北で育ちました。
2. 一度行ったことがあります。
3. 懐かしいと感じています。(とても楽しかったです。)

## 🇹🇼 Chinese Notes

- **度（一度／二度）**: The character matches Traditional Chinese 度 and both languages use it for "degree" (温度／溫度). But for counting "times/occurrences," Japanese uses 度 (一度、二度) where Mandarin normally reaches for 次 (一次、兩次) instead — don't assume the counting word transfers directly.
- **実（実は／実際に）**: This is the shinjitai form of Traditional Chinese 實. It looks closer to Simplified Chinese 实 than to the 實 Taiwanese learners write by hand — a good reminder that Japanese kanji forms sometimes resemble Simplified rather than Traditional Chinese.
- **懐かしい vs. 懷念**: In Chinese, 懷念 is a verb — "to miss/reminisce about" someone or something. In Japanese, 懐かしい is an い-adjective describing a feeling ("nostalgic"), not an action. You cannot use 懐かしい the way you'd use 懷念 as a verb; to say "I miss you" in Japanese, use 会いたい instead.

### Vocabulary in Depth

**1. 趣味（しゅみ）— hobby**
- Example: 私の趣味は音楽を聞くことです。
- Translation: 我的興趣是聽音樂。
- Common mistake: Answering 趣味は何ですか with just a bare verb (❌ 趣味は音楽を聞きます) instead of nominalizing it with こと: 趣味は音楽を聞くことです.
- Memory tip: 趣（interest/inclination）+ 味（taste/flavor）= "a taste for something you're interested in."

**2. 音楽（おんがく）— music**
- Example: 音楽を聞くのが好きです。
- Translation: 我喜歡聽音樂。
- Common mistake: Using 音楽を見る (to watch music) instead of 聞く (to listen) — music is always "listened to," never "watched," even for a music video (in that case ビデオを見る, but the music itself is 聞く).
- Memory tip: 音（sound）+ 楽（enjoyment/music）= "enjoyable sound."

**3. 映画（えいが）— movie**
- Example: 週末、映画を見ます。
- Translation: 週末我要看電影。
- Common mistake: Using 聞く with 映画 (confusing it with 音楽) — movies are watched: 映画を見る, not ❌ 映画を聞く.
- Memory tip: 映（reflect/project）+ 画（picture）= "projected pictures," i.e. a movie.

**4. スポーツ — sports**
- Example: 私はスポーツが好きです。
- Translation: 我喜歡運動。
- Common mistake: Using する directly with a specific sport name the same way as with スポーツ in general — スポーツをする is fine generally, but specific sports (テニス, サッカー) also pair with する: テニスをする, not ❌ テニスを泳ぐ-style verb mixing.
- Memory tip: From English "sports" — pronounced su-poo-tsu.

**5. 本（ほん）— book**
- Example: 暇な時、本を読みます。
- Translation: 有空的時候，我看書。
- Common mistake: Confusing 本（ほん, book, standalone noun, Lesson 14 vocabulary）with 日本（にほん, Japan, Lesson 1）— same kanji, but 本 alone in this context simply means "book."
- Memory tip: Pair with 読む (Lesson 6: to read) — 本を読む is one of the most common hobby phrases.

**6. テレビ — TV**
- Example: 休日はテレビを見ます。
- Translation: 假日我看電視。
- Common mistake: Using 聞く instead of 見る for watching TV — like movies, TV is watched: テレビを見る.
- Memory tip: Shortened from English "television" — テレビ is one of the earliest and most common katakana loanwords learners encounter.

**7. ゲーム — game**
- Example: 弟はゲームが好きです。
- Translation: 我弟弟喜歡玩遊戲。
- Common mistake: Using 遊びます alone without ゲーム when specifically talking about video games — ゲームをします／ゲームで遊びます are both fine, but dropping ゲーム entirely loses the specific meaning.
- Memory tip: From English "game" — ゲームをする is the standard verb pairing.

**8. 写真（しゃしん）— photo**
- Example: 写真を撮るのが趣味です。
- Translation: 拍照是我的興趣。
- Common mistake: Using 撮る（とる, to take a photo）interchangeably with 取る（とる, to take/pick up, general) — same pronunciation, different kanji; for photos specifically, 撮る is correct.
- Memory tip: 写（copy/reflect）+ 真（truth/reality）= "copying reality," i.e. a photograph.

**9. 絵（え）— picture, drawing**
- Example: 絵を描くのが好きです。
- Translation: 我喜歡畫畫。
- Common mistake: Confusing 絵（え, a picture/drawing）with 映画（えいが, movie）due to the shared え sound — different kanji and meaning; context (art vs. film) disambiguates.
- Memory tip: Pair with 描く (item 17 below) — 絵を描く is a fixed, frequently used phrase.

**10. 歌（うた）— song**
- Example: 好きな歌がありますか。
- Translation: 你有喜歡的歌嗎？
- Common mistake: Confusing the noun 歌（うた, song）with the verb 歌う（うたう, to sing, item 16 below) — 歌 is the thing, 歌う is the action; だ conjugation differences matter (歌です vs 歌います, though 歌う is a regular u-verb, not います group).
- Memory tip: 歌 contains 哥 (elder brother, phonetic in Chinese) + 欠 (yawn/open mouth) — associate the open-mouth radical with singing.

**11. ダンス — dance**
- Example: ダンスを習っています。
- Translation: 我正在學跳舞。
- Common mistake: Confusing ダンス (the noun "dance," a katakana loanword) with 踊る (おどる, the native verb "to dance," item 15 below) — ダンスをする and 踊る are near-synonyms, but ダンス is a noun requiring する, while 踊る is already a verb.
- Memory tip: From English "dance" — ダンスをする vs. 踊る, both mean "to dance," useful to know as a pair.

**12. 泳ぐ（およぐ）— to swim**
- Example: 夏はよく泳ぎます。
- Translation: 夏天我常游泳。
- Common mistake: Using を with 泳ぐ when specifying location incorrectly, saying ❌ プールを泳ぎます instead of プールで泳ぎます (the location-で rule from Lesson 12 applies here: swim *at* the pool).
- Memory tip: Contains 水 radical (as 氵) on the left — directly tied to water activities.

**13. 走る（はしる）— to run**
- Example: 毎朝、公園で走ります。
- Translation: 我每天早上在公園跑步。
- Common mistake: Confusing 走る（はしる, to run) with 歩く（あるく, to walk, Lesson 17) due to both being locomotion verbs — different pace, different kanji.
- Memory tip: 走 resembles Chinese 走 but note the false friend: Chinese 走 usually means "to walk/go," while Japanese 走る specifically means "to run" — walking in Japanese is 歩く, not 走る.

**14. 遊ぶ（あそぶ）— to play**
- Example: 子供と公園で遊びます。
- Translation: 我和小孩在公園玩。
- Common mistake: Using 遊ぶ to mean "play a game/sport" the way English "play tennis" works — for specific games/sports/instruments, する or 弾く is used instead; 遊ぶ is for general playing/hanging out.
- Memory tip: Distinct from スポーツをする (playing sports) and 楽器を弾く (playing an instrument) — 遊ぶ is the general "have fun/play around" verb.

**15. 踊る（おどる）— to dance**
- Example: パーティーで踊りました。
- Translation: 我在派對上跳了舞。
- Common mistake: Mixing up 踊る（おどる, to dance）with 躍る（おどる, to leap/jump excitedly) — same pronunciation, different kanji and nuance; 踊る is the standard word for dancing.
- Memory tip: Contains 足 (foot) radical on the left — dancing involves footwork.

**16. 歌う（うたう）— to sing**
- Example: カラオケで歌います。
- Translation: 我在KTV唱歌。
- Common mistake: Conjugating 歌う as an いる/える (ichidan) verb by mistake, saying ❌ 歌ます instead of the correct godan conjugation 歌います — 歌う ends in う and follows godan (u-verb) rules.
- Memory tip: Same kanji as 歌 (song, item 10) + う to make it a verb — "song-ify," i.e. to sing.

**17. 描く（かく）— to draw**
- Example: 絵を描くのが得意です。
- Translation: 我很擅長畫畫。
- Common mistake: Confusing 描く（かく, to draw/paint pictures）with 書く（かく, to write, Lesson 6) — identical pronunciation, different kanji; 描く is for pictures/art, 書く is for writing text.
- Memory tip: Contains 扌(hand radical) + 苗 (seedling, phonetic) — the hand radical reminds you it's a hands-on creative action, distinct from 書く's writing-brush-derived kanji.

**18. 楽器（がっき）— musical instrument**
- Example: 何か楽器ができますか。
- Translation: 你會什麼樂器嗎？
- Common mistake: Using する with a specific instrument name instead of 弾く (ひく, to play a stringed/keyboard instrument) — ピアノを弾く, ギターを弾く, not ❌ ピアノをします in careful speech (casual speech sometimes allows it, but 弾く is the precise verb).
- Memory tip: 楽（music/enjoyment）+ 器（tool/instrument）= "a tool for music."

**19. ピアノ — piano**
- Example: ピアノを弾くことができます。
- Translation: 我會彈鋼琴。
- Common mistake: Using 吹く（ふく, to blow, for wind instruments）instead of 弾く（ひく, to pluck/press, for piano/guitar）— different verbs for different instrument families.
- Memory tip: From English "piano" — ピアノを弾く is a fixed, common collocation.

**20. ギター — guitar**
- Example: ギターを習いたいです。
- Translation: 我想學吉他。
- Common mistake: Same as ピアノ — using the wrong verb (吹く instead of 弾く) for a stringed instrument.
- Memory tip: From English "guitar" — pair with ピアノ as the two most common instrument loanwords in N5.

**21. 得意な（とくいな）— good at**
- Example: 料理が得意です。
- Translation: 我很擅長做菜。
- Common mistake: Confusing 得意（とくい, good at, a skill you're confident in）with 上手（じょうず, skillful, more about objective skill level, and awkward to use about yourself）— 得意 is safer and more natural for describing your own strengths.
- Memory tip: 得（gain/obtain）+ 意（intention/mind）= "having gained mastery," i.e. good at something.

**22. 苦手な（にがてな）— poor at**
- Example: 歌うのが苦手です。
- Translation: 我不擅長唱歌。
- Common mistake: Confusing 苦手（にがて, poor at/have difficulty with, can include dislike）with 下手（へた, unskillful, purely about skill level）— 苦手 often implies discomfort or weakness, not just lack of skill; both are valid but carry slightly different nuance.
- Memory tip: 苦（bitter/suffering）+ 手（hand）= "a bitter hand at it," i.e. not good at something.

**23. 休日（きゅうじつ）— day off**
- Example: 休日は何をしますか。
- Translation: 你放假的時候都做什麼？
- Common mistake: Confusing 休日（きゅうじつ, a day off from work/school, slightly formal）with 週末（しゅうまつ, weekend, Lesson 16) — 休日 doesn't have to be a weekend (could be a national holiday), while 週末 specifically means Saturday/Sunday.
- Memory tip: 休（rest）+ 日（day）= "a rest day."

**24. 楽しむ（たのしむ）— to enjoy**
- Example: 旅行を楽しみました。
- Translation: 我享受了這趟旅行。
- Common mistake: Confusing the verb 楽しむ（たのしむ, to enjoy [something], transitive）with the adjective 楽しい（たのしい, fun/enjoyable, Lesson 8) — 楽しい describes something as fun, 楽しむ is the act of enjoying it.
- Memory tip: Same 楽 kanji as 楽器 and 音楽 — all rooted in the idea of enjoyment/music.

**25. 集める（あつめる）— to collect**
- Example: 切手を集めるのが趣味です。
- Translation: 收集郵票是我的興趣。
- Common mistake: Using 集まる（あつまる, intransitive, "to gather/assemble [by itself]"）when the transitive 集める（to collect [something], someone actively gathers items) is meant — 切手を集める (I collect stamps) vs. 人が集まる (people gather).
- Memory tip: Contains 隹 (bird radical) — birds flocking together is the old image behind "gathering/collecting."

## ② Kanji (2): 楽, 歌

**楽（music, fun）**
- Meaning (Chinese): 樂
- On-reading: ガク、ラク
- Kun-reading: たの.しい
- Words: 音楽, 楽器, 楽しい
- Example sentence: 音楽を聞くのは楽しいです。
- Stroke order tip: The top two "幺" side strokes are drawn first, then the central 白-like shape, then the bottom 木-like legs — 13 strokes, take it slowly.
- Chinese difference: This is the simplified Japanese form of traditional Chinese 樂; the shape is close enough to recognize, but written slightly differently from both traditional 樂 and simplified 乐 — note also the dual reading ガク (music, as in 音楽) vs. ラク (comfort/ease, as in 楽（らく）— easy/comfortable), a meaning split that doesn't exist as clearly in Chinese, where 樂/乐 leans toward "happy/music" and 易 covers "easy" separately.

**歌（song）**
- Meaning (Chinese): 歌
- On-reading: カ
- Kun-reading: うた、うた.う
- Words: 歌, 歌う, 歌手（かしゅ, singer）
- Example sentence: カラオケで歌を歌います。
- Stroke order tip: Write the left 哥-like component first (two 可 shapes stacked), then the right 欠 component (open-mouth/yawn shape) last — 14 strokes.
- Chinese difference: Nearly identical to Chinese 歌 in shape and meaning ("song/to sing"); a smooth, low-risk kanji for Chinese speakers, though remember the verb form 歌う needs the okurigana う to conjugate properly, unlike Chinese where 唱歌 uses a separate character entirely.

**Kanji Review:** Can you read these? 音楽／楽しい／歌う／歌手

## ③ Grammar

### Grammar 1 — な形容詞＋です (na-adjective + です)

**Meaning/Structure:** Na-adjectives attach directly to です (no な needed) when ending a sentence in the plain polite present affirmative. The な only appears when the adjective directly modifies a following noun. Structure: [Noun] は [na-adjective]です. / [na-adjective]な[Noun].

**Examples:**
- 歌が得意です。(I'm good at singing.)
- 彼はダンスが得意な人です。(He is a person who is good at dancing.)
- 音楽が好きです。(I like music.)

**Common mistakes (Chinese speakers):**
- ❌ 得意なです → ⭕ 得意です — learners over-apply the な rule and add it even before です, where it should be dropped; な is only needed before a noun, not before です.
- ❌ 得意人です → ⭕ 得意な人です — the opposite mistake: forgetting な entirely when directly modifying a noun.

**Exercises (translate into Japanese):**
1. I am good at swimming.
2. She is poor at singing.
3. He is a hobby-loving (趣味が多い, or simply) person who likes hobbies. (Simplify: He likes hobbies.)
4. This is a convenient app.

### Grammar 2 — な形容詞の否定形（〜じゃないです）(na-adjective negative form)

**Meaning/Structure:** To negate a na-adjective politely, replace です with じゃないです (or the slightly more formal ではありません). Structure: [Noun] は [na-adjective] じゃないです.

**Examples:**
- 歌は得意じゃないです。(Singing isn't something I'm good at.)
- ダンスは苦手じゃないです。(I'm not bad at dancing.)
- この歌手は有名じゃないです。(This singer isn't famous.)

**Common mistakes (Chinese speakers):**
- ❌ 得意くないです → ⭕ 得意じゃないです — learners apply the i-adjective negative pattern (くない, Lesson 13) to na-adjectives by mistake; na-adjectives use じゃない, not くない.
- Confusing じゃ (casual contraction of では) — じゃないです and ではありません are interchangeable in meaning, but じゃ is more casual/spoken, では more formal/written.

**Exercises (translate into Japanese):**
1. This song is not famous.
2. I'm not good at drawing.
3. This game is not convenient.
4. He is not free (暇) today.

## ④ Reading

私の趣味は音楽です。特に歌を歌うのが好きです。週末はよくカラオケに行きます。友達もいつも一緒に来ます。私は歌うのが得意ですが、ダンスは苦手です。友達はダンスが得意なので、いつも踊ってくれます。休日は他に、写真を撮るのも楽しみます。町を歩きながら、きれいな写真を集めています。今度、みんなで公園へ行って、写真を撮りましょう。

**Comprehension Questions:**
1. この人の趣味は何ですか。
2. この人は何が得意ですか。何が苦手ですか。
3. 休日、他に何を楽しみますか。

**Answer Key:**
1. 音楽（特に歌を歌うこと）です。(Music, especially singing.)
2. 歌うのが得意で、ダンスが苦手です。(Good at singing, poor at dancing.)
3. 写真を撮ることを楽しみます。(Enjoys taking photos.)

## 🇹🇼 Chinese Notes

- **走る is a false friend**: In Chinese, 走 usually means "to walk/go," but in Japanese 走る specifically means "to run." Walking in Japanese is 歩く — mixing these up is one of the most common Chinese-speaker errors in this lesson.
- **描く vs 書く**: Both read かく, but 描く is for drawing pictures and 書く is for writing text — Chinese distinguishes these clearly with 畫 vs 寫, so map that distinction onto the two different kanji rather than assuming one かく covers everything.
- **得意 vs 上手, 苦手 vs 下手**: Chinese speakers often reach for 上手/下手 (skill-based) when 得意/苦手 (confidence/comfort-based, and safer to use about yourself) would sound more natural — especially avoid describing your own skill as 上手, which can sound like bragging.

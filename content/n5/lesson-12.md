### Vocabulary in Depth

**1. 頭（あたま）— head**
- Example: 頭が痛いです。
- Translation: 我頭痛。
- Common mistake: Confusing 頭（あたま, the physical head）with 頭がいい（idiom for "smart"）— learners sometimes think 頭 always relates to intelligence rather than the body part.
- Memory tip: 頭 resembles Chinese 頭 (head) almost exactly — easy transfer.

**2. 顔（かお）— face**
- Example: 顔を洗います。
- Translation: 我洗臉。
- Common mistake: Using 顔 where 頭（head, the whole top of the body）is meant — 顔 is specifically the face, not the whole head.
- Memory tip: Pair with 洗う (Lesson 6: to wash) — 顔を洗う is a very common daily-routine phrase.

**3. 目（め）— eye**
- Example: 目が痛いです。
- Translation: 我眼睛痛。
- Common mistake: Confusing kun-reading め with the on-reading モク used in compounds like 目的（もくてき, purpose）— as a standalone body part it's always め.
- Memory tip: The kanji 目 literally looks like an eye turned sideways — one of the most pictographic kanji.

**4. 耳（みみ）— ear**
- Example: 耳が痛いです。音楽を聞きすぎました。
- Translation: 我耳朵痛，聽太多音樂了。
- Common mistake: Mixing up 耳（みみ, ear）with 目（め, eye）since both are short body-part nouns — practice them as a pair.
- Memory tip: 耳 looks like a simplified drawing of an ear's outer shape.

**5. 鼻（はな）— nose**
- Example: 鼻が高いですね。
- Translation: 你的鼻子好挺喔。
- Common mistake: Confusing 鼻（はな, nose）with 花（はな, flower）— identical pronunciation, different kanji and meaning; context (body vs. plants) disambiguates.
- Memory tip: Because 鼻 and 花 sound the same, associate 鼻 with the 鼻 kanji's "self" (自) component at top, since in old gesture language pointing to one's nose meant "me."

**6. 口（くち）— mouth**
- Example: 口を大きく開けてください。
- Translation: 請把嘴巴張大。
- Common mistake: Confusing 口（くち, mouth, standalone noun）with the counter 〜人（にん/じん）for people, or with 人口（じんこう, population, an unrelated compound using the same character）.
- Memory tip: 口 is a simple square representing an open mouth — very close to Chinese 口 in both shape and meaning.

**7. 手（て）— hand**
- Example: 手を洗ってください。
- Translation: 請洗手。
- Common mistake: Using 手 to also mean "arm," but Japanese usually treats the whole arm as 腕（うで）— 手 specifically emphasizes the hand.
- Memory tip: 手 is identical to Chinese 手 — direct transfer.

**8. 足（あし）— leg, foot**
- Example: 足が痛くて、歩けません。
- Translation: 我腳痛，走不動。
- Common mistake: Japanese 足 covers both "leg" and "foot," unlike Chinese which distinguishes 腿 (leg) from 腳 (foot) — don't look for two separate words.
- Memory tip: Remember 足 is also the kanji in 足りる (to be sufficient, Lesson 12 kanji note below) — think of "having enough legs to stand on."

**9. お腹（おなか）— stomach**
- Example: お腹が痛いです。
- Translation: 我肚子痛。
- Common mistake: Forgetting the polite お prefix and just saying ❌ 腹が痛い in casual contexts when speaking politely — お腹 is the standard polite/neutral form.
- Memory tip: Pair with お腹が空く (Lesson 9: to get hungry) — お腹 shows up constantly with feelings about eating.

**10. 体（からだ）— body**
- Example: 体が弱いです。
- Translation: 我身體虛弱。
- Common mistake: Confusing 体（からだ, physical body）with 体重（たいじゅう, body weight）— 体 alone is general, 体重 is the specific measurement.
- Memory tip: 体 is the simplified form of Chinese 體; the 人＋本 shape can be remembered as "a person's foundation."

**11. 病気（びょうき）— sickness**
- Example: 病気で学校を休みました。
- Translation: 因為生病，我請假沒去學校。
- Common mistake: Saying ❌ 病気です to mean "I am sick right now" in the middle of illness — 病気です more naturally describes having an illness/condition in general; for feeling sick right now, 気分が悪い is common too.
- Memory tip: 病（sickness）+ 気（spirit/energy）— literally "sickness of spirit/body."

**12. 元気（げんき）— healthy, well**
- Example: お元気ですか。はい、元気です。
- Translation: 你好嗎？是的，我很好。
- Common mistake: Treating 元気 as an i-adjective and saying ❌ 元気いです — it's a na-adjective: 元気です／元気な人.
- Memory tip: This is the classic greeting response — memorize お元気ですか／元気です as a fixed pair from day one.

**13. 薬（くすり）— medicine**
- Example: 薬を飲んでください。
- Translation: 請吃藥。
- Common mistake: Using 食べる (to eat) with 薬 instead of 飲む (to drink) — in Japanese, medicine (pills included) is always "drunk" (薬を飲む), never "eaten," even for solid pills.
- Memory tip: Remember the fixed collocation 薬を飲む as an exception to the "eat solids, drink liquids" rule.

**14. 医者（いしゃ）— doctor**
- Example: 医者に見てもらいました。
- Translation: 我請醫生看診了。
- Common mistake: Using 医者 to address a doctor directly (❌ 医者さん) — when speaking to or about a specific doctor politely, 先生 is used instead, not 医者さん.
- Memory tip: 医（medicine/healing）+ 者（person）= "a person who heals."

**15. 看護師（かんごし）— nurse**
- Example: 看護師さんはとても親切です。
- Translation: 護理師人很親切。
- Common mistake: Using the older term 看護婦（かんごふ, female nurse only) which is now considered outdated/gender-specific — 看護師 is the current gender-neutral standard term.
- Memory tip: 看（watch over）+ 護（protect）+ 師（professional）= "a professional who watches over and protects [patients]."

**16. 熱（ねつ）— fever**
- Example: 熱があります。
- Translation: 我發燒了。
- Common mistake: Using が instead of the correct verb pattern — 熱があります is fixed; learners sometimes try ❌ 熱です which sounds unnatural.
- Memory tip: 熱 also means "heat" generally (as in 熱い, hot); a fever is "heat in the body."

**17. 痛い（いたい）— painful**
- Example: 頭が痛いです。
- Translation: 我頭痛。
- Common mistake: Using を with 痛い instead of が — the body part that hurts takes が: 頭が痛い, not ❌ 頭を痛い.
- Memory tip: 痛 shares a root idea with Chinese 痛 (pain) — meaning transfers directly, only the grammar (が, not を) is different.

**18. 風邪（かぜ）— a cold (illness)**
- Example: 風邪を引きました。
- Translation: 我感冒了。
- Common mistake: Confusing 風邪（かぜ, a cold, illness）with 風（かぜ, wind, Lesson 13）— same pronunciation, different kanji and meaning; also note the verb used is 引く (to catch/pull), not なる or する: 風邪を引く.
- Memory tip: 風（wind）+ 邪（evil）= old belief that colds were caused by "evil wind" entering the body.

**19. 怪我（けが）— injury**
- Example: 足に怪我をしました。
- Translation: 我的腳受傷了。
- Common mistake: Confusing 怪我（けが, injury from an accident）with 病気（びょうき, illness/sickness）— 怪我 is for cuts, sprains, broken bones, not for colds or diseases.
- Memory tip: The verb pattern is 怪我をする ("to do an injury," i.e. to get injured) — memorize as a set phrase.

**20. 咳（せき）— cough**
- Example: 咳が止まりません。
- Translation: 咳嗽停不下來。
- Common mistake: Using する directly without が, saying ❌ 咳します instead of the more natural 咳が出ます (a cough comes out) or 咳をします.
- Memory tip: The character 亥 (in 咳) appears in several onomatopoeia-like kanji; associate the mouth radical 口 on the left with the sound of coughing.

**21. 体重（たいじゅう）— body weight**
- Example: 体重は何キロですか。
- Translation: 你的體重是幾公斤？
- Common mistake: Asking 体重はいくらですか (using いくら, "how much [money/price]") instead of 何キロ／どのくらい — いくら is for prices, not weight.
- Memory tip: 体（body）+ 重（heavy/weight）= "the weight of the body."

**22. 身長（しんちょう）— height**
- Example: 身長は百七十センチです。
- Translation: 身高是170公分。
- Common mistake: Confusing 身長（しんちょう, height of a person）with 高い（たかい, tall, an adjective）— 身長 is the noun used for stating a specific measurement.
- Memory tip: 身（body）+ 長（long）= "the length of the body," i.e. height.

**23. 健康（けんこう）— health**
- Example: 健康のために、毎日運動します。
- Translation: 為了健康，我每天運動。
- Common mistake: Confusing 健康（noun/na-adjective, "health/healthy"）with 元気（げんき, "well/energetic," more about current condition）— 健康 leans toward overall physical wellness as a state or topic, 元気 leans toward current vitality/mood.
- Memory tip: 健（strong）+ 康（peaceful/well）— both characters independently suggest wellness.

**24. 入院する（にゅういんする）— to be hospitalized**
- Example: 祖父は入院しています。
- Translation: 我爺爺正在住院。
- Common mistake: Confusing 入院する（to be admitted/hospitalized）with 病院に行く（simply "to go to the hospital," e.g. for an outpatient visit）— 入院 specifically implies staying overnight or longer.
- Memory tip: 入（enter）+ 院（institution）= "enter the institution [hospital]," opposite of 退院する (to be discharged).

**25. 注射（ちゅうしゃ）— injection**
- Example: 病院で注射をしました。
- Translation: 我在醫院打了針。
- Common mistake: Using 打つ（うつ, "to hit/strike," used in some contexts for injections in casual speech) inconsistently — the standard textbook verb is 注射をする, not a literal "hit."
- Memory tip: 注（pour in）+ 射（shoot）= "pouring in by shooting," i.e., injecting.

## ② Kanji (5): 耳, 手, 足, 目, 口

**耳（ear）**
- Meaning (Chinese): 耳
- On-reading: ジ
- Kun-reading: みみ
- Words: 耳, 耳鼻科（じびか, ENT department）
- Example sentence: 耳が痛いです。
- Stroke order tip: Draw the outer frame first (top, then sides), then the two internal horizontal strokes — 6 strokes total, top to bottom.
- Chinese difference: Nearly identical to Chinese 耳 in shape and meaning — one of the easiest kanji for Chinese speakers.

**手（hand）**
- Meaning (Chinese): 手
- On-reading: シュ
- Kun-reading: て
- Words: 手, 上手（じょうず, skillful）, 歌手（かしゅ, singer）
- Example sentence: 手を洗ってください。
- Stroke order tip: The three horizontal-ish strokes cross a central vertical stroke that hooks at the bottom — write top to bottom, left to right.
- Chinese difference: Identical to Chinese 手 (hand) in both form and core meaning; on-reading シュ appears in compounds like 歌手 (singer), unfamiliar territory for Chinese speakers used to reading 手 as "shǒu" only.

**足（leg, foot, sufficient）**
- Meaning (Chinese): 足
- On-reading: ソク
- Kun-reading: あし、た.りる
- Words: 足, 足りる（たりる, to be sufficient）, 不足（ふそく, shortage）
- Example sentence: 足が痛いです。
- Stroke order tip: Top 口-like shape first, then the legs-like strokes below — 7 strokes.
- Chinese difference: Same as Chinese 足 in shape; but Japanese uses 足 for both "leg" and "foot" (unlike Chinese, which separates 腿/腳), and also gives it the extra meaning "to be enough" (足りる) — a meaning Chinese 足 shares only in fixed expressions like 充足, so don't be surprised to see 足 outside body-part contexts.

**目（eye）**
- Meaning (Chinese): 目
- On-reading: モク
- Kun-reading: め
- Words: 目, 目的（もくてき, purpose）, 科目（かもく, subject/course）
- Example sentence: 目が大きいですね。
- Stroke order tip: Draw the outer rectangle first, then the two internal horizontal lines — 5 strokes.
- Chinese difference: Same shape and core meaning as Chinese 目 (eye), but in Japanese 目 extends metaphorically to "purpose/goal" (目的) and "item/order" (一番目 = "the first one") — uses beyond the literal eye that go further than typical Chinese usage.

**口（mouth）**
- Meaning (Chinese): 口
- On-reading: コウ
- Kun-reading: くち
- Words: 口, 人口（じんこう, population）, 入り口（いりぐち, entrance）
- Example sentence: 口を開けてください。
- Stroke order tip: A simple three-stroke square — left side, then top+right in one stroke, then the bottom.
- Chinese difference: Identical to Chinese 口 in shape and meaning; note that in compounds like 入り口 (entrance) and 出口 (exit), 口 shifts from "mouth" to "opening/gateway" — a natural metaphorical extension shared with Chinese (人口, 港口), so this one transfers smoothly.

**Kanji Review:** Can you read these? 耳／手／足／目／口が痛いです

## ③ Grammar

### Grammar 1 — 〜で（場所＋動作）(location of an action, particle で)

**Meaning/Structure:** で marks the location where an action takes place (as opposed to に, which marks location of existence). Structure: [Place] で [Action Verb].

**Examples:**
- 病院で注射をしました。(I got an injection at the hospital.)
- 公園で走ります。(I run in the park.)
- 家で薬を飲みます。(I take medicine at home.)

**Common mistakes (Chinese speakers):**
- ❌ 病院に注射をしました → ⭕ 病院で注射をしました — に marks location of existence/destination (公園に行きます, 病院にいます), but で marks the location of an *action*; confusing these two is one of the most common N5 errors for Chinese speakers, since Chinese 在 covers both.
- ❌ 病院は注射をしました → ⭕ 病院で — は marks topic, not location.

**Exercises (translate into Japanese):**
1. I study at the library.
2. She eats breakfast at home.
3. We met at the station.
4. The doctor examined me at the hospital.

### Grammar 2 — 〜で（手段・道具）(means/instrument, particle で)

**Meaning/Structure:** で also marks the means, tool, or method by which an action is performed. Structure: [Tool/Method] で [Action Verb].

**Examples:**
- 薬で熱が下がりました。(The fever went down because of/with the medicine.)
- 箸でご飯を食べます。(I eat rice with chopsticks.)
- 日本語で話します。(I speak in Japanese.)

**Common mistakes (Chinese speakers):**
- ❌ 箸を食べます → ⭕ 箸でご飯を食べます — learners sometimes attach を to the tool itself instead of marking it with で and keeping を on the true object (ご飯).
- Confusing this instrumental で with the location で above — both use the same particle, but context (a place vs. a tool/method) tells them apart.

**Exercises (translate into Japanese):**
1. I write with a pen (ペン).
2. She goes to school by bicycle (自転車).
3. We talked on the phone (電話).
4. He got better with this medicine.

## ④ Reading

先週、お腹がとても痛かったです。それに、熱もありました。だから、病院へ行きました。病院で医者に見てもらいました。医者は「風邪ですね。薬を飲んでください」と言いました。看護師さんは注射をしてくれました。家で三日間休みました。薬を飲んで、たくさん寝ました。今は元気です。もう痛くありません。

**Comprehension Questions:**
1. どこが痛かったですか。
2. 医者は何と言いましたか。
3. 家で何をしましたか。

**Answer Key:**
1. お腹が痛かったです。(The stomach hurt.)
2. 「風邪ですね。薬を飲んでください」と言いました。("It's a cold. Please take medicine.")
3. 三日間休んで、薬を飲んで、たくさん寝ました。(Rested for three days, took medicine, and slept a lot.)

## ⑤ Listening

*Read once slowly, then once at natural speed.*

A: どうしましたか。
B: 昨日から頭が痛いです。それに、咳も出ます。
A: 熱はありますか。
B: はい、少しあります。
A: わかりました。薬を出しますね。一日三回、飲んでください。
B: はい、ありがとうございます。お風呂に入ってもいいですか。
A: 今日はやめたほうがいいですよ。

**Comprehension Questions:**
1. 患者（かんじゃ, patient）はどこが痛いですか。
2. 薬は一日に何回飲みますか。

**Answer Key:**
1. 頭が痛いです。(The head hurts.)
2. 一日三回飲みます。(Three times a day.)

## 🇹🇼 Chinese Notes

- **薬を飲む, not 薬を食べる**: Chinese speakers naturally think of "吃藥" (eating medicine), but Japanese always uses 飲む (drink) for taking medicine, even pills — 食べる would sound wrong here.
- **足 covers both "leg" and "foot"**: Chinese distinguishes 腿 (leg) and 腳 (foot), but Japanese 足 covers the whole limb below the hip. Context or extra words (足首 = ankle, もも = thigh) are needed for precision.
- **風邪 vs. Chinese 風邪 (rare term)**: 風邪 literally reads as "wind evil," an old East Asian medical concept; in modern Chinese this exact compound is rarely used for "a cold" (感冒 is standard), so don't expect the kanji to be self-explanatory from Chinese alone.

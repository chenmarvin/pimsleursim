### Word-by-word details

**1. 一（いち）— 一**
- 例文: 答えは一です。
- 翻譯: 答案是一。
- Common mistake: Confusing 一 (いち) with 一 read as ひと in ひとつ／一人 (ひとり) — same kanji, different reading depending on the word; いち is the "bare" counting number.
- Memory tip: Same shape as Chinese 一 — the easiest kanji in the whole course.

**2. 二（に）— 二**
- 例文: 私は二人家族です。
- 翻譯: 我家有兩個人（暫用，正式助数詞在第7課學）。
- Common mistake: Reading 二 as じ instead of に — a rare slip, but watch for it in fast speech.
- Memory tip: Identical to Chinese 二; two horizontal strokes.

**3. 三（さん）— 三**
- 例文: 答えは三です。
- 翻譯: 答案是三。
- Common mistake: Confusing the number 三（さん）with the honorific ～さん (Mr./Ms.) from Lesson 1 — same sound, totally different meaning, disambiguated by context.
- Memory tip: Three horizontal strokes, same as Chinese 三.

**4. 四（よん）— 四**
- 例文: 教室に学生が四人います。（先取り：人の数え方は第7課）
- 翻譯: 教室裡有四個學生。
- Common mistake: Using the on-reading し instead of よん — し is avoided in many contexts because it sounds like 死 (death); よん is the safer everyday reading.
- Memory tip: Remember よん by its avoidance-of-death association — Japanese speakers actively prefer it.

**5. 五（ご）— 五**
- 例文: 今、五時です。（時間は後の課で詳しく学びます）
- 翻譯: 現在五點。
- Common mistake: Confusing ご (five) with the honorific prefix ご〜 (ご主人, ご飯) — again, same sound, different function.
- Memory tip: Shape looks nothing like Chinese 五's usual handwriting but the character is identical — just memorize it as a set.

**6. 六（ろく）— 六**
- 例文: 六月に日本へ行きます。（月の言い方は後で詳しく）
- 翻譯: 六月要去日本。
- Common mistake: Mixing up ろく (six) with 録/緑 which sound similar in some compounds later — not an issue yet, but be aware of homophones ahead.
- Memory tip: ろく rhymes with "rock" — six is a rock-solid number.

**7. 七（なな）— 七**
- 例文: 答えは七です。
- 翻譯: 答案是七。
- Common mistake: Using しち instead of なな in casual/spoken contexts — しち is common in formal or fixed compounds (七月), but なな is safer as a standalone counting word.
- Memory tip: なな has a soft, easy double-な sound — easy to say twice quickly.

**8. 八（はち）— 八**
- 例文: 八時に起きます。（動詞は後の課で詳しく）
- 翻譯: 八點起床。
- Common mistake: None major, but learners sometimes shorten はち to は in fast speech, which is unclear — always pronounce both syllables.
- Memory tip: Same shape as Chinese 八 (a splitting stroke shape) — instantly recognizable.

**9. 九（きゅう）— 九**
- 例文: 答えは九です。
- 翻譯: 答案是九。
- Common mistake: Using く instead of きゅう as a standalone number — く is used in specific compounds (九月 = くがつ) but きゅう is the default counting reading.
- Memory tip: きゅう sounds like "cue" — cue up to nine.

**10. 十（じゅう）— 十**
- 例文: 全部で十です。
- 翻譯: 總共是十。
- Common mistake: Forgetting that combining forms (二十、三十…) simply stack 二＋十, 三＋十, etc. — learners sometimes try to invent separate words instead of combining.
- Memory tip: Same cross shape as Chinese 十 — instantly familiar.

**11. 百（ひゃく）— 百**
- 例文: これは百円です。
- 翻譯: 這是一百日圓。
- Common mistake: Mispronouncing compound forms — 三百 becomes さんびゃく (not さんひゃく) and 六百 becomes ろっぴゃく — sound changes (rendaku) are easy to miss.
- Memory tip: Learn 三百(さんびゃく)・六百(ろっぴゃく)・八百(はっぴゃく) as memorized exceptions early.

**12. 千（せん）— 千**
- 例文: 財布に千円あります。
- 翻譯: 錢包裡有一千日圓。
- Common mistake: Forgetting that 三千 is さんぜん (voiced ぜん, not せん) — another rendaku sound change to memorize.
- Memory tip: せん sounds like "sen" in "yen" — helps link money vocabulary together.

**13. 万（まん）— 萬**
- 例文: 車は百万円です。
- 翻譯: 車子要一百萬日圓。
- Common mistake: Chinese speakers instinctively try to group numbers in units of 千 (thousand) as in English "ten thousand," but Japanese (like Chinese) groups in units of 万 (10,000) — 十万 = 100,000, not "10 thousand."
- Memory tip: 万 matches Chinese 萬's grouping logic exactly — trust your Chinese number intuition here, not English.

**14. 円（えん）— 日圓**
- 例文: これはいくらですか。三百円です。
- 翻譯: 這個多少錢？三百日圓。
- Common mistake: Writing "¥" or "元" instead of 円 in Japanese context, or confusing 円 (yen, currency) with 円い (round, shape) — different words sharing a kanji root.
- Memory tip: 円 also means "circle" — coins are round, so 円 = the round currency.

**15. 歳（さい）— 歲**
- 例文: 私は二十歳です。
- 翻譯: 我二十歲。
- Common mistake: Forgetting that 20 years old has a special irregular reading はたち, not にじゅっさい (both exist, but はたち is the traditional/common one) — a classic N5 exception.
- Memory tip: Memorize はたち as a special case just for "20 years old," like a memorized idiom.

**16. いくつ — 幾個**
- 例文: お子さんはいくつですか。
- 翻譯: 您的孩子幾歲？（也可問數量）
- Common mistake: Using いくつ only for age — it also generally means "how many," so context (with people vs. objects) determines the meaning.
- Memory tip: いくつ = "how many/how old" — a flexible catch-all question word for countable things.

**17. いくら — 多少錢**
- 例文: これはいくらですか。
- 翻譯: 這個多少錢？
- Common mistake: Confusing いくつ (how many/how old) with いくら (how much money) — similar sound, different meaning; いくら is strictly about price/amount.
- Memory tip: いくら sounds like "ikura" (salmon roe) — expensive food, easy to link to "how much does it cost?"

**18. 半分（はんぶん）— 一半**
- 例文: これを半分ください。
- 翻譯: 請給我一半。
- Common mistake: Confusing 半分 (half, a noun) with 半（はん, "half past" in time expressions) — related but used differently.
- Memory tip: 半 (half) + 分 (part/minute) = "a half part."

**19. 全部（ぜんぶ）— 全部**
- 例文: 全部でいくらですか。
- 翻譯: 全部多少錢？
- Common mistake: Word order — learners sometimes put 全部 at the end like Chinese "全部多少錢" structure would suggest, but Japanese typically places 全部で at the start of the price question.
- Memory tip: 全 (all) + 部 (part) = "all the parts" = everything.

**20. 同じ（おなじ）— 一樣**
- 例文: これとそれは同じです。
- 翻譯: 這個和那個一樣。
- Common mistake: Treating 同じ like a normal な-adjective and adding な before a noun (❌ 同じな物) — 同じ is an exception that attaches directly to nouns without な (同じ物).
- Memory tip: Memorize 同じ as an irregular na-adjective exception early — it will save confusion later.

**21. 電話番号（でんわばんごう）— 電話號碼**
- 例文: 電話番号は何番ですか。
- 翻譯: 電話號碼是幾號？
- Common mistake: Forgetting that phone numbers are read digit-by-digit (using this lesson's number words) rather than as one big number — 090 is read ゼロ・キュウ・ゼロ, not as "ninety."
- Memory tip: 電話 (telephone) + 番号 (number) = "telephone's number."

**22. お金（おかね）— 錢**
- 例文: お金がありません。
- 翻譯: 沒有錢。
- Common mistake: Dropping the polite prefix お and just saying 金 — 金 alone can sound blunt or refer to "gold"; お金 is the standard, natural word for money.
- Memory tip: お + 金 (gold/metal) = "the honorable gold" = money.

**23. 財布（さいふ）— 錢包**
- 例文: 財布はどこですか。
- 翻譯: 錢包在哪裡？
- Common mistake: Confusing 財布 (wallet) with かばん (bag) — different sizes/objects, easy to mix up as beginner vocabulary.
- Memory tip: 財 (wealth) + 布 (cloth) = "a cloth [pouch] for wealth."

**24. 数える（かぞえる）— 數（動詞）**
- 例文: 一緒に数えましょう。
- 翻譯: 一起數吧。
- Common mistake: Confusing the verb 数える (to count, transitive) with the noun 数字 (すうじ, "number/numeral") introduced later — related root 数 but different words.
- Memory tip: 数 (number) + える (verb ending) = "to make into numbers" = to count.

**25. ちょうど — 剛好**
- 例文: ちょうど千円です。
- 翻譯: 剛好一千日圓。
- Common mistake: Placing ちょうど after the number instead of before it — natural Japanese word order puts ちょうど immediately before the amount (ちょうど千円), not after (千円ちょうど is also acceptable but less common for beginners to produce correctly).
- Memory tip: Think of ちょうど as "exactly/precisely" — useful when giving exact change.

## ② Kanji (14): 一, 二, 三, 四, 五, 六, 七, 八, 九, 十, 百, 千, 万, 円

**一** — Meaning: 一 (one). On: イチ、イツ. Kun: ひと.つ. Words: 一人（ひとり）, 一つ（ひとつ）, 一番. Example: 一つください。 Stroke order: a single left-to-right horizontal stroke. Chinese difference: Identical to Chinese 一; only new element is the kun-reading ひと.

**二** — Meaning: 二 (two). On: ニ. Kun: ふた.つ. Words: 二つ（ふたつ）, 二人（ふたり）, 二十. Example: 二つあります。 Stroke order: two horizontal strokes, top shorter than bottom. Chinese difference: Identical to Chinese 二; irregular readings ふたり／はたち (for 20) should be memorized separately.

**三** — Meaning: 三 (three). On: サン. Kun: みっ.つ. Words: 三つ（みっつ）, 三人（さんにん）, 三月. Example: 三つ食べました。 Stroke order: three horizontal strokes, evenly spaced. Chinese difference: Identical to Chinese 三; note さん also happens to sound like the honorific さん (different kanji, no relation).

**四** — Meaning: 四 (four). On: シ. Kun: よ.つ、よん. Words: 四つ（よっつ）, 四月, 四人（よにん）. Example: 四月に生まれました。 Stroke order: outer box first, then the two internal strokes. Chinese difference: Same shape as Chinese 四; Japanese avoids the on-reading し in casual speech due to its association with 死 (death), unlike Chinese which has no such taboo with sì.

**五** — Meaning: 五 (five). On: ゴ. Kun: いつ.つ. Words: 五つ（いつつ）, 五月, 五人（ごにん）. Example: 五月は忙しいです。 Stroke order: top horizontal, then the zigzag body, then bottom horizontal. Chinese difference: Shape differs slightly from common Chinese handwriting styles but is the standard kanji form; meaning is unchanged.

**六** — Meaning: 六 (six). On: ロク. Kun: むっ.つ. Words: 六つ（むっつ）, 六月, 六人（ろくにん）. Example: 六時に寝ます。 Stroke order: top dot, then the left and right strokes forming a small "roof" shape. Chinese difference: Identical to Chinese 六; meaning and usage match closely.

**七** — Meaning: 七 (seven). On: シチ. Kun: なな.つ. Words: 七つ（ななつ）, 七月, 七人（しちにん／ななにん）. Example: 七つあります。 Stroke order: horizontal stroke first, then the vertical hook stroke crossing down through it. Chinese difference: Similar shape to Chinese 七; both readings しち／なな are acceptable, unlike single-reading Chinese qī.

**八** — Meaning: 八 (eight). On: ハチ. Kun: やっ.つ. Words: 八つ（やっつ）, 八月, 八人（はちにん）. Example: 八月は暑いです。 Stroke order: left stroke sweeping down-left, right stroke sweeping down-right, not touching at the bottom. Chinese difference: Identical shape to Chinese 八; meaning unchanged.

**九** — Meaning: 九 (nine). On: キュウ、ク. Kun: ここの.つ. Words: 九つ（ここのつ）, 九月（くがつ）, 九人（きゅうにん）. Example: 九月に日本へ行きます。 Stroke order: curved stroke first, then the hooking diagonal stroke. Chinese difference: Identical to Chinese 九; note the special reading く only in 九月 (くがつ), otherwise きゅう.

**十** — Meaning: 十 (ten). On: ジュウ. Kun: とお. Words: 十（じゅう）, 二十（にじゅう）, 十人（じゅうにん）. Example: 全部で十人います。 Stroke order: horizontal stroke first, then vertical stroke crossing through the center. Chinese difference: Identical cross shape to Chinese 十; meaning unchanged.

**百** — Meaning: 百 (hundred). On: ヒャク. Kun: none in common use. Words: 百円, 三百（さんびゃく）, 八百（はっぴゃく）. Example: 百円ください。 Stroke order: write 一 on top, then 白 (white) beneath it. Chinese difference: Identical to Chinese 百; sound changes in compounds (三百, 六百, 八百) don't exist in Mandarin's regular counting, so these must be memorized individually.

**千** — Meaning: 千 (thousand). On: セン. Kun: none in common use. Words: 千円, 三千（さんぜん）, 八千（はっせん）. Example: 千円札があります。 Stroke order: top-left short diagonal stroke, then the long horizontal stroke, then the vertical stroke down through the center. Chinese difference: Identical to Chinese 千; again watch for voiced sound changes in compounds (三千 → さんぜん).

**万** — Meaning: 萬 (ten thousand). On: マン、バン. Kun: none in common use. Words: 一万円（いちまんえん）, 十万（じゅうまん）, 万一（まんいち）. Example: 一万円です。 Stroke order: short top horizontal stroke, then a long sweeping curved stroke below it. Chinese difference: This is the simplified Japanese/Simplified-Chinese-style form of Traditional Chinese 萬 used in Taiwan — recognize that Japanese numeral kanji sometimes match Simplified Chinese forms rather than Traditional ones.

**円** — Meaning: 圓/元 (yen, circle). On: エン. Kun: none in common use. Words: 百円, 千円, 円い（まるい, "round"）. Example: これは五百円です。 Stroke order: outer frame first (like 冂 with a bottom stroke), then the two internal horizontal strokes. Chinese difference: This is the simplified Japanese form corresponding to Traditional Chinese 圓 (round/currency); Taiwanese learners will recognize the shape as close to Simplified Chinese 园/元-adjacent forms rather than Traditional 圓/元.

## ③ Grammar (2 points)

### Grammar 1 — 〜も ("also, too")
**Meaning:** Adds "also/too" to a noun, replacing は or が in that position.
**Structure:** Noun + も + です／predicate
**Examples:**
1. 私は学生です。あなたも学生です。(I am a student. You are also a student.)
2. これは百円です。それも百円です。(This is 100 yen. That is also 100 yen.)
3. 弟も台湾人です。(My younger brother is also Taiwanese.)

**Common mistakes (Chinese speakers):**
- ❌ 私は学生ですもあなた → ⭕ 私は学生です。あなたも学生です — も must directly replace は on the second noun; it cannot be tacked onto です itself.
- ❌ Using も when there is no shared property being compared (using it just to mean "and") — も strictly means "also/too" for the same predicate, not "and" as a general connector.

**Exercises (translate into Japanese):**
1. I am a student. You are also a student.
2. This is 500 yen. That is also 500 yen.
3. My older sister is also a teacher.

### Grammar 2 — 〜の (possessive/modifying particle: A の B)
**Meaning:** Connects two nouns where the first modifies/possesses the second ("A's B").
**Structure:** Noun A + の + Noun B
**Examples:**
1. これは私の財布です。(This is my wallet.)
2. 田中さんのお金です。(It's Mr. Tanaka's money.)
3. 大学の学生です。(A university's student = a university student.)

**Common mistakes (Chinese speakers):**
- ❌ Dropping の because Chinese sometimes omits 的 in tight compounds (❌ 私財布) → ⭕ 私の財布 — の is required in Japanese possessive phrases even where Chinese might drop 的.
- ❌ Using の between an adjective and noun where it isn't needed — の is only for noun+noun, not adjective+noun (大きい家, not 大きいの家).

**Exercises (translate into Japanese):**
1. This is my wallet.
2. This is Mr. Tanaka's phone number.
3. She is a university student.

## 🇹🇼 Chinese Notes

- **万（まん）**: Japanese groups large numbers in units of 万 (10,000), exactly like Chinese 萬 — trust your Chinese-number intuition here rather than English "thousand" grouping. 十万 = 100,000, not "ten thousand plus."
- **円（えん）**: The kanji looks similar to shapes in Simplified Chinese but represents Japan's currency (yen), not the Taiwanese dollar. Don't confuse 円 with Chinese 圓/元 in meaning — they're currency-related cognates, not interchangeable in cross-context translation.
- **歳（さい）**: Used identically to Chinese 歲 for age, but Japanese has one true irregular exception: 20 years old is normally はたち, not にじゅっさい — there's no equivalent irregularity in Mandarin's 二十歲.

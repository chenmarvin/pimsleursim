### Vocabulary in Depth

**1. 電車（でんしゃ）— train**
- Example: 電車で学校へ行きます。
- Translation: 我搭電車去學校。
- Common mistake: Confusing 電車（でんしゃ, an electric commuter train）with 汽車（きしゃ, an old-fashioned steam train, rarely used today）— for everyday train travel, 電車 is the word you want, not 汽車.
- Memory tip: 電（electric）+ 車（vehicle）= "electric vehicle [on rails]."

**2. バス — bus**
- Example: バスは電車より遅いです。
- Translation: 公車比電車慢。
- Common mistake: Using に instead of で with the means-of-transport particle — バスに乗ります (ride ON the bus) uses に because 乗る specifically takes に, but "traveling BY bus" in other sentences uses で: バスで行きます. Learners often apply で to 乗る by mistake.
- Memory tip: From English "bus" — pronounced ba-su.

**3. タクシー — taxi**
- Example: 駅までタクシーで行きましょう。
- Translation: 我們搭計程車去車站吧。
- Common mistake: Assuming タクシー is always cheaper because it's convenient — in Japan taxis are typically more expensive than trains/buses; don't confuse convenience with cost.
- Memory tip: From English "taxi" — ta-ku-shii.

**4. 飛行機（ひこうき）— airplane**
- Example: 飛行機で台湾へ帰ります。
- Translation: 我搭飛機回台灣。
- Common mistake: Using 空港（airport, the place）where 飛行機（airplane, the vehicle）is meant — 空港へ行きます (go to the airport) vs. 飛行機に乗ります (board the plane).
- Memory tip: 飛（fly）+ 行（go）+ 機（machine）= "a flying-going machine."

**5. 車（くるま）— car**
- Example: 車で買い物に行きます。
- Translation: 我開車去買東西。
- Common mistake: Using 車 to mean "vehicle in general" in formal writing where 自動車（じどうしゃ）is more standard — in casual speech 車 alone is fine and by far the most common word.
- Memory tip: Shares the kanji 車 with 電車 and 自転車 — think of 車 as the base "wheeled vehicle" kanji.

**6. 自転車（じてんしゃ）— bicycle**
- Example: 自転車で駅まで行きます。
- Translation: 我騎腳踏車去車站。
- Common mistake: Using 乗ります with を instead of に — 自転車に乗ります (ride a bicycle) always takes に, never を, even though English "ride A BICYCLE" suggests an object marker.
- Memory tip: 自（self）+ 転（turn/rotate）+ 車（vehicle）= "a self-turning vehicle," i.e., you pedal it yourself.

**7. 空港（くうこう）— airport**
- Example: 空港まで電車で行きます。
- Translation: 我搭電車去機場。
- Common mistake: Confusing 空港（くうこう, airport）with 港（みなと, seaport) — 空港 is specifically for air travel; the shared kanji 港 (harbor) just means "port" generally.
- Memory tip: 空（sky）+ 港（port）= "sky port."

**8. 切符（きっぷ）— ticket**
- Example: 切符を買ってください。
- Translation: 請買車票。
- Common mistake: Using チケット (a loanword usually for event/movie tickets) interchangeably with 切符 (specifically transportation tickets) — the two overlap in modern usage, but 切符 is the traditional word for train/bus tickets.
- Memory tip: 切（cut）+ 符（tally/token）= a ticket you get "cut" or punched at the gate.

**9. 乗る（のる）— to ride**
- Example: 九時の電車に乗ります。
- Translation: 我要搭九點的電車。
- Common mistake: Using を instead of に — the vehicle you board takes に: 電車に乗る, not ❌ 電車を乗る (a very common N5 error, since English "ride A TRAIN" implies an object).
- Memory tip: Pair with its opposite 降りる below: 乗る (get on) → 降りる (get off).

**10. 降りる（おりる）— to get off**
- Example: 次の駅で降ります。
- Translation: 我在下一站下車。
- Common mistake: Using から instead of で — the station where you get off takes で: 次の駅で降ります, treating it like the location of the action "getting off," not a starting point.
- Memory tip: Shares its kanji-less reading pattern with 下りる (to go down) — both involve downward/exiting motion.

**11. 歩く（あるく）— to walk**
- Example: 駅まで歩いて十分です。
- Translation: 走到車站要十分鐘。
- Common mistake: Using で with 歩く when describing walking AS a means of transport in the pattern "歩いて + time," instead learners sometimes insert an unnecessary で: ❌ 歩くで — the correct te-form pattern is 歩いて (no で needed here).
- Memory tip: Pair with 走る (to run, Lesson 14) — 歩く is the slower, everyday counterpart.

**12. 運転する（うんてんする）— to drive**
- Example: 父は車を運転します。
- Translation: 爸爸開車。
- Common mistake: Using 乗る (to ride) where 運転する (to actively drive/operate) is meant — 車に乗る just means "get in/ride in a car" (as a passenger too), while 運転する specifically means being the driver.
- Memory tip: 運（transport/carry）+ 転（turn/rotate the wheels）= "operating the turning [wheels] to transport."

**13. 速い（はやい）— fast**
- Example: 電車はバスより速いです。
- Translation: 電車比公車快。
- Common mistake: Confusing 速い（はやい, fast in speed）with 早い（はやい, early in time）— identical pronunciation, different kanji and meaning; 電車が速い (the train is fast) vs. 朝早く起きる (wake up early).
- Memory tip: 速 contains 束 (bundle) + the movement radical 辶 — think of "bundling up distance quickly."

**14. 遅い（おそい）— slow**
- Example: このバスは遅いです。
- Translation: 這班公車很慢。
- Common mistake: Using 遅い to mean only "late" (as in Chinese 遲) without realizing it also plainly means "slow" in speed — バスは遅い primarily describes slowness, not lateness, though 遅い also covers "late" in other contexts (e.g. 遅く起きる).
- Memory tip: Opposite of 速い — always drill adjective pairs together: 速い⇔遅い.

**15. 事故（じこ）— accident**
- Example: 今日、電車の事故がありました。
- Translation: 今天電車發生了事故。
- Common mistake: Confusing 事故（じこ, an accident/mishap）with 用事（ようじ, Lesson 16, an errand）— similar-feeling "事" compounds with very different meanings.
- Memory tip: 事（matter）+ 故（reason/cause, also "old/former"）— historically "an unfortunate incident."

**16. 渋滞（じゅうたい）— traffic jam**
- Example: 道が渋滞しています。
- Translation: 路上塞車了。
- Common mistake: Treating 渋滞 as a verb directly (❌ 渋滞ます) instead of using it with する: 渋滞しています (a traffic jam is occurring right now).
- Memory tip: 渋（astringent/stagnant）+ 滞（stagnate/delay）= "stagnant and delayed," a vivid image of cars not moving.

**17. 時刻表（じこくひょう）— timetable**
- Example: 時刻表で電車の時間を調べます。
- Translation: 我查時刻表確認電車時間。
- Common mistake: Confusing 時刻表（じこくひょう, a schedule/timetable document）with 予定（よてい, Lesson 4, a personal plan/schedule) — 時刻表 refers specifically to published transit times.
- Memory tip: 時刻（time/moment）+ 表（table/chart）= "a chart of times."

**18. 乗り換える（のりかえる）— to transfer**
- Example: 次の駅で乗り換えてください。
- Translation: 請在下一站轉乘。
- Common mistake: Using を with the destination line instead of に, or forgetting the base verb 乗る hiding inside 乗り換える — remember it literally means "ride and change": 乗る (ride) + 換える (change/exchange).
- Memory tip: Built directly from 乗る (this lesson) + 換える (to exchange) — once you know 乗る, this compound is easy to decode.

**19. 定期券（ていきけん）— commuter pass**
- Example: 定期券を持っていますから、切符はいりません。
- Translation: 我有定期票，所以不需要車票。
- Common mistake: Confusing 定期券（a prepaid pass for regular commuting, valid on a fixed route）with just buying 切符 every time — 定期券 is a subscription-like pass, not a single-use ticket.
- Memory tip: 定期（fixed period）+ 券（ticket/coupon）= "a ticket for a fixed period."

**20. 運賃（うんちん）— fare**
- Example: バスの運賃はいくらですか。
- Translation: 公車車資多少錢？
- Common mistake: Using 値段（ねだん, Lesson 11, price of goods）instead of 運賃 for transportation costs specifically — 運賃 is the specialized word for fares.
- Memory tip: 運（transport）+ 賃（wages/fee）= "the fee for being transported."

**21. 混んでいる（こんでいる）— crowded**
- Example: 朝の電車はいつも混んでいます。
- Translation: 早上的電車總是很擁擠。
- Common mistake: Treating 混んでいる as a plain adjective and trying to conjugate it like an い-adjective (❌ 混んでいくない) — it's actually the te-form of 混む + いる (a state resulting from an action), so negate the いる part: 混んでいません.
- Memory tip: Pair directly with its opposite 空いている below — both describe an ongoing state using 〜ている.

**22. 空いている（すいている）— empty, not crowded**
- Example: 今の時間は電車が空いています。
- Translation: 現在這個時間電車很空。
- Common mistake: Confusing 空いている（すいている, not crowded）with 空いている（あいている, "to be open/vacant," e.g. a store) — same kanji, different readings and closely related but distinct meanings; context (transportation vs. shops/schedules) usually disambiguates.
- Memory tip: Shares the kanji 空 (empty/sky) with 空港 (airport) in this same lesson.

**23. 到着する（とうちゃくする）— to arrive**
- Example: 飛行機は三時に到着します。
- Translation: 飛機三點抵達。
- Common mistake: Using 着く（つく, a simpler, more casual verb for "arrive"）interchangeably without realizing 到着する is the more formal/written equivalent, common on timetables and announcements.
- Memory tip: Pair directly with its opposite below: 到着する (arrive) ⇔ 出発する (depart).

**24. 出発する（しゅっぱつする）— to depart**
- Example: バスは八時に出発します。
- Translation: 公車八點出發。
- Common mistake: Using 始まる（Lesson 16, "to begin," for events/classes）where 出発する（specifically for a journey/vehicle leaving）is meant — different verbs for "starting a class" vs. "a vehicle departing."
- Memory tip: 出（exit/go out）+ 発（set off）= "setting off and going out."

**25. 地図（ちず）— map**
- Example: 地図を見て、駅を探します。
- Translation: 我看地圖找車站。
- Common mistake: Confusing 地図（ちず, map）with 時刻表（じこくひょう, timetable) — both are travel documents but show completely different information.
- Memory tip: 地（land/ground）+ 図（diagram）= "a diagram of the land."

## ② Kanji (3): 電, 車, 道

**電（electricity）**
- Meaning (Chinese): 電
- On-reading: デン
- Kun-reading: — (no common kun-reading at N5 level)
- Words: 電車（でんしゃ）, 電話番号（でんわばんごう, Lesson 2）, 電気（でんき, Lesson 18）
- Example sentence: 電車で会社へ行きます。
- Stroke order tip: The 雨 (rain) radical goes on top first (its own four strokes), then 电-like body underneath — 13 strokes total; think of electricity as "energy falling from the rain/sky."
- Chinese difference: Nearly identical to Chinese 電 (traditional); Taiwanese learners will recognize it immediately, unlike mainland China's simplified 电.

**車（car, vehicle）**
- Meaning (Chinese): 車
- On-reading: シャ
- Kun-reading: くるま
- Words: 車（くるま）, 電車（でんしゃ）, 自転車（じてんしゃ）
- Example sentence: この車は新しいです。
- Stroke order tip: A vertical line straight down through the middle of a box-like frame with a horizontal cross-stroke — 7 strokes, resembling a simplified wheel-and-axle image.
- Chinese difference: Identical to Chinese 車 (traditional) in shape and meaning; on-reading シャ appears in compounds like 電車, 自転車, while the everyday standalone word "car" uses the kun-reading くるま — Chinese has no reading distinction like this, so remember which reading a word calls for.

**道（road, way）**
- Meaning (Chinese): 道
- On-reading: ドウ
- Kun-reading: みち
- Words: 道（みち, Lesson 7）, 北海道（ほっかいどう, Hokkaido）, 道路（どうろ, roadway）
- Example sentence: この道はいつも渋滞しています。
- Stroke order tip: Write the 首 (head) component first, then add the 辶 movement radical last, wrapping around the bottom-left — 12 strokes.
- Chinese difference: Same shape and meaning as Chinese 道 (road, also "way/path/doctrine" as in 道理); Japanese extends it further into words like 柔道（じゅうどう, judo）and 書道（しょどう, calligraphy) for "the way/art of ~," a metaphorical usage Chinese shares only partially (武道 exists, but 道 as a suffix is far more productive in Japanese).

**Kanji Review:** Can you read these? 電車／車／道／自転車

## ③ Grammar

### Grammar 1 — 〜と〜と、どちらが〜ですか (which is more...? comparison)

**Meaning/Structure:** To ask which of two options is more X, list both items each followed by と, then ask どちらが〜ですか ("which one is more ~?").
Structure: A と B と、どちらが [adjective] ですか。

**Examples:**
- 電車とバスと、どちらが速いですか。(Which is faster, the train or the bus?)
- コーヒーとお茶と、どちらが好きですか。(Which do you like more, coffee or tea?)
- 車と自転車と、どちらが便利ですか。(Which is more convenient, a car or a bicycle?)

**Common mistakes (Chinese speakers):**
- ❌ 電車バスと、どちらが速いですか → ⭕ 電車とバスと、どちらが速いですか — both compared items need their own と; dropping the first と is a common omission influenced by Chinese, which doesn't need a connective particle here.
- ❌ どちらは速いですか → ⭕ どちらが速いですか — どちら is the target of a question about a quality, so it takes が, not は.

**Exercises (translate into Japanese):**
1. Which is faster, the train or the bus?
2. Which do you like more, coffee or tea?
3. Which is more convenient, a car or a bicycle?
4. Which is more famous, this school or that school?

### Grammar 2 — 〜より〜のほうが〜です (X is more ... than Y)

**Meaning/Structure:** To directly compare two things and state that one is more X, mark the "losing" side with より ("than") and the "winning" side with のほうが ("the one that is more").
Structure: [B] より [A] のほうが [adjective] です。("Compared to B, A is more ~.")

**Examples:**
- バスより電車のほうが速いです。(The train is faster than the bus.)
- 飛行機より電車のほうが安いです。(The train is cheaper than the plane.)
- 歩くより自転車のほうが便利です。(A bicycle is more convenient than walking.)

**Common mistakes (Chinese speakers):**
- ❌ バス電車のほうが速いです → ⭕ バスより電車のほうが速いです — omitting より, the essential marker for "than," often because Chinese 比 sits in a different position in the sentence.
- ❌ バスが電車のほうが速いです → ⭕ バスより電車のほうが速いです — using が instead of より for the item being compared against.

**Exercises (translate into Japanese):**
1. The airplane is faster than the train.
2. Coffee is more delicious than tea.
3. The bicycle is more convenient than walking.
4. This exam was more difficult than last week's exam.

## ④ Reading

私は毎日、電車で学校へ行きます。駅まで歩いて、十分ぐらいです。電車はバスより速いですが、朝はいつも混んでいます。今日は電車の事故があって、少し遅かったです。友達は自転車で来ますが、雨の日は大変だと言っていました。週末、友達と電車で新しい町へ行く予定です。出発する前に、地図を見ました。

**Comprehension Questions:**
1. 私は学校まで何で行きますか。
2. どうして今日は少し遅かったですか。
3. 友達はどうやって学校に来ますか。

**Answer Key:**
1. 電車で行きます。(By train.)
2. 電車の事故があったからです。(Because there was a train accident.)
3. 自転車で来ます。(By bicycle.)

## ⑤ Listening

*Read once slowly, then once at natural speed.*

A: すみません、空港までどう行きますか。
B: 電車が一番速いですよ。バスより速くて、便利です。
A: 切符はどこで買いますか。

A: どのぐらい時間がかかりますか。
B: 三十分ぐらいです。今の時間は空いていますから、大丈夫です。

**Comprehension Questions:**
1. 空港まで何で行くのが一番速いですか。
2. 空港まで、どのぐらい時間がかかりますか。

**Answer Key:**
1. 電車です。(The train.)
2. 三十分ぐらいです。(About thirty minutes.)

## 🇹🇼 Chinese Notes

- **汽車 is a false friend**: In Chinese, 汽車 means "car/automobile." In Japanese, 汽車（きしゃ）means an old-fashioned steam train and is rarely used in modern conversation — for "car," use 車 or 自動車; for "train," use 電車.
- **切符 has no direct Chinese equivalent**: Chinese speakers usually think in terms of 車票; 切符 is a distinctly Japanese word for a transportation ticket and doesn't decode transparently from its kanji (切 = cut, 符 = token/tally) the way many Sino-Japanese compounds do.

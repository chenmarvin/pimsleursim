# 🇯🇵 Japanese for Taiwanese Learners — JLPT N5, Lesson 26

**Theme:** 「旅行に行きます」(I'm going on a trip)

**Objectives**
- Talk about travel plans
- Express obligation with 〜なければなりません
- Say what's not required with 〜なくてもいいです

*(Note: this lesson's actual grammar points, per the syllabus, are 〜前に／〜てから／〜ながら — see ③ Grammar below. 〜なければなりません and 〜なくてもいいです are covered in Lesson 31.)*

### Word-by-word details

**1. 旅行（りょこう）— 旅行**
- 例文: 来月、日本へ旅行します。
- 翻譯: 下個月要去日本旅行。
- Common mistake: Learners often mark the destination with を like English "travel Japan" (❌ 日本を旅行します as the default choice) instead of へ／に for "travel *to*."
- Memory tip: 旅（journey）＋行（go）＝ literally "journey-going" — identical meaning and characters to Chinese 旅行, one of the easiest words in this lesson.

**2. ホテル — 飯店**
- 例文: ホテルを予約しました。
- 翻譯: 我已經訂了飯店。
- Common mistake: Treating ホテル and 旅館 (#25) as interchangeable — ホテル is specifically a Western-style hotel, while 旅館 implies tatami rooms, futon, and included meals.
- Memory tip: Katakana loanword from English "hotel" — pronounce all vowels clearly: ho-te-ru.

**3. 観光地（かんこうち）— 觀光景點**
- 例文: ここは有名な観光地です。
- 翻譯: 這裡是有名的觀光景點。
- Common mistake: Misreading 観光 as こうかん (confusing it with 交換 "exchange," Lesson 22) since both share the こう sound.
- Memory tip: 観（view）＋光（scenery/light）＋地（place）＝ "a place to view scenery."

**4. パスポート — 護照**
- 例文: パスポートを見せてください。
- 翻譯: 請給我看護照。
- Common mistake: Mispronouncing the long vowels, saying "パスポト" instead of パスポート.
- Memory tip: Direct loanword from English "passport" — easy to recognize once the katakana spelling is fixed in memory.

**5. 予約する（よやくする）— 預約**
- 例文: レストランを予約します。
- 翻譯: 我要預約餐廳。
- Common mistake: Using 約束する (to promise / make an appointment with a *person*) where 予約する (to reserve a *facility or service*) is meant — these are not interchangeable.
- Memory tip: 予（in advance）＋約（arrange）＝ "arrange in advance" = to reserve.

**6. 観光する（かんこうする）— 觀光**
- 例文: 京都で観光します。
- 翻譯: 在京都觀光。
- Common mistake: Overusing 見る (to see) where 観光する better captures "sightseeing" as a whole activity, not just looking at one thing.
- Memory tip: Same 観光 as in 観光地 (#3), plus する — "to do sightseeing."

**7. お土産（おみやげ）— 名產、伴手禮**
- 例文: 家族にお土産を買いました。
- 翻譯: 我買了名產給家人。
- Common mistake: Confusing お土産 with プレゼント (a gift for any occasion) — お土産 specifically means something brought back from a trip to share, often local food.
- Memory tip: 土（local/regional）＋産（product）＝ "a product of the region."

**8. 泊まる（とまる）— 住宿、過夜**
- 例文: 旅館に一泊泊まります。
- 翻譯: 在旅館住一晚。
- Common mistake: Confusing with 止まる (とまる, "to stop," e.g. a car) — identical pronunciation, different kanji/meaning; a place + に signals "stay overnight."
- Memory tip: ホテルに泊まる = "to stay at a hotel" — always pair with a location + に.

**9. 案内する（あんないする）— 導覽、帶路**
- 例文: 観光地を案内してもらいました。
- 翻譯: 有人帶我去觀光景點導覽。
- Common mistake: Using 紹介する (to introduce a person, Lesson 28) when 案内する (to guide someone through a place) is meant.
- Memory tip: 案（plan）＋内（inside）＝ "leading someone through a plan" = to guide.

**10. 荷物（にもつ）— 行李**
- 例文: 荷物が重いです。
- 翻譯: 行李很重。
- Common mistake: Using カバン (bag) when the broader term 荷物 (luggage/cargo in general) is meant.
- Memory tip: 荷（cargo）＋物（thing）＝ "cargo thing" = luggage.

**11. 外国（がいこく）— 外國**
- 例文: 外国で働きたいです。
- 翻譯: 我想在國外工作。
- Common mistake: Confusing 外国 (foreign country, a place) with 外国人 (a foreigner, a person) and forgetting to add 人 when talking about a person.
- Memory tip: 外（outside）＋国（country）＝ "outside country."

**12. 国（くに）— 國家**
- 例文: あなたの国はどこですか。
- 翻譯: 你的國家是哪裡？
- Common mistake: Using 国 where a specific country name (台湾, 日本) is expected — 国 alone is the generic noun "country."
- Memory tip: The core building block already seen in 外国, 中国 (Lesson 1) — reused constantly.

**13. 文化（ぶんか）— 文化**
- 例文: 日本の文化に興味があります。
- 翻譯: 我對日本文化有興趣。
- Common mistake: Misreading 文化 as 文法 (ぶんぽう, "grammar") — visually similar first kanji 文.
- Memory tip: 文（civilization/writing）＋化（-ization）＝ "civilization-ization" = culture.

**14. 習慣（しゅうかん）— 習慣**
- 例文: 国によって習慣が違います。
- 翻譯: 每個國家的習慣不同。
- Common mistake: Confusing with 週間 (しゅうかん, "week [duration]") — identical pronunciation, different kanji and meaning.
- Memory tip: 習（practice, from 習う "to learn," Lesson 22）＋慣（accustomed）＝ "a practiced, accustomed way."

**15. 言葉（ことば）— 詞語、語言**
- 例文: この言葉の意味が分かりません。
- 翻譯: 我不懂這個詞的意思。
- Common mistake: Using 言葉 interchangeably with 語 (the suffix in 日本語) — 言葉 is a general noun for "word(s)/language as spoken," while 語 attaches to a country name.
- Memory tip: 言（say）＋葉（leaf, used here as a poetic suffix）＝ "leaves of speech" = words.

**16. 漢字（かんじ）— 漢字**
- 例文: 漢字を勉強するのは難しいです。
- 翻譯: 學漢字很難。
- Common mistake: Assuming every kanji meaning transfers directly from Chinese 漢字 — many do, but readings and some meanings (false friends) differ, so don't assume automatically.
- Memory tip: 漢（Han/China）＋字（character）＝ the same word essentially as Chinese 漢字.

**17. 英語（えいご）— 英語**
- 例文: 英語が少し話せます。
- 翻譯: 我會說一點英語。
- Common mistake: Dropping the 語 suffix and saying just 英 to mean "the English language" — 語 must be attached.
- Memory tip: 英（England, from 英国）＋語（language）＝ "the language of England."

**18. 日本語（にほんご）— 日語**
- 例文: 日本語を勉強しています。
- 翻譯: 我正在學日文。
- Common mistake: Confusing 日本語 (the language) with 日本人 (にほんじん, the nationality/people) — both start with 日本 but the suffix changes everything.
- Memory tip: 日本（Japan）＋語（language）— the same construction pattern as 英語.

**19. 両替する（りょうがえする）— 換錢**
- 例文: 空港でお金を両替します。
- 翻譯: 在機場換錢。
- Common mistake: Using 交換する (こうかんする, general "exchange," Lesson 22) for currency — 両替する is the specific term for exchanging money.
- Memory tip: 両（both sides）＋替（change/replace）＝ "changing between two [currencies]."

**20. 免税店（めんぜいてん）— 免稅店**
- 例文: 空港の免税店でお土産を買いました。
- 翻譯: 在機場的免稅店買了名產。
- Common mistake: Dropping 店 and saying just 免税 to mean "the shop" — 免税 alone only means "tax-exempt."
- Memory tip: 免（exempt）＋税（tax）＋店（shop）＝ "tax-exempt shop."

**21. ツアー — 旅行團**
- 例文: ツアーに参加します。
- 翻譯: 參加旅行團。
- Common mistake: Confusing ツアー (an organized, guided tour) with plain 旅行 (travel in general, can be independent).
- Memory tip: Katakana loanword from English "tour," pronounced with a long ー: tsu-a-a.

**22. 一人旅（ひとりたび）— 一個人旅行**
- 例文: 一人旅が好きです。
- 翻譯: 我喜歡一個人旅行。
- Common mistake: Constructing ❌ 一人の旅行 instead of using the natural fixed compound 一人旅.
- Memory tip: 一人（one person, Lesson 1's 人）＋旅（journey）＝ "solo journey."

**23. 海外（かいがい）— 海外、國外**
- 例文: 海外旅行をしたことがあります。
- 翻譯: 我有出國旅行過的經驗。
- Common mistake: Treating 海外 and 外国 (#11) as fully interchangeable — 海外 emphasizes crossing the sea (natural for island nations like Japan/Taiwan going anywhere abroad), while 外国 more neutrally means "a foreign country."
- Memory tip: 海（sea）＋外（outside）＝ "outside the sea" = overseas.

**24. 記念（きねん）— 紀念**
- 例文: 記念に写真を撮りました。
- 翻譯: 拍照留念。
- Common mistake: Confusing with 記憶 (きおく, "memory," Lesson 36) due to the shared 記 kanji and similar sound.
- Memory tip: 記（record）＋念（remembrance）＝ "recorded remembrance" = memento.

**25. 旅館（りょかん）— 日式旅館**
- 例文: 旅館に泊まりたいです。
- 翻譯: 我想住旅館（日式）。
- Common mistake: See #2 — using ホテル and 旅館 interchangeably out of habit from Chinese 旅館, which is more generic; in Japanese 旅館 specifically means a traditional-style inn.
- Memory tip: 旅（journey）＋館（large building/hall）＝ "journey building."

## ② Kanji (5): 行, 外, 国, 言, 語

**行 — Meaning: 行 (go)**
- On-reading: コウ、ギョウ
- Kun-reading: い.く
- Words: 旅行, 行く, 銀行 (preview)
- Example sentence: 旅行に行きます。
- Stroke order: 6 strokes — the left component (two short strokes, like 彳) first, then the right component (four strokes).
- Chinese difference: Same shape as Chinese 行, but Japanese pronunciation varies heavily by context (こう／ぎょう／い.く／おこな.う). Chinese 行 (xíng/háng) also has multiple readings but the verb "to go" is usually 走/去 in Chinese, whereas 行く is the everyday "go" verb in Japanese.

**外 — Meaning: 外 (outside)**
- On-reading: ガイ
- Kun-reading: そと
- Words: 外国, 外, 海外
- Example sentence: 外国に行きます。
- Stroke order: 5 strokes — the 夕-shape first, then the ト stroke added to the right.
- Chinese difference: Same meaning as Chinese 外, but Japanese uses the kun-reading そと for physical "outside" and the on-reading ガイ in compounds like 外国, while Chinese uses a single reading wài for both.

**国 — Meaning: 國 (country)**
- On-reading: コク
- Kun-reading: くに
- Words: 外国, 国, 中国
- Example sentence: 台湾はいい国です。
- Stroke order: 8 strokes — the outer box 囗 first (left side, then top+right, then bottom), then 玉 drawn inside.
- Chinese difference: Japanese 国 is the shinjitai (simplified-style) form with 玉 inside 囗, versus Traditional Chinese 國 (with 或 inside) used in Taiwan — same meaning, visibly different shape.

**言 — Meaning: 言 (say, word)**
- On-reading: ゲン、ゴン
- Kun-reading: い.う、こと
- Words: 言葉, 言う
- Example sentence: 言葉が分かりません。
- Stroke order: 7 strokes — a short top stroke, then three horizontal-ish strokes, finishing with 口 at the bottom.
- Chinese difference: Same character as Chinese 言, but modern Chinese rarely uses 言 alone as a verb (说/讲 are used instead), whereas Japanese 言う remains the everyday verb "to say."

**語 — Meaning: 語 (language, word)**
- On-reading: ゴ
- Kun-reading: かた.る
- Words: 日本語, 英語
- Example sentence: 日本語を勉強します。
- Stroke order: 14 strokes — the 言 radical on the left (7 strokes), then 吾 on the right (五 then 口, 7 strokes).
- Chinese difference: Identical to the Traditional Chinese form 語 used in Taiwan, and the meaning "language" matches exactly — one of the most reassuring kanji in this lesson.

## ③ Grammar

### Grammar 1 — 〜前に (before doing)
**Meaning:** States that one action happens before another, or before a point in time.
**Structure:** V(dictionary form) ＋ 前に、〜 ／ Noun ＋ の ＋ 前に、〜
**Examples:**
1. 旅行に行く前に、パスポートを見ます。(Before going on the trip, I check my passport.)
2. 寝る前に、歯を磨きます。(Before sleeping, I brush my teeth.)
3. 旅行の前に、お金を両替します。(Before the trip, I exchange money.)

**Common mistakes (Chinese speakers):**
- ❌ 行った前に (using the た-form) → ⭕ 行く前に — 前に always attaches to the plain dictionary form, regardless of when the main action happens.
- ❌ 旅行前に as a fixed rule → with most nouns, standard textbook Japanese inserts の: 旅行の前に.

**Exercises (translate into Japanese):**
1. Before eating, I say いただきます.
2. Before the trip, I reserve a hotel.

### Grammar 2 — 〜てから (after doing, and then)
**Meaning:** Emphasizes that one action is completed before the next one starts.
**Structure:** V(て-form) ＋ から、〜
**Examples:**
1. パスポートを見せてから、飛行機に乗ります。(After showing my passport, I board the plane.)
2. 日本へ行ってから、観光地を見ます。(After going to Japan, I'll see the tourist spots.)
3. 荷物を置いてから、観光に行きます。(After putting down my luggage, I'll go sightseeing.)

**Common mistakes (Chinese speakers):**
- Confusing 〜てから (after doing) with 〜前に (before doing) — the direction is easy to reverse under time pressure; double-check which action happens first.
- Using plain 〜て to connect actions when the "first X, then Y" sequence needs emphasis — 〜てから makes the order explicit.

**Exercises (translate into Japanese):**
1. After arriving in Japan, I will buy souvenirs.
2. After making a reservation, I will pack my luggage.

### Grammar 3 — 〜ながら (while doing, simultaneously)
**Meaning:** Describes two actions performed at the same time by the same person.
**Structure:** V(ます-stem) ＋ ながら、〜
**Examples:**
1. 音楽を聞きながら、荷物を作ります。(While listening to music, I pack my luggage.)
2. 地図を見ながら、観光地を歩きます。(While looking at the map, I walk around the tourist spot.)
3. お土産を選びながら、店員と話します。(While choosing souvenirs, I talk with the shop clerk.)

**Common mistakes (Chinese speakers):**
- ❌ 見るながら／見てながら → ⭕ 見ながら — always attach ながら to the ます-stem, not the dictionary or て-form.
- Using 〜ながら when the two actions have different subjects — ながら requires the *same* person doing both actions at once.

**Exercises (translate into Japanese):**
1. While walking around town, I take photos.
2. While traveling, I study Japanese.

## 🇹🇼 Chinese Notes

- **旅館（りょかん）**: In Taiwanese Mandarin, 旅館 often just generically means "hotel" (interchangeable with 飯店). In Japanese, 旅館 specifically refers to a traditional Japanese-style inn with tatami rooms and included meals — for a Western-style hotel, use ホテル instead.
- **案内する**: Chinese doesn't have a single common everyday word that maps onto 案内する — Chinese speakers naturally reach for 帶路 or 介紹, but 介紹 in Japanese is actually 紹介する (to introduce a *person*, Lesson 28), a different verb with a different object. Don't confuse the two.
- **言葉**: This everyday Japanese word for "word(s)/language" has no natural single-word Chinese equivalent (Chinese would say 詞語 or 話), so it can feel like an oddly literary term to a Chinese speaker at first — but in Japanese it's completely ordinary, everyday vocabulary.

### Word-by-word details

**1. 朝（あさ）— 早上**
- 例文: 朝、六時に起きます。
- 翻譯: 早上六點起床。
- Common mistake: Learners drop ご飯 and expect 朝 alone to mean "breakfast" — 朝 only means "morning"; you must say 朝ご飯 for the meal.
- Memory tip: Visualize the sun (日-like shape) rising between blades of grass — the character depicts sunrise.

**2. 昼（ひる）— 中午、白天**
- 例文: 昼はいつも忙しいです。
- 翻譯: 中午總是很忙。
- Common mistake: Same trap as 朝 — 昼 alone means "midday/daytime," not "lunch"; 昼ご飯 is the meal.
- Memory tip: You already know 昼ご飯 (Lesson 9) — 昼 is just the time-of-day part of that word.

**3. 夜（よる）— 晚上**
- 例文: 夜、テレビを見ます。
- 翻譯: 晚上看電視。
- Common mistake: Using 夜 and 夕方 interchangeably — 夜 is after dark/nighttime, while 夕方 is the earlier dusk period before dinner.
- Memory tip: 夜 visually contains 夕 (evening) inside it — night grows out of evening.

**4. 夕方（ゆうがた）— 傍晚**
- 例文: 夕方、家に帰ります。
- 翻譯: 傍晚回家。
- Common mistake: Treating 夕方 and 晩（in 晩ご飯）as identical — 夕方 is the dusk period itself, while 晩 leans toward the nighttime mealtime hours.
- Memory tip: 夕 (crescent-moon evening) + 方 (time/direction) = "the evening time."

**5. 午前（ごぜん）— 上午**
- 例文: 午前九時に会議があります。
- 翻譯: 上午九點有會議。
- Common mistake: Word order — 午前/午後 always comes *before* the clock time in Japanese (午前九時), unlike a literal English-influenced "9 午前."
- Memory tip: 午 (noon) + 前 (before) = "before noon."

**6. 午後（ごご）— 下午**
- 例文: 午後三時に病院へ行きます。
- 翻譯: 下午三點去醫院。
- Common mistake: Same ordering issue as 午前 — always 午後＋時刻, never reversed.
- Memory tip: 午 (noon) + 後 (after) = "after noon."

**7. 毎日（まいにち）— 每天**
- 例文: 毎日日本語を勉強します。
- 翻譯: 每天學習日文。
- Common mistake: Adding は after 毎日 in every sentence out of habit — 毎日 works fine as a plain adverb without は unless you're contrasting it with another time.
- Memory tip: 毎 (every) + 日 (day) = literally "every day."

**8. 毎朝（まいあさ）— 每天早上**
- 例文: 毎朝コーヒーを飲みます。
- 翻譯: 每天早上喝咖啡。
- Common mistake: Reversing the order to ❌ 朝毎 — the pattern is always 毎 + [time word], never the other way around.
- Memory tip: Same 毎 pattern as 毎日 — just swap 日 for 朝.

**9. 毎晩（まいばん）— 每天晚上**
- 例文: 毎晩十一時に寝ます。
- 翻譯: 每天晚上十一點睡覺。
- Common mistake: Using ❌ 毎夜 instead — it exists but sounds literary; 毎晩 is the natural everyday word.
- Memory tip: 晩 also appears in 晩ご飯 (dinner), a word you already know.

**10. 時々（ときどき）— 有時候**
- 例文: 時々映画を見ます。
- 翻譯: 有時候看電影。
- Common mistake: Treating 時々 and たまに as the same frequency — 時々 is more frequent than たまに; mixing them up understates or overstates how often something happens.
- Memory tip: 時 repeated ("time, time") = now and then. The 々 mark means "repeat the previous kanji."

**11. いつも（いつも）— 總是**
- 例文: いつも忙しいです。
- 翻譯: 總是很忙。
- Common mistake: Using いつも with a negative verb to mean "never" — いつも〜ない actually means "not *always*" (a partial negative), not "never" (全然〜ない).
- Memory tip: いつ (when) + も (even/also) = "whenever" = always.

**12. たまに（たまに）— 偶爾**
- 例文: たまにビールを飲みます。
- 翻譯: 偶爾喝啤酒。
- Common mistake: Confusing frequency with 時々 — たまに is rarer. Order from most to least frequent: いつも ＞ 時々 ＞ たまに.
- Memory tip: Remember it as the *lowest* frequency word of the set in this lesson.

**13. 約束（やくそく）— 約定**
- 例文: 友達と約束があります。
- 翻譯: 和朋友有約。
- Common mistake: Using 約束 for a formal appointment like a doctor's visit — that's closer to 予定 (Lesson 4) or 予約; 約束 leans toward a personal promise or arrangement between people.
- Memory tip: 約 (promise) + 束 (bundle, tie) = "tying a promise together."

**14. 遅刻する（ちこくする）— 遲到**
- 例文: 今朝、学校に遅刻しました。
- 翻譯: 今天早上上學遲到了。
- Common mistake: Using 遅い (slow/late, Lesson 17 adjective) instead — 遅い describes something being generally slow or late, while 遅刻する specifically means being late for a scheduled event.
- Memory tip: 遅 (late) + 刻 (a carved moment/time) = "missing the carved-in moment."

**15. 間に合う（まにあう）— 來得及**
- 例文: 走れば間に合います。
- 翻譯: 用跑的就來得及。
- Common mistake: Direct-translating "來得及" into ❌ 時間があります — 間に合う is the fixed idiomatic verb for "making it in time," not a literal "there is time" construction.
- Memory tip: 間 (interval) + に + 合う (fit/match) = "to fit within the time."

**16. 早く（はやく）— 早**
- 例文: 明日は早く起きます。
- 翻譯: 明天要早起。
- Common mistake: Confusing 早く (adverb, about timing) with 速い (Lesson 17, about the speed of motion) — different concepts that happen to both translate as "fast/early" in English.
- Memory tip: Same 早 kanji you already saw in 朝 (rising sun) — think "early."

**17. 遅く（おそく）— 晚**
- 例文: 昨日、遅く帰りました。
- 翻譯: 昨天很晚回家。
- Common mistake: Using 遅く where 遅刻して (being late for something specific) is meant — 遅く just means "late" generally in time, e.g. late at night.
- Memory tip: Memorize 早く and 遅く together as an opposite pair.

**18. もうすぐ（もうすぐ）— 快要**
- 例文: もうすぐ夏休みです。
- 翻譯: 快要放暑假了。
- Common mistake: Confusing もうすぐ (imminent, very soon) with これから (from now on, no specific timing implied) — they answer different questions.
- Memory tip: もう (already) + すぐ (immediately) = "it's already almost here."

**19. さっき（さっき）— 剛才**
- 例文: さっき駅で友達に会いました。
- 翻譯: 剛才在車站遇到朋友。
- Common mistake: Using さっき with future-tense verbs — さっき always points to the recent past, never to something upcoming.
- Memory tip: The short, sharp sound mimics a quick glance backward — "just now."

**20. これから（これから）— 從現在起**
- 例文: これから勉強します。
- 翻譯: 從現在開始要唸書。
- Common mistake: Confusing これから (an action about to begin and continue) with 今 (just the current moment) — これから is forward-looking.
- Memory tip: これ (this) + から (from) = "from this point on."

**21. ずっと（ずっと）— 一直**
- 例文: 今日はずっと家にいました。
- 翻譯: 今天一直待在家。
- Common mistake: Confusing ずっと (one continuous stretch of time) with いつも (a repeated habit) — ずっと describes an unbroken period, not a recurring pattern.
- Memory tip: The elongated "zu-tto" sound mimics something stretched out over time.

**22. しばらく（しばらく）— 暫時**
- 例文: しばらく待ってください。
- 翻譯: 請稍等一下。
- Common mistake: Treating しばらく (a longer, vaguer stretch of time) the same as ちょっと (a short bit) — しばらく generally implies more time has passed or will pass.
- Memory tip: Recall the set phrase 久しぶり(お久しぶりです) — しばらく is used the same greeting-like way: しばらくですね.

**23. 久しぶり（ひさしぶり）— 好久不見**
- 例文: 久しぶりですね。元気ですか。
- 翻譯: 好久不見，你好嗎？
- Common mistake: Trying to conjugate 久しぶり like a verb — it's a fixed exclamatory phrase, always used as 久しぶり／久しぶりです／久しぶりですね, never with verb endings.
- Memory tip: 久しい (long-lasting) + ぶり (span/interval) = "after a long interval."

**24. 最近（さいきん）— 最近**
- 例文: 最近、忙しいです。
- 翻譯: 最近很忙。
- Common mistake: Using 最近 with clearly future events — 最近 anchors to the recent past/present, unlike some looser Chinese usages that can lean forward in time.
- Memory tip: 最 (most) + 近 (near/close) = "closest [in time]" = recently.

**25. そろそろ（そろそろ）— 差不多該**
- 例文: そろそろ寝ましょう。
- 翻譯: 差不多該睡了。
- Common mistake: Confusing そろそろ (signals the speaker's own intention that it's about time to act) with もうすぐ (an event simply approaching, no personal intention implied).
- Memory tip: The repeated そろ sound suggests easing gradually into motion — "it's about that time."

## ② Kanji (6): 朝, 夜, 午, 後, 分, 毎

**朝 — Meaning: 朝 (morning)**
- On-reading: チョウ
- Kun-reading: あさ
- Words: 朝, 毎朝, 朝ご飯 (L9)
- Example sentence: 毎朝六時に起きます。
- Stroke order: 12 strokes — write the left component (十＋日, resembling grass with the sun rising through it) first, then the right-hand 月-like component, top to bottom.
- Chinese difference: Same meaning as Chinese 朝 ("morning"), but Chinese 朝 also commonly means "dynasty/court" (朝代, 朝廷). Japanese 朝 is used only for "morning" — the "dynasty" meaning survives only in compounds like 王朝, not as a general sense of the character.

**夜 — Meaning: 夜 (night)**
- On-reading: ヤ
- Kun-reading: よる
- Words: 夜, 今夜 (preview), 夜中 (よなか, preview)
- Example sentence: 夜、勉強します。
- Stroke order: 8 strokes — top-left roof-like shape (一＋ノ) first, then 夕 tucked underneath to the right; finish top to bottom, left to right.
- Chinese difference: Same character and meaning as Chinese 夜, quite consistent. The adjustment for Chinese speakers is the reading split: native よる in everyday words like 夜 alone, versus the on-reading ヤ locked inside compounds like 夜間 — Mandarin uses one reading "yè" everywhere.

**午 — Meaning: 午 (noon)**
- On-reading: ゴ
- Kun-reading: — (no common kun-reading)
- Words: 午前, 午後, 正午 (preview しょうご)
- Example sentence: 午後、買い物に行きます。
- Stroke order: 4 strokes — resembles 牛 (cow) missing one stroke; top horizontal, a vertical hook, then the crossing strokes. Be careful not to confuse it with 牛.
- Chinese difference: In Chinese, 午 also appears almost exclusively inside 上午/中午/下午 rather than standing alone — this usage pattern transfers cleanly to Japanese 午前/午後, making 午 one of the easier kanji in this lesson.

**後 — Meaning: 後 (after, behind)**
- On-reading: ゴ、コウ
- Kun-reading: あと、うし.ろ
- Words: 午後, 後ろ (L7), 最後 (preview さいご)
- Example sentence: 授業の後、友達に会います。
- Stroke order: 9 strokes — left radical 彳 (three strokes) first, then the right side (幺 over 夂), top to bottom.
- Chinese difference: Identical to Traditional Chinese 後 (the form used in Taiwan, not simplified 后), so the shape is instantly recognizable. The difference is that Japanese switches readings by context — あと for general "afterward," うしろ for spatial "behind," ゴ/コウ inside compounds — while Chinese uses one reading "hòu" regardless of meaning.

**分 — Meaning: 分 (minute, part, divide)**
- On-reading: ブン、フン、ブ
- Kun-reading: わ.ける
- Words: 半分 (L2), 何分 (preview なんぷん), 自分 (preview じぶん)
- Example sentence: 十分待ってください。
- Stroke order: 4 strokes — 八 (a splitting shape) on top, 刀 (knife) below; the knife visually "divides" things, matching the meaning.
- Chinese difference: Same character and overlapping meaning as Chinese 分 ("divide/minute/part"). The key adjustment: Japanese changes the reading by context (ふん when counting minutes, ぶん for "a portion," ぷん after certain sounds for euphony) — Mandarin keeps a single reading "fēn" throughout.

**毎 — Meaning: 每 (every)**
- On-reading: マイ
- Kun-reading: — (no common kun-reading)
- Words: 毎日, 毎朝, 毎晩
- Example sentence: 毎日日本語を勉強します。
- Stroke order: 6 strokes — the short slanted stroke and horizontal line on top first, then the wider 母-like base underneath.
- Chinese difference: Same meaning and similar shape to Traditional Chinese 每, used identically in compounds like 每日/每天. Unlike many other kanji, 毎 has no common native kun-reading — Chinese speakers should just memorize マイ rather than expect a separate everyday reading.

## ③ Grammar (3 points)

### Grammar 1 — 〜ましょうか (Shall I...?)
**Meaning:** Offers to do something for the listener, or proposes a joint action, more polite/tentative than plain ましょう.
**Structure:** Verb ます-stem + ましょうか
**Examples:**
1. 掃除しましょうか。(Shall I clean?)
2. 写真を撮りましょうか。(Shall I take a photo?)
3. コーヒーを買いましょうか。(Shall I buy some coffee?)

**Common mistakes (Chinese speakers):**
- ❌ Using ましょうか to state plain future intention ("I will...") — it specifically offers or proposes an action, usually for the listener's benefit, not a neutral statement about the future.
- ❌ Confusing ましょうか (an offer: "shall I do this for you?") with ませんか (an invitation: "won't you do this with me?" — see Grammar 2). They invite different responses.

**Exercises (translate into Japanese):**
1. Shall I read it?
2. Shall I call (make a phone call)?
3. Shall I clean the room?

### Grammar 2 — 〜ませんか (Won't you...?)
**Meaning:** A polite, indirect way to invite the listener to do something together.
**Structure:** Verb ます-stem + ませんか
**Examples:**
1. 明日、公園で遊びませんか。(Won't you play at the park with me tomorrow?)
2. 週末、映画を見ませんか。(Won't you watch a movie with me this weekend?)
3. お茶を飲みませんか。(Won't you have some tea?)

**Common mistakes (Chinese speakers):**
- ❌ Answering with はい to mean "no, I won't" by literal "不要嗎？" logic — because the verb is negative in form, はい actually agrees with the *positive* action: はい、行きます means "yes, I'll go," not "yes, I won't."
- ❌ Reading ませんか only as a plain negative question ("Don't you go to school?") — tone and context distinguish the inviting sense from a genuine yes/no question about a negative fact.

**Exercises (translate into Japanese):**
1. Won't you eat dinner with me?
2. Won't you go together?
3. Won't you drink coffee?

### Grammar 3 — 動詞のて形 (te-form of verbs)
**Meaning:** A connecting form used to link sequential or simultaneous actions ("do X, then/while Y"), and the base for later patterns like 〜てください and 〜ています.
**Structure:**
- る-verbs (ichidan): drop る, add て → 食べる → 食べて
- う-verbs (godan), by final syllable:
  - う／つ／る → って（買う→買って、待つ→待って、帰る→帰って）
  - む／ぶ／ぬ → んで（飲む→飲んで、遊ぶ→遊んで）
  - く → いて（書く→書いて）— exception: 行く→行って
  - ぐ → いで（泳ぐ→泳いで）
  - す → して（話す→話して）
- irregular: する→して、来る→きて

**Examples:**
1. 朝起きて、シャワーを浴びます。(I wake up in the morning, and take a shower.)
2. 朝ご飯を食べて、学校へ行きます。(I eat breakfast, and go to school.)
3. 家に帰って、宿題をします。(I go home, and do my homework.)

**Common mistakes (Chinese speakers):**
- ❌ Stringing two ます-form verbs together the way Chinese lists actions in sequence (先吃飯，然後去學校 → ❌ 食べます、学校へ行きます) — the natural connector for sequential daily actions is the て-form, not two separate ます-sentences.
- ❌ Misapplying the く→いて rule to 行く, which is the well-known irregular exception 行って (not ❌行いて).
- ❌ Treating 帰る like an ichidan verb because it ends in る, producing ❌ 帰て instead of the correct godan form ⭕ 帰って.

**Exercises (translate into Japanese, using the te-form):**
1. I wake up, and then eat breakfast.
2. I go to the station, and then take the train.
3. I finish work, and then go home.

## ④ Reading

私は毎朝六時に起きます。朝ご飯を食べて、会社へ行きます。会社は九時に始まります。午前は忙しいです。午後はあまり忙しくないです。会社で働いて、夕方、友達に会います。時々、友達と晩ご飯を食べます。夜、本を読みます。テレビも見ます。いつも十一時に寝ます。

**Comprehension questions:**

2. 午後は忙しいですか。

**Answer key:**
1. 六時に起きます。
2. いいえ、あまり忙しくないです。
3. 本を読みます。テレビも見ます。

## 🇹🇼 Chinese Notes

- **々 (repetition mark, as in 時々)**: This small mark tells the reader to repeat the previous kanji (時々 = 時＋時). It is unique to Japanese orthography — modern Chinese writing does not use this mark, so don't mistake it for a punctuation error or expect it to appear in Chinese text.
- **後（あと／うしろ／ゴ・コウ）**: Traditional Chinese 後 (used in Taiwan) is visually identical and covers both "behind" (space) and "after" (time) with one reading, "hòu." Japanese uses the *same character* but switches the reading depending on meaning: うしろ for spatial "behind" (後ろ), あと for temporal "afterward" (後で), and ゴ／コウ inside fixed compounds (午後, 最後). Don't assume one reading works everywhere just because the character is unchanged.
- **朝**: In Chinese, 朝 can mean "morning" but also commonly means "dynasty/court" (朝代, 朝廷, e.g. 唐朝). In Japanese, 朝 by itself only ever means "morning" — the dynastic sense survives only inside specific compounds like 王朝, so don't expect 朝 alone to carry that meaning in everyday Japanese.

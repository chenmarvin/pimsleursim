### Word-by-word details

**1. 曲がる（まがる）— 轉彎**
- 例文: 次の角で右に曲がってください。
- 翻譯: 請在下個轉角右轉。
- Common mistake: This looks like a ru-verb because it ends in る, but it's actually a u-verb (五段動詞) — the nai-form is 曲がらない, not ❌曲がない.
- Memory tip: 曲 already means "bend/curve" (as in 曲がった道, a curved road) — 曲がる is literally "to bend your path."

**2. まっすぐ — 直直地**
- 例文: この道をまっすぐ行ってください。
- 翻譯: 請沿著這條路直走。
- Common mistake: Learners add を after まっすぐ (❌まっすぐを行きます) — it's an adverb and attaches directly before the verb, no particle needed.
- Memory tip: The doubled っす sound feels like a straight line shooting forward — "massugu."

**3. 信号（しんごう）— 紅綠燈**
- 例文: 二つ目の信号を左に曲がります。
- 翻譯: 在第二個紅綠燈左轉。
- Common mistake: Chinese 信號 broadly covers any "signal" (phone signal, Morse code); in everyday Japanese 信号 almost always means specifically a traffic light.
- Memory tip: 信（trust）+ 号（mark）= a trusted marker that tells drivers when to go.

**4. 交差点（こうさてん）— 十字路口**
- 例文: 交差点を渡ってください。
- 翻譯: 請過十字路口。
- Common mistake: Learners sometimes invent a literal calque like 十字路 instead of the natural word 交差点.
- Memory tip: 交差（cross/intersect）+ 点（point）= "the point where roads cross."

**5. 近く（ちかく）— 附近**
- 例文: 駅の近くにコンビニがあります。
- 翻譯: 車站附近有便利商店。
- Common mistake: Confusing the noun 近く ("the nearby area") with the i-adjective 近い ("near") — 近くに vs 近いです behave differently in a sentence.
- Memory tip: 近い + く turns "near" into a noun meaning "the nearby area."

**6. 隣（となり）— 隔壁、旁邊**
- 例文: 銀行は郵便局の隣です。
- 翻譯: 銀行在郵局隔壁。
- Common mistake: Using 隣 and 横 interchangeably — 隣 implies "directly adjacent, same kind of thing" (house next to house), while 横 is a looser "beside."
- Memory tip: Picture houses lined up "next door" to each other, like immediate neighbors.

**7. 間（あいだ）— 之間**
- 例文: 銀行と郵便局の間に公園があります。
- 翻譯: 銀行和郵局之間有公園。
- Common mistake: This is the same kanji seen in Lesson 4 (時間, じかん) — but here in 〜の間 it reads あいだ, a completely different reading for the same character.
- Memory tip: 間 shows a gate 門 with the sun 日 peeking through the middle — the space "between" two doors.

**8. 横（よこ）— 旁邊、側面**
- 例文: 駅の横にタクシー乗り場があります。
- 翻譯: 車站旁邊有計程車招呼站。
- Common mistake: See #6 — 横 is more general than 隣, and can also mean "sideways/horizontal" (横になる, to lie down), a meaning unrelated to location.
- Memory tip: 横 contains 木 (tree) — imagine trees planted "off to the side" along a road.

**9. そば — 旁邊、附近**
- 例文: 駅のそばに交番があります。
- 翻譯: 車站附近有派出所。
- Common mistake: そば is also the name of a buckwheat noodle dish (蕎麦) — same sound, unrelated meaning; context tells them apart.
- Memory tip: そば sounds soft and close, like something "right there beside you."

**10. 向かい（むかい）— 對面**
- 例文: 郵便局は駅の向かいです。
- 翻譯: 郵局在車站對面。
- Common mistake: Confusing the location noun 向かい ("across from") with the verb 向かう ("to head toward") — related but used very differently.
- Memory tip: 向 (face toward) — picture two buildings "facing" each other across the street.

**11. 遠く（とおく）— 遠處**
- 例文: 学校はここから遠くないです。
- 翻譯: 學校離這裡不遠。
- Common mistake: Same trap as 近く — mixing up the noun 遠く ("a distant place") with the i-adjective 遠い ("far").
- Memory tip: 遠 already looks long and stretched out, like the distance itself.

**12. 分かりやすい（わかりやすい）— 容易懂**
- 例文: この地図はとても分かりやすいです。
- 翻譯: 這張地圖非常容易懂。
- Common mistake: Learners translate "easy to understand" word-for-word with 簡単 instead of using the productive verb-stem + やすい pattern (分かり＋やすい).
- Memory tip: 分かる's stem 分かり + やすい (easy to...) = "easy to understand."

**13. 分かりにくい（わかりにくい）— 難懂**
- 例文: この説明は分かりにくいです。
- 翻譯: 這個說明很難懂。
- Common mistake: Saying 分かりません難しい as a clumsy calque instead of the single adjective 分かりにくい.
- Memory tip: にくい (hard to...) is the mirror opposite of やすい — learn them as a pair.

**14. 渡る（わたる）— 過（馬路）**
- 例文: 信号が青の時に渡ってください。
- 翻譯: 請在紅綠燈是綠燈的時候過馬路。
- Common mistake: Confusing 渡る with 渡す（わたす, "to hand over," Lesson 22) — same root kanji, very different verbs.
- Memory tip: 渡る takes を for the space crossed (道を渡る), just like 歩く (道を歩く) — motion-through verbs share this pattern.

**15. 通る（とおる）— 經過、通過**
- 例文: この道を通ると、駅に着きます。
- 翻譯: 通過這條路的話，就會到車站。
- Common mistake: Confusing 通る (intransitive "to pass through") with the unrelated-looking but similarly pronounced 通う (かよう, "to commute regularly").
- Memory tip: 通 contains 辶 (movement) — picture a road going straight through a tunnel.

**16. 突き当たり（つきあたり）— 盡頭**
- 例文: 突き当たりを右に曲がってください。
- 翻譯: 請在路的盡頭右轉。
- Common mistake: Learners reach for 終わり ("the end," used for finishing an event) instead of this specific spatial term for where a road physically stops.
- Memory tip: 突き (thrust/poke) + 当たり (hit) = "where you'd bump into a wall if you kept going straight."

**17. 角（かど）— 角落、轉角**
- 例文: あの角にコンビニがあります。
- 翻譯: 那個轉角有便利商店。
- Common mistake: The same kanji 角 is also read かく in other compounds (三角形, さんかくけい, triangle) — beginners default to one reading everywhere.
- Memory tip: Picture the "horn" shape suggested by 角 sticking out — a sharp corner.

**18. 東（ひがし）— 東**
- 例文: 東の空が明るくなりました。
- 翻譯: 東邊的天空變亮了。
- Common mistake: Mixing up compass compound word order — Japanese says 北東 (north-east), and note that 東北（とうほく）specifically names the Tōhoku region of Japan, not just a generic "northeast."
- Memory tip: The sun 日 rises inside this character's shape — east is where the sun comes up.

**19. 西（にし）— 西**
- 例文: 太陽は西に沈みます。
- 翻譯: 太陽在西邊落下。
- Common mistake: Learners use the on-reading せい where the kun-reading にし is expected in plain direction words (西 alone is always にし, not せい).
- Memory tip: Picture a bird settling into a nest-like shape at sunset — 西 resembles a nest.

**20. 南（みなみ）— 南**
- 例文: 南口を出ると、バス停があります。
- 翻譯: 出南口就有公車站。
- Common mistake: Compass order again — Japanese says 南西 (southwest), N/S word always first, unlike casual word-order habits some learners carry over.
- Memory tip: The shape resembles a tent or shelter facing the warm southern sun.

**21. 北（きた）— 北**
- 例文: この道をまっすぐ北に行きます。
- 翻譯: 沿著這條路直直往北走。
- Common mistake: Forgetting the consistent rule: Japanese compass compounds always put 北/南 first — 北東、北西、南東、南西.
- Memory tip: 北 looks like two people standing back-to-back, facing away — the cold, "away" direction.

**22. 方向（ほうこう）— 方向**
- 例文: この方向で合っていますか。
- 翻譯: 這個方向對嗎？
- Common mistake: Confusing the general word 方向 ("which way") with the more technical 方角（ほうがく, compass bearing) — for N5 purposes 方向 is the safe, everyday word.
- Memory tip: 方 (direction/side) + 向 (facing) = "the way you are facing."

**23. 迷う（まよう）— 迷路**
- 例文: 道に迷ってしまいました。
- 翻譯: 我迷路了。
- Common mistake: In Chinese, 迷 alone often means "to be a fan of / obsessed with" (影迷, 球迷); in Japanese 迷う only ever means "to lose one's way" or "to hesitate," never "to be a fan of something."
- Memory tip: 迷 shows 辶 (walking) + 米 (scattered grains of rice) — like footprints scattered every which way, going nowhere in particular.

**24. 標識（ひょうしき）— 標誌、路標**
- 例文: あの標識を見てください。
- 翻譯: 請看那個路標。
- Common mistake: Reaching for the loanword サイン (which in Japanese usually means "signature/autograph") when a physical road sign is meant — 標識 is the correct word.
- Memory tip: 標 (mark/label) + 識 (recognize) = "a mark that helps you recognize [where you are]."

**25. 目印（めじるし）— 標記、地標**
- 例文: あの高い建物が目印です。
- 翻譯: 那棟高的建築物是地標。
- Common mistake: Defaulting to the loanword ランドマーク; 目印 is the more natural, commonly used N5-level word for a landmark you navigate by.
- Memory tip: 目 (eye) + 印 (mark/seal) = "a mark for the eye to find your way by."

## ② Kanji (5): 前, 北, 西, 東, 南

**前 — Meaning: 前 (front, before)**
- On-reading: ゼン
- Kun-reading: まえ
- Words: 前（まえ, front/before）, 名前（なまえ, name — Lesson 1）, 午前（ごぜん, a.m. — Lesson 21）
- Example sentence: 銀行の前に公園があります。
- Stroke order: 9 strokes — the two short top-left strokes first, then the horizontal stroke, then the 月-shaped body on the right, finishing with the two short strokes (刂) on the far right.
- Chinese difference: Shape and core meaning match Traditional Chinese 前 exactly, and both languages use it productively as a time suffix (三年前, "three years ago"). The main trap for Taiwanese learners is simply remembering when to use the kun-reading まえ versus the on-reading ゼン across different compounds.

**北 — Meaning: 北 (north)**
- On-reading: ホク
- Kun-reading: きた
- Words: 北（きた, north）, 北東（ほくとう, northeast）, 東北（とうほく, the Tōhoku region of Japan）
- Example sentence: 北海道は日本の北にあります。
- Stroke order: 5 strokes — left half first (two strokes, like a standing figure), then the mirrored right half (three strokes).
- Chinese difference: Same shape and meaning as Chinese 北, but the compound order is systematically reversed: Japanese always puts 北/南 first (北東, 北西, 南東, 南西), while Chinese always puts 東/西 first (東北, 西北, 東南, 西南). Memorize this as a fixed rule, not word-by-word.

**西 — Meaning: 西 (west)**
- On-reading: セイ、サイ
- Kun-reading: にし
- Words: 西（にし, west）, 北西（ほくせい, northwest）, 関西（かんさい, the Kansai region of Japan）
- Example sentence: 大阪は関西にあります。
- Stroke order: 6 strokes — top horizontal stroke first, then the outer frame (left side, then top-right and right side), finishing with the two internal vertical strokes and the bottom horizontal stroke.
- Chinese difference: Identical shape and meaning to Chinese 西. Note the two on-readings, セイ (西洋, せいよう, "the West/Occident") and サイ (関西) — Chinese uses a single reading "xī" for this character in every context.

**東 — Meaning: 東 (east)**
- On-reading: トウ
- Kun-reading: ひがし
- Words: 東（ひがし, east）, 北東（ほくとう, northeast）, 東京（とうきょう, Tokyo）
- Example sentence: 私は東京に行きたいです。
- Stroke order: 8 strokes — write the enclosing frame first (top, left side, right side), then the two crossing strokes through the middle, finishing with the bottom-left and bottom-right diagonal strokes.
- Chinese difference: Same shape/meaning as Chinese 東 (or Simplified 东). As a place name, 東京 (Tokyo, "Eastern Capital") uses characters Taiwanese learners already recognize instantly — one of the easiest proper nouns in the whole language.

**南 — Meaning: 南 (south)**
- On-reading: ナン、ナ
- Kun-reading: みなみ
- Words: 南（みなみ, south）, 南口（みなみぐち, south exit）, 南西（なんせい, southwest）
- Example sentence: 駅の南口で待っています。
- Stroke order: 9 strokes — outer frame first (top, left, right, bottom), then the internal horizontal and vertical strokes filling the middle.
- Chinese difference: Same shape and meaning as Chinese 南. The on-reading ナン (南西, southwest) is far more common than ナ, which appears only in a couple of set words — Chinese uses one reading "nán" throughout.

## ③ Grammar (3 points)

### Grammar 1 — 動詞のない形 (nai-form of verbs)
**Meaning:** The plain negative form of a verb, used in casual negative sentences and as the building block for many other patterns (must do, don't have to, etc.).
**Structure:**
- U-verbs (五段動詞): change the final -u sound to -a, then add ない (曲がる→曲がらない, 渡る→渡らない, 通る→通らない). Verbs ending in う take わない (迷う→迷わない).
- Ru-verbs (一段動詞): drop る, add ない (見る→見ない, 教える→教えない).
- Irregular: する→しない, 来る→こない.

**Examples:**
1. 私はお酒を飲まない。(I don't drink alcohol.)
2. 明日は学校へ行かない。(I'm not going to school tomorrow.)
3. この道はあまり通らない。(I don't often pass through this road.)

**Common mistakes (Chinese speakers):**
- ❌ 曲がない → ⭕ 曲がらない — 曲がる looks like a ru-verb because it ends in る, but it's a u-verb; misclassifying verb groups is the single most common nai-form error.
- ❌ Mixing plain ない with polite ですます in the same clause without a reason (この道は通らないです is acceptable casually, but be consistent within formal writing).

**Exercises (translate into Japanese):**
1. I don't turn right.
2. I don't eat meat.
3. He doesn't come today.

### Grammar 2 — 〜なければなりません (must do)
**Meaning:** Expresses obligation — "must do X," "have to do X."
**Structure:** Verb nai-form, drop い, add ければなりません (曲がる→曲がらない→曲がらなければなりません).
**Examples:**
1. ここで右に曲がらなければなりません。(You must turn right here.)
2. 明日は早く起きなければなりません。(I must wake up early tomorrow.)
3. 毎日日本語を勉強しなければなりません。(I must study Japanese every day.)

**Common mistakes (Chinese speakers):**
- ❌ Dropping ければ entirely and just saying ❌曲がらなりません — the full ければ chunk is required, even though it visually resembles the ば-conditional taught later.
- ❌ Mixing the casual contraction なきゃ with formal ですます elsewhere in the same sentence.

**Exercises (translate into Japanese):**
1. I must go straight.
2. You must turn left at the corner.
3. We must arrive on time. (使用 間に合う)

### Grammar 3 — 〜なくてもいいです (don't have to do)
**Meaning:** States that an action is not required — "it's fine even if you don't do X."
**Structure:** Verb nai-form, drop い, add くてもいいです (曲がる→曲がらない→曲がらなくてもいいです).
**Examples:**
1. 今日は行かなくてもいいです。(You don't have to go today.)
2. 道を渡らなくてもいいです。(You don't have to cross the road.)
3. 心配しなくてもいいです。(You don't have to worry.)

**Common mistakes (Chinese speakers):**
- ❌ Confusing 〜なくてもいいです (permission not to do something) with 〜てはいけません (Lesson 23, prohibition) — these are near-opposite meanings that learners frequently swap.
- ❌ Dropping the も (❌なくていいです is actually also acceptable casually, but beginners should first master the full なくてもいいです form before shortening it).

**Exercises (translate into Japanese):**
1. You don't have to turn right.
2. You don't have to worry.
3. You don't have to go straight.

## ④ Reading

Read the passage aloud:

すみません、駅までどう行きますか。
はい。この道をまっすぐ行ってください。二つ目の信号を右に曲がります。交差点の近くに銀行があります。銀行の隣に郵便局があります。駅は郵便局の向かいです。分かりやすいですよ。道を渡らなくてもいいです。

**Comprehension questions:**
1. 駅はどこにありますか。(Where is the station?)
2. 銀行の隣に何がありますか。(What is next to the bank?)
3. 道を渡らなければなりませんか。(Do you have to cross the road?)

**Answer key:**
1. 郵便局の向かいにあります。(It's across from the post office.)
2. 郵便局があります。(The post office is next to it.)
3. いいえ、渡らなくてもいいです。(No, you don't have to cross it.)

## ⑤ Listening

*Read once slowly, then once at natural speed.*

A: すみません、この近くに病院がありますか。
B: はい、あります。あの交差点を左に曲がってください。まっすぐ行くと、右側に病院があります。
A: 遠いですか。
B: いいえ、遠くないです。歩いて五分ぐらいです。
A: 分かりました。ありがとうございます。

**Comprehension questions:**
1. 病院はどの方向にありますか。(In which direction is the hospital?)
2. 病院まで歩いてどのくらいかかりますか。(How long does it take to walk to the hospital?)

**Answer key:**
1. 交差点を左に曲がって、まっすぐ行った右側です。(Turn left at the intersection, go straight, and it's on the right.)
2. 五分ぐらいです。(About five minutes.)

## 🇹🇼 Chinese Notes

- **信号（しんごう）**: Chinese 信號 is a broad word for any kind of signal (phone signal, Morse code, hand signal). In everyday Japanese, 信号 almost always means specifically a traffic light — don't assume it covers every sense of "signal."
- **迷う（まよう）**: In Chinese, 迷 by itself often means "to be a fan of / obsessed with" something (影迷, movie fan; 球迷, sports fan). In Japanese, 迷う only ever means "to lose one's way" or "to hesitate" — it never carries the "fan/enthusiast" meaning.
- **Compass compound order**: Japanese always states north/south before east/west — 北東 (northeast), 北西 (northwest), 南東 (southeast), 南西 (southwest). Chinese does the reverse, stating east/west first — 東北, 西北, 東南, 西南. This is a consistent, mechanical difference worth memorizing as a rule rather than case by case.

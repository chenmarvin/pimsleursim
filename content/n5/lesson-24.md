### Word-by-word details

**1. 山（やま）— 山**
- 例文: あの山は高いです。
- 翻譯: 那座山很高。
- Common mistake: Confusing the standalone kun-reading やま with the on-reading サン used in named mountains like 富士山（ふじさん） — beginners sometimes guess ふじやま, which is an old alternate reading, not the standard modern one.
- Memory tip: The kanji visually resembles three mountain peaks.

**2. 川（かわ）— 河**
- 例文: この川はきれいです。
- 翻譯: 這條河很乾淨。
- Common mistake: Confusing 川（かわ, river）with 皮／革（かわ, "skin/leather," unrelated kanji, identical reading) — a homophone trap distinguished only by kanji and context.
- Memory tip: The kanji literally depicts three flowing lines of water.

**3. 海（うみ）— 海**
- 例文: 夏に海へ行きます。
- 翻譯: 夏天去海邊。
- Common mistake: Assuming 海 only means the grand "ocean" — in everyday Japanese it casually covers "the beach/seaside" too (海に行く = go to the beach).
- Memory tip: Contains 氵(water radical) + 毎 (every, L21) — "water everywhere."

**4. 空（そら）— 天空**
- 例文: 空が青いです。
- 翻譯: 天空是藍色的。
- Common mistake: Confusing 空（そら, "sky," a noun) with 空く（あく／すく, "to become empty," a verb, seen in お腹が空く, L9) — same kanji, totally different readings and grammatical roles.
- Memory tip: The same 空 kanji you know from お腹が空く broadly means "empty/open space," which extends naturally to the open sky.

**5. 星（ほし）— 星星**
- 例文: 星がきれいです。
- 翻譯: 星星很美。
- Common mistake: Guessing the on-reading せい (used in compounds like 火星, "Mars") when the word stands alone — like many kanji, the standalone reading differs from the compound reading.
- Memory tip: 日 (light) + 生 (born, L1) = "light that is born" in the night sky = star.

**6. 月（つき）— 月亮**
- 例文: 今夜、月がきれいです。
- 翻譯: 今晚月亮很美。
- Common mistake: Confusing 月（つき, "moon," here) with 月（がつ／げつ, "month," L4) — same character, two unrelated everyday meanings depending on context.
- Memory tip: You already know this kanji from 月曜日 and 今月 (L4) — here it returns to its original picture-meaning, a crescent moon.

**7. 太陽（たいよう）— 太陽**
- 例文: 空に太陽があります。
- 翻譯: 天空中有太陽。
- Common mistake: Confusing 太陽（たいよう, "the sun," an astronomy noun) with the unrelated higher-level word 陽気な ("cheerful") that shares the 陽 kanji — no connection at the N5 level.
- Memory tip: 太 (great/big) + 陽 (sunlight) = "the great sun."

**8. 花（はな）— 花**
- 例文: 花がきれいに咲いています。
- 翻譯: 花開得很漂亮。
- Common mistake: Confusing 花（はな, flower) with 鼻（はな, "nose," L12) — identical pronunciation, completely unrelated meaning, a classic homophone pair.
- Memory tip: Contains 艹 (grass/plant radical) on top, visually marking it as plant-related, unlike 鼻.

**9. 木（き）— 樹**
- 例文: 大きい木があります。
- 翻譯: 有一棵大樹。
- Common mistake: Confusing standalone 木（き, tree) with the day-name reading もく in 木曜日 (L4) — same kanji, different reading depending on whether it stands alone or appears in the day-name compound.
- Memory tip: The kanji is a simple pictograph of a tree with branches and roots — one of the most recognizable kanji for beginners.

**10. 犬（いぬ）— 狗**
- 例文: 犬が好きです。
- 翻譯: 喜歡狗。
- Common mistake: Visually confusing 犬 with 大（おお.きい, big, L8) since they differ by only one small dot — a very common mix-up when reading quickly.
- Memory tip: The extra dot on 犬 (compared to 大) represents the dog's tail or a small companion by your side.

**11. 猫（ねこ）— 貓**
- 例文: 猫と遊びます。
- 翻譯: 和貓玩。
- Common mistake: Trying to use the rare on-reading ビョウ in daily conversation — 猫 is almost always read with its kun-reading ねこ in everyday speech.
- Memory tip: Contains 犭(animal radical, related to 犬) + 苗 (seedling, phonetic hint) — a small animal, like a cat.

**12. 鳥（とり）— 鳥**
- 例文: 鳥が飛んでいます。
- 翻譯: 鳥在飛。
- Common mistake: Confusing 鳥（とり, bird) with 取る（とる, "to take," different kanji/meaning) — close but distinct pronunciation (とり vs とる).
- Memory tip: The kanji is a simplified pictograph of a bird with wings and a tail.

**13. 魚（さかな）— 魚（動物）**
- 例文: 川に魚がいます。
- 翻譯: 河裡有魚。
- Common mistake: This is the exact same word taught back in Lesson 9 (as food) — no new reading here, but note the existence-verb changes with context: いる for a living fish swimming, versus food-related verbs like 食べる when it's on your plate.
- Memory tip: Same word, same kanji — just a live animal this time instead of a meal.

**14. 動物（どうぶつ）— 動物**
- 例文: 動物が好きです。
- 翻譯: 喜歡動物。
- Common mistake: None major in meaning — it matches Chinese 動物 closely — but the pronunciation どうぶつ doesn't resemble Mandarin "dòngwù" closely enough to guess by ear, so it still needs deliberate memorizing.
- Memory tip: 動 (move) + 物 (thing) = "a moving thing" = animal, exactly matching the logic of Chinese 動物.

**15. 自然（しぜん）— 自然**
- 例文: 自然が好きです。
- 翻譯: 喜歡大自然。
- Common mistake: None major in meaning, but learners often mispronounce it with a Mandarin-influenced "zì-rán" rhythm instead of the correct four-mora Japanese し・ぜ・ん reading.
- Memory tip: 自 (self) + 然 (as it is) = "as it is by itself" = nature — the same logic as Chinese 自然.

**16. 森（もり）— 森林**
- 例文: 森を散歩します。
- 翻譯: 在森林散步。
- Common mistake: Confusing 森（もり, "forest," three 木 stacked) with 林（はやし, "woods/grove," two 木 stacked, a related but smaller-scale word not covered in N5) — visually similar, related but distinct scope.
- Memory tip: Three 木 (tree) stacked together = a forest.

**17. 散歩する（さんぽする）— 散步**
- 例文: 毎朝、公園を散歩します。
- 翻譯: 每天早上在公園散步。
- Common mistake: Assuming only one particle works with 散歩する — it commonly takes を for the area walked through (公園を散歩する) which can surprise learners expecting に everywhere.
- Memory tip: 散 (scatter) + 歩 (step/walk) = "scattered walking" = strolling aimlessly, i.e. a walk.

**18. 咲く（さく）— 開花**
- 例文: 春に花が咲きます。
- 翻譯: 春天花會開。
- Common mistake: Confusing 咲く (specifically for flowers/blossoms blooming) with the more general "to open" (a different, not-yet-taught verb) — 咲く is reserved for flowers.
- Memory tip: Contains 口 (mouth) — imagine a flower's petals opening like a mouth.

**19. 飛ぶ（とぶ）— 飛**
- 例文: 鳥が空を飛びます。
- 翻譯: 鳥在天空飛。
- Common mistake: Using plain 飛ぶ for an airplane's scheduled departure the same way — while 飛行機が飛びます is technically fine, the common set verb for a flight's schedule is 出発する (L17); plain 飛ぶ leans toward the literal act of flying through the air.
- Memory tip: The kanji visually resembles wings spread out in flight.

**20. 鳴く（なく）— 叫（動物）**
- 例文: 鳥が鳴いています。
- 翻譯: 鳥在叫。
- Common mistake: Confusing 鳴く（なく, animal sounds — chirping, barking, meowing) with 泣く（なく, L23, humans crying) — identical pronunciation, different kanji and entirely different subject.
- Memory tip: Contains 口 (mouth) + 鳥 (bird) — literally "a bird's mouth [making sound]," extended to all animal sounds.

**21. 虫（むし）— 蟲**
- 例文: 虫が嫌いです。
- 翻譯: 討厭蟲子。
- Common mistake: None phonetically, but note this is the Japanese shinjitai simplified form of Traditional Chinese 蟲 (three 虫 stacked, used in Taiwan) — recognize 虫 as its simplified single-component equivalent.
- Memory tip: The kanji is a simple pictograph resembling a small crawling creature.

**22. 石（いし）— 石頭**
- 例文: 大きい石があります。
- 翻譯: 有一顆大石頭。
- Common mistake: Guessing the on-reading セキ (used in higher-level compounds like 石油) in everyday standalone use instead of the correct kun-reading いし.
- Memory tip: Pictograph resembling a rock under a cliff (厂 shape) — 石 depicts a stone at the base of a cliff.

**23. 土（つち）— 土壤**
- 例文: 土が黒いです。
- 翻譯: 土壤是黑色的。
- Common mistake: Confusing 土（つち, "soil/earth," here) with 土曜日（どようび, "Saturday," L4) — same kanji, different reading depending on standalone use vs. the day-name compound.
- Memory tip: You already know this kanji from 土曜日 (L4) — here it returns to its literal meaning, "earth/soil."

**24. 草（くさ）— 草**
- 例文: 庭に草があります。
- 翻譯: 院子裡有草。
- Common mistake: None major in meaning, but remember the 艹 (grass radical) on top is shared with many plant kanji like 花 (#8) and 茶 (お茶, L9) — a helpful pattern, easy to overlook at first.
- Memory tip: 艹 (grass radical) + 早 (early, shares a root with L21's 早く) = young grass sprouting early.

**25. 湖（みずうみ）— 湖**
- 例文: きれいな湖があります。
- 翻譯: 有一個漂亮的湖。
- Common mistake: Trying to sound out 湖 piece-by-piece from its individual on/kun readings — みずうみ is a fused native reading (水／みず "water" + 海／うみ "sea") that isn't predictable from the character's normal readings alone.
- Memory tip: Remember the reading as "water" (水) + "sea" (海) = "a landlocked sea" = lake.

## ② Kanji (3): 山, 川, 花

**山 — Meaning: 山 (mountain)**
- On-reading: サン
- Kun-reading: やま
- Words: 山, 富士山 (preview ふじさん), 火山 (preview かざん)
- Example sentence: 富士山は日本で一番高い山です。
- Stroke order: 3 strokes — center vertical stroke first, then the left and right strokes, resembling three peaks.
- Chinese difference: Same character and meaning as Chinese 山 (shān) — one of the most transparent kanji in the whole course. The only real adjustment is knowing when to use the on-reading サン (in named mountains like 富士山) versus the kun-reading やま (standalone or generic "a mountain").

**川 — Meaning: 川 (river)**
- On-reading: セン
- Kun-reading: かわ
- Words: 川, 河川 (preview かせん)
- Example sentence: この川はきれいです。
- Stroke order: 3 strokes — three vertical strokes left to right, depicting flowing water.
- Chinese difference: Same character and meaning as Chinese 川 (chuān), also highly transparent. Note that modern Mandarin more often uses 河/江 for "river" in everyday speech, while Japanese commonly uses 川 alone or in river names (隅田川, etc.) — a difference in frequency of use, not meaning.

**花 — Meaning: 花 (flower)**
- On-reading: カ
- Kun-reading: はな
- Words: 花, 花見 (preview はなみ), 花瓶 (preview かびん)
- Example sentence: 花がきれいに咲いています。
- Stroke order: 7 strokes — 艹 (grass radical, 3 strokes) on top first, then 化 (change/transform) below.
- Chinese difference: Same character and meaning as Chinese 花 (huā), fully transparent. Watch the homophone trap versus 鼻 (はな, "nose," L12) — identical pronunciation but completely unrelated meaning, distinguished only by kanji and context.

## ③ Grammar (3 points)

### Grammar 1 — 〜ので (because, softer)
**Meaning:** Gives a reason for something, sounding softer and more objective than から.
**Structure:** Plain form (い-adjective/verb) or [na-adjective/noun] + な + ので
**Examples:**
1. 忙しいので、寝ません。(Because I'm busy, I won't sleep [yet].)
2. 雨なので、傘を持っていきます。(Because it's raining, I'll bring an umbrella.)
3. 静かなので、この公園が好きです。(Because it's quiet, I like this park.)

**Common mistakes (Chinese speakers):**
- ❌ Forgetting to insert な before ので with a na-adjective or noun (❌ 静かので instead of ⭕ 静かなので) — this な-insertion rule trips up learners used to Chinese's connector-free sentence structure.
- ❌ Assuming ので and から are perfectly interchangeable in every context — ので tends to sound softer and more objective (good for polite/formal reasoning), while から can sound more direct and personal (see Grammar 2).

**Exercises (translate into Japanese):**
1. Because it is cold, I stay home.
2. Because I am a student, I don't have much money.
3. Because it is Sunday, I don't work.

### Grammar 2 — 〜から（理由, because, direct reason）
**Meaning:** Gives a reason, more direct and subjective than ので — common in casual speech and when justifying your own choices or feelings.
**Structure:** Plain form / です→だ + から
**Examples:**
1. 忙しいから、行きません。(Because I'm busy, I won't go.)
2. 好きだから、毎日食べます。(Because I like it, I eat it every day.)
3. 遅刻したから、走りました。(Because I was late, I ran.)

**Common mistakes (Chinese speakers):**
- ❌ Avoiding から in formal writing altogether — から is grammatically fine in formal contexts too, but textbooks often recommend ので for a softer tone; から isn't "wrong," just more direct.
- ❌ Forgetting だ before から with a na-adjective or noun in the plain form (❌ 好きから instead of ⭕ 好きだから) — です becomes だ in the plain form before から.

**Exercises (translate into Japanese):**
1. Because I am busy, I want to rest.
2. Because it is Sunday, I go shopping.
3. Because I forgot, I am sorry.

### Grammar 3 — 〜とき (when...)
**Meaning:** Marks the time or circumstance during which the main clause's action happens ("when X, Y").
**Structure:** Verb (plain form) / い-adjective / [na-adjective + な] / [noun + の] + とき
**Examples:**
1. 忙しいとき、休みません。(When I'm busy, I don't rest.)
2. 子供のとき、公園で遊びました。(When I was a child, I played in the park.)
3. 雨のとき、傘を持っていきます。(When it rains, I bring an umbrella.)

**Common mistakes (Chinese speakers):**
- ❌ Using は or が instead of の before とき with a noun (❌ 子供はとき instead of ⭕ 子供のとき) — nouns need の before とき, the same noun-modifying pattern from Lesson 2's 〜の.
- ❌ Misreading the tense inside the とき clause: 行くとき (before/while going) versus 行ったとき (after having arrived) describe different moments, and the clause's tense doesn't always match the main clause's tense the way learners expect.

**Exercises (translate into Japanese):**
1. When I am free, I read books.
2. When it rains, I stay home.
3. When I was a student, I studied a lot.

## 🇹🇼 Chinese Notes

- **虫**: This is the Japanese shinjitai (simplified) form of Traditional Chinese 蟲 (three insect radicals stacked, the form used in Taiwan). Like 学 (vs. 學) and 伝 (vs. 傳) in earlier lessons, this is another case where a Japanese kanji looks "simplified" compared to what Taiwanese learners are used to writing.
- **空（そら）**: One kanji, two very different everyday readings and meanings depending on grammatical role — そら as the noun "sky," and あく／すく as the verb "to become empty" (as in お腹が空く, L9). Chinese 空 also carries two readings (kōng "empty/sky," kòng "free time"), but the Japanese split is sharper and tied to a full verb conjugation Chinese doesn't have.
- **湖（みずうみ）**: Chinese 湖 (hú) has one simple, predictable reading. The Japanese reading みずうみ is a fused native compound of 水 (water) + 海 (sea) that cannot be sounded out from the character's normal on/kun readings — a reminder that some Japanese kanji readings must simply be memorized as whole words, not derived piece by piece.

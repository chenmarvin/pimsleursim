### Vocabulary in Depth

**1. 学校（がっこう）— school**
- Example: 学校はあそこです。
- Translation: 學校在那邊。
- Common mistake: Confusing 学校 (がっこう, the general institution) with 大学（だいがく, university, Lesson 1) — 学校 is the broad, everyday word for "school," while 大学 specifically means "university."
- Memory tip: 学（study, from 学生 in Lesson 1）+ 校（school, this lesson's kanji）= "a place of study."

**2. 会社（かいしゃ）— company**
- Example: 会社は町の中にあります。
- Translation: 公司在城鎮裡面。
- Common mistake: Confusing 会社 (the workplace/company itself) with 会社員（かいしゃいん, company employee, Lesson 1) — 会社 is the organization or building, 会社員 is the person who works there.
- Memory tip: 会（meet/gather）+ 社（organization）= "a gathering organization" = company.

**3. 駅（えき）— station**
- Example: 駅はどこですか。
- Translation: 車站在哪裡？
- Common mistake: Assuming 駅 must map onto a familiar Chinese character — Taiwan uses 站 (as in 火車站, 捷運站) for "station." 駅 has no common traditional-Chinese equivalent, so treat it as a genuinely new shape, not a variant of anything you already know.
- Memory tip: Learn 駅 fresh: the 馬 (horse) radical on the left recalls a time when travelers changed horses at stations along the road.

**4. 病院（びょういん）— hospital**
- Example: 病院は郵便局の後ろにあります。
- Translation: 醫院在郵局後面。
- Common mistake: Mishearing 病院（びょういん, hospital) as 美容院（びよういん, beauty salon) — the two words differ by only one vowel and are a classic listening trap for beginners.
- Memory tip: 病（illness）+ 院（institution）= "an institution for illness" = hospital.

**5. 銀行（ぎんこう）— bank**
- Example: 銀行はデパートの前です。
- Translation: 銀行在百貨公司前面。
- Common mistake: Because 銀行 looks identical to Chinese 銀行, learners often guess at the pronunciation instead of learning it properly — remember the vowel is long in こう (ぎんこう), not a short "ko."
- Memory tip: 銀（silver/money）+ 行（go, also used for institutions）— the logic matches Chinese 銀行 exactly, so this one is nearly a free pass.

**6. 郵便局（ゆうびんきょく）— post office**
- Example: 郵便局は図書館の左にあります。
- Translation: 郵局在圖書館左邊。
- Common mistake: Shortening 郵便局 to just 郵便 in speech — 郵便 alone means "mail/postal matter," not "the post office"; dropping 局 (bureau) changes the meaning.
- Memory tip: 郵便（mail）+ 局（bureau）= "the mail bureau" = post office.

**7. 図書館（としょかん）— library**
- Example: 図書館は学校の中にあります。
- Translation: 圖書館在學校裡面。
- Common mistake: Shortening the long vowel and saying としょか instead of としょかん — dropping the ん changes/breaks the word entirely.
- Memory tip: 図書（books/diagrams）+ 館（large building）= "a book building" = library.

**8. 公園（こうえん）— park**
- Example: 公園は町の外にあります。
- Translation: 公園在城鎮外面。
- Common mistake: Because 公園 looks and means exactly like Chinese 公園, learners sometimes flatten the pronunciation — remember こう has a long vowel, unlike a quick Mandarin-style "gong."
- Memory tip: 公（public）+ 園（garden）= "public garden" = park, same logic as the Chinese word.

**9. デパート — department store**
- Example: デパートは駅の前にあります。
- Translation: 百貨公司在車站前面。
- Common mistake: Treating デパート and スーパー as interchangeable — デパート (department store) is a large, upscale, multi-floor store, while スーパー (supermarket) is for daily groceries; the Chinese distinction 百貨公司 vs 超市 maps directly, so don't blur them in Japanese either.
- Memory tip: Katakana loanword from English "department (store)" — say it in four beats: de-pa-a-to.

**10. スーパー — supermarket**
- Example: スーパーはレストランの右にあります。
- Translation: 超市在餐廳右邊。
- Common mistake: Confusing スーパー (supermarket, groceries) with デパート (department store, upscale) — see note above; keep them as a clear pair.
- Memory tip: From English "super(market)" — short and easy: su-u-pa-a.

**11. レストラン — restaurant**
- Example: 友達とレストランに行きます。
- Translation: 我和朋友去餐廳。
- Common mistake: Stressing レストラン like the English word "restaurant" (with strong stress on the middle syllable) instead of the even, flat katakana rhythm re-su-to-ran.
- Memory tip: From English "restaurant" — the four syllables map almost directly onto the English word.

**12. トイレ — toilet, bathroom**
- Example: トイレはエレベーターの後ろです。
- Translation: 廁所在電梯後面。
- Common mistake: Assuming トイレ is impolite or too casual — it is actually the standard, neutral word for "restroom" in everyday Japanese, unlike some blunter native words.
- Memory tip: From English "toilet" — to-i-re, a high-priority survival word to memorize early.

**13. エレベーター — elevator**
- Example: エレベーターは建物の中にあります。
- Translation: 電梯在建築物裡面。
- Common mistake: Confusing エレベーター (elevator) with the similar-sounding loanword エスカレーター (escalator) — both appear together in buildings, so learn them as a contrasting pair.
- Memory tip: From English "elevator" — e-re-be-e-ta-a; note the long vowels (ー), which Chinese speakers often shorten by mistake.

**14. 道（みち）— road, way**
- Example: あの道は駅の前です。
- Translation: 那條路在車站前面。
- Common mistake: Confusing the spatial noun 道（みち, a physical road/path) with the abstract on-reading どう (as in 柔道, judo, not yet taught) — same kanji, very different reading and use.
- Memory tip: 道 keeps the same shape and core meaning ("road/way") as Chinese 道 — one of the easiest transfers in this lesson.

**15. 右（みぎ）— right**
- Example: 銀行は右です。
- Translation: 銀行在右邊。
- Common mistake: Mixing up 右（みぎ）and 左（ひだり）under time pressure — practice pointing with your hand while saying each word to build physical muscle memory alongside the vocabulary.
- Memory tip: 右 contains 口 (mouth) — picture using your right hand to bring food to your mouth.

**16. 左（ひだり）— left**
- Example: 公園は左です。
- Translation: 公園在左邊。
- Common mistake: Same 右/左 mix-up as above, compounded by pointing the wrong direction while saying the correct word — always practice 右 and 左 together, never in isolation.
- Memory tip: 左 contains 工 (a craftsman's tool) — picture holding a tool in your left hand while your right hand (右, with 口) feeds you.

**17. 前（まえ）— front, before**
- Example: 図書館は病院の前にあります。
- Translation: 圖書館在醫院前面。
- Common mistake: Assuming 前 always refers to time ("before") — here it is purely spatial ("in front of"); context (with a place vs. a time word) tells you which meaning applies.
- Memory tip: 前 keeps the same meaning as Chinese 前 (front/before) — lean directly on your Chinese intuition here.

**18. 後ろ（うしろ）— behind**
- Example: 銀行は郵便局の後ろです。
- Translation: 銀行在郵局後面。
- Common mistake: Dropping the okurigana and writing just 後 for "behind" — the spatial meaning specifically requires 後ろ; plain 後 usually signals a time-related "after," not a place.
- Memory tip: 後（behind/after）+ ろ marks it as the spatial word, distinguishing it from time-related 後.

**19. 中（なか）— inside**
- Example: 学校の中に図書館があります。
- Translation: 學校裡面有圖書館。
- Common mistake: Confusing spatial 中（なか, inside) with 中国（ちゅうごく, China, Lesson 1) — same kanji, completely different reading depending on whether it's a standalone spatial noun or part of a compound word.
- Memory tip: 中 looks like a line drawn straight through the middle of a box — literally "the inside/middle" of something.

**20. 外（そと）— outside**
- Example: 外に子供がいます。
- Translation: 外面有小孩。
- Common mistake: Confusing 外（そと, outside, spatial) with 外国（がいこく, foreign country, not yet taught) — same kanji, different reading depending on the word.
- Memory tip: 外 is the direct opposite of 中 — learn the pair together: 中（inside）⇔ 外（outside）.

**21. 建物（たてもの）— building**
- Example: あの建物は病院です。
- Translation: 那棟建築物是醫院。
- Common mistake: Using 建物 where a specific word (like 会社 or 病院) would be more natural — 建物 is a broad, somewhat formal term for "structure," best used when you don't know or don't need to specify what kind of building it is.
- Memory tip: 建てる（to build）+ 物（thing）= "a built thing" = building.

**22. 町（まち）— town**
- Example: この町に病院があります。
- Translation: 這個城鎮有醫院。
- Common mistake: Using 町 and 市 as if they were plain synonyms — 町（まち）is a smaller town/neighborhood-level unit, while 市（し）is an official city-level administrative unit; they are not interchangeable in formal contexts.
- Memory tip: 町 contains 田（rice field）+ 丁 — picture a small settlement that grew up around fields, smaller than a full city.

**23. 市（し）— city**
- Example: 私はこの市に住みます。
- Translation: 我住在這個城市。
- Common mistake: Inserting の between a place name and 市 (❌ 台北の市) — 市 attaches directly as a suffix onto a place name (台北市), unlike more independent nouns like 町 or 村.
- Memory tip: 市 keeps the same shape and meaning as Chinese 市 (city/market) — an easy direct transfer; the reading し is just a coincidental homophone of the number four's し, unrelated in meaning.

**24. 村（むら）— village**
- Example: おじいさんは村に住みます。
- Translation: 爺爺住在村莊。
- Common mistake: Reading 村 with the on-reading そん (used in compounds like 村人 or 農村) instead of the plain kun-reading むら needed for the standalone noun "village."
- Memory tip: 村 = 木（tree）+ 寸（a small measure）— a small settlement among the trees.

**25. 場所（ばしょ）— place**
- Example: この場所は公園です。
- Translation: 這個地方是公園。
- Common mistake: Reaching for 所（ところ, a more casual near-synonym, not yet taught) — for now, 場所 is the standard, safe word to use for "place/location" in any register.
- Memory tip: 場（location/field）+ 所（place）= a doubled-up "place" word, reinforcing the meaning through repetition.

## ② Kanji (5): 校, 右, 左, 駅, 中

**校（school）**
- Meaning (Chinese): 學校的「校」
- On-reading: コウ
- Kun-reading: — (no common kun-reading)
- Words: 学校（がっこう, this lesson）, 高校（こうこう, high school — bonus）
- Example sentence: 兄は高校の先生です。
- Stroke order tip: Write the 木（tree）radical on the left first (four strokes), then 交 on the right (the roof stroke, then the crossing "legs" strokes) — 10 strokes total.
- Chinese difference: Matches Chinese 校 (as in 學校) directly in shape and meaning — a low-risk, easy kanji.

**右（right）**
- Meaning (Chinese): 右邊
- On-reading: ウ、ユウ
- Kun-reading: みぎ
- Words: 右（みぎ, this lesson）, 左右（さゆう, left and right — bonus）
- Example sentence: 銀行は右です。
- Stroke order tip: The horizontal stroke is written first (sweeping slightly downward left to right), then 口 (mouth) is added underneath — note this stroke order is the opposite starting point from 左 below, a distinction drilled early in Japanese elementary schools.
- Chinese difference: Identical shape and meaning to Chinese 右; the two on-readings ウ／ユウ (heard together in 左右, さゆう) are the only new twist versus Mandarin's single reading yòu.

**左（left）**
- Meaning (Chinese): 左邊
- On-reading: サ
- Kun-reading: ひだり
- Words: 左（ひだり, this lesson）, 左右（さゆう, left and right — bonus, shared with 右）
- Example sentence: 公園は左です。
- Stroke order tip: The diagonal stroke is written first (top-left sweeping to lower-right), then 工 (work/tool) is added underneath — the opposite starting stroke from 右.
- Chinese difference: Identical shape and meaning to Chinese 左; only one on-reading サ, simpler than 右's two on-readings.

**駅（station）**
- Meaning (Chinese): 車站
- On-reading: エキ
- Kun-reading: — (no common kun-reading)
- Words: 駅（えき, this lesson）, 駅員（えきいん, station staff — bonus）
- Example sentence: 駅はどこですか。
- Stroke order tip: Write the 馬（horse）radical on the left first (it has its own multi-stroke sequence, top to bottom), then 尺 on the right — 14 strokes total, one of the more complex kanji in this lesson.
- Chinese difference: Genuinely new — Taiwan uses 站 (火車站, 捷運站) for "station," and there is no common traditional-Chinese character corresponding to 駅. Unlike most kanji in this lesson, this is not a simplified variant or a false friend of anything you already know; memorize it as a fresh shape and meaning.

**中（middle, inside）**
- Meaning (Chinese): 中間，裡面
- On-reading: チュウ
- Kun-reading: なか
- Words: 中（なか, this lesson）, 中国（ちゅうごく, China — Lesson 1）, 中学校（ちゅうがっこう, middle school — bonus）
- Example sentence: 学校の中に図書館があります。
- Stroke order tip: Write the outer box shape first (left stroke, then top-and-right stroke, then bottom stroke), then pierce the vertical stroke straight through the center last.
- Chinese difference: Identical shape to Chinese 中, but watch the reading shift: なか when used alone as a spatial noun ("inside," this lesson) versus ちゅう inside compounds like 中国 or 中学校 — same kanji, different reading depending on the word.

## ③ Grammar (2 points)

### Grammar 1 — 数字＋つ／人／個（助数詞, counters）

**Meaning:** Japanese attaches a "counter" word to a number depending on what is being counted. This lesson introduces three common counters: the generic つ counter (used for miscellaneous objects when no specific counter is known), 人 (for people), and 個 (for small, round, or discrete objects).

**Structure:** [Number] + [Counter]

**Reference table:**

| Number | つ (generic) | 人 (people) | 個 (small objects) |
|---|---|---|---|
| 1 | ひとつ | ひとり | いっこ |
| 2 | ふたつ | ふたり | にこ |
| 3 | みっつ | さんにん | さんこ |
| 4 | よっつ | よにん | よんこ |
| 5 | いつつ | ごにん | ごこ |

Note the irregulars: 1 and 2 people are ひとり／ふたり (not いちにん／ににん), and 4 people is よにん (not よんにん, even though the number 4 alone is よん).

**Examples:**
1. この町に公園が三つあります。(There are three parks in this town.)
2. 家族は四人です。(My family has four people.)
3. 財布を二個買いました。(I bought two wallets.)

**Common mistakes (Chinese speakers):**
- Overusing 個 as a universal counter for everything (❌ 学生が三個います) — 個 is only for small, discrete objects; people must use 人 (⭕ 学生が三人います).
- Saying よんにん instead of よにん for "four people" — 4 is normally よん, but 人 is a fixed exception that keeps the older reading よ.
- Forgetting that 1 and 2 people are irregular (❌ いちにん、ににん) — the correct forms are ひとり and ふたり.

**Exercises (translate into Japanese):**
1. There are two parks in this town.
2. My family has four people.
3. I bought three wallets.
4. Five students are in the library.

**Answer key:**
1. この町に公園が二つあります。
2. 家族は四人です。
3. 財布を三個買いました。
4. 図書館に学生が五人います。

### Grammar 2 — 〜は〜が好きです ("I like...")

**Meaning:** Expresses liking something. 好き ("liked, likable") is a な-adjective, not a verb — it describes the liked thing as being in a state of "likable-ness," so the liked object is marked with が, not を.

**Structure:** [Topic]は[Object]が好きです。

**Examples:**
1. 私は公園が好きです。(I like parks.)
2. 弟は図書館が好きです。(My younger brother likes the library.)
3. 姉はレストランが好きです。(My older sister likes restaurants.)

**Common mistakes (Chinese speakers):**
- ❌ 私は公園を好きです → ⭕ 私は公園が好きです — Mandarin 喜歡 behaves like a transitive verb taking a direct object, so Chinese speakers instinctively reach for を; but 好き is a な-adjective, and the thing liked is grammatically the "subject" of the adjective, marked by が.
- ❌ Conjugating 好き like a verb (好きます、好きました) — 好き conjugates like です／じゃありません／でした, the same way any な-adjective does, never with verb endings.

**Exercises (translate into Japanese):**
1. I like libraries.
2. My mother likes department stores.
3. Do you like parks?
4. My friend likes restaurants.

**Answer key:**
1. 私は図書館が好きです。
2. 母はデパートが好きです。
3. あなたは公園が好きですか。
4. 友達はレストランが好きです。

## 🇹🇼 Chinese Notes

- **駅（えき）**: There is no common traditional-Chinese character equivalent to 駅. Taiwan uses 站 for "station" (火車站, 捷運站). Don't look for a hanzi shortcut here — treat 駅 as a genuinely new kanji to memorize from scratch, unlike most other kanji in this lesson.
- **〜が好きです**: Mandarin 喜歡 acts like a transitive verb and takes a direct object directly, which pushes Chinese speakers toward using を with 好き. But 好き is a な-adjective in Japanese, not a verb, so the liked thing is marked with が — this is one of the most common Chinese-speaker mistakes in early N5 Japanese.
- **町 vs. 市 vs. 村**: Chinese doesn't always keep firm boundaries between "town," "city," and "village" in casual speech, but Japanese treats 町（まち）, 市（し）, and 村（むら）as distinct, formal administrative units of different sizes — city (市) > town (町) > village (村) — and they are not freely interchangeable, especially in official place names like 台北市 or 石垣市.

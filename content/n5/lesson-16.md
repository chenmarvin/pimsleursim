### Vocabulary in Depth

**1. 授業（じゅぎょう）— class, lesson**
- Example: 今日は日本語の授業があります。
- Translation: 今天有日文課。
- Common mistake: Saying ❌ 授業に行きます when you mean "attend class" — the natural set phrase is 授業に出ます or simply 授業があります／を受けます, not treating 授業 like a place you "go to."
- Memory tip: 授（give/impart）+ 業（learning, work）= "what the teacher imparts" — think of it as the thing the teacher gives, not the room.

**2. 宿題（しゅくだい）— homework**
- Example: 毎日、宿題があります。
- Translation: 每天都有功課。
- Common mistake: Confusing 宿題（homework given by a teacher）with 予習／復習（self-directed prep/review) — 宿題 is specifically assigned work.
- Memory tip: 宿（lodging, staying overnight）+ 題（topic/problem）= "a topic you take home [overnight] to work on."

**3. 試験（しけん）— exam**
- Example: 来週、試験があります。
- Translation: 下星期有考試。
- Common mistake: Using テスト and 試験 as if always interchangeable — テスト is often a small quiz, while 試験 tends to refer to a more formal/important exam; not wrong to know both, but don't assume they're always identical in weight.
- Memory tip: Pair with 合格する／失敗する below — 試験に合格する (pass the exam) / 試験に失敗する (fail the exam) are the two outcomes to memorize together.

**4. 教室（きょうしつ）— classroom**
- Example: 教室はとても静かです。
- Translation: 教室非常安靜。
- Common mistake: Confusing 教室（きょうしつ, classroom）with 教師（きょうし, teacher) — similar-sounding compounds built from the same first kanji 教 but with very different meanings.
- Memory tip: 教（teach）+ 室（room）= "the room where teaching happens."

**5. 会議（かいぎ）— meeting**
- Example: 会議は九時に始まります。
- Translation: 會議九點開始。
- Common mistake: Using 会議 for any casual get-together — 会議 specifically means a formal (often work-related) meeting, not a social gathering with friends.
- Memory tip: Shares 会 with 会社 (company) and 会う (to meet) — a 会議 is literally "a discussion (議) where people meet (会)."

**6. 用事（ようじ）— errand, business**
- Example: 今日は用事がありますから、行けません。
- Translation: 今天有事，所以不能去。
- Common mistake: Translating 用事 too literally word-for-word from Chinese — Taiwanese learners often reach for 事情 instead; 用事がある is the natural fixed Japanese phrase for "I have something to attend to," best learned as a set expression rather than built from parts.
- Memory tip: Memorize the whole phrase 用事があります (I have an errand/something to do) together, the way you'd memorize a single vocabulary chunk.

**7. 週末（しゅうまつ）— weekend**
- Example: 週末は何をしますか。
- Translation: 你週末要做什麼？
- Common mistake: Confusing 週末（しゅうまつ, weekend）with 今週／来週／先週 (this/next/last week, Lesson 4) — 週末 refers only to Saturday/Sunday, not the whole week.
- Memory tip: 週（week）+ 末（end）= "the end of the week."

**8. 始まる（はじまる）— to begin**
- Example: 授業は九時に始まります。
- Translation: 課九點開始。
- Common mistake: Using 始まる (intransitive: "[something] begins by itself") where 始める (transitive: "[someone] begins something") is needed — 授業が始まります (the class begins) vs. 先生が授業を始めます (the teacher begins the class).
- Memory tip: Pair it with its opposite 終わる below — both are intransitive verbs describing something happening to the subject on its own.

**9. 終わる（おわる）— to end**
- Example: 会議は何時に終わりますか。
- Translation: 會議幾點結束？
- Common mistake: Mixing up 終わる (intransitive, "to end") with 終える (transitive, "to finish something") — for N5 purposes, 終わる is enough: 授業が終わります.
- Memory tip: Learn 始まる／終わる as a matched pair, like "start/stop" — practice saying both about the same event: 授業は九時に始まって、十時半に終わります。

**10. 簡単な（かんたんな）— simple**
- Example: この試験は簡単でした。
- Translation: 這次考試很簡單。
- Common mistake: Forgetting this is a na-adjective and adding い by mistake (❌ 簡単い) or forgetting な before a noun (❌ 簡単試験 instead of ⭕ 簡単な試験).
- Memory tip: Opposite of 難しい (difficult, Lesson 8) — always review adjective pairs together.

**11. 頑張る（がんばる）— to do one's best**
- Example: 試験を頑張ります。
- Translation: 我會努力準備考試。
- Common mistake: Treating 頑張る as needing an object with を every time — it's often used alone as an encouragement: 頑張ってください／頑張ってね, without necessarily naming what.
- Memory tip: One of the most-used words in Japanese daily life — imagine a coach shouting 頑張って！ from the sidelines.

**12. 塾（じゅく）— cram school**
- Example: 子供は毎日、塾へ行きます。
- Translation: 小孩每天去補習班。
- Common mistake: Translating 補習班 directly and assuming any word with 補 exists in Japanese — the standard Japanese word is simply 塾, a single kanji, not a compound built the way the Chinese term is.
- Memory tip: Common in child-rearing conversations: 塾に通う (to attend cram school regularly).

**13. 留学（りゅうがく）— studying abroad**
- Example: 来年、アメリカへ留学します。
- Translation: 明年我要去美國留學。
- Common mistake: Saying ❌ 留学に行きます — the standard verb form is 留学する (noun+する), used directly: アメリカへ留学します, not "留学に行く."
- Memory tip: 留（stay）+ 学（study）= "staying [somewhere else] to study."

**14. 卒業する（そつぎょうする）— to graduate**
- Example: 来年、大学を卒業します。
- Translation: 明年我大學畢業。
- Common mistake: Using に instead of を with 卒業する — the school you graduate FROM takes を: 大学を卒業する, not ❌ 大学に卒業する.
- Memory tip: Pair with its opposite 入学する below: 入学する (enter) → 卒業する (graduate/exit).

**15. 入学する（にゅうがくする）— to enter school**
- Example: 今年、小学校に入学します。
- Translation: 今年我要上小學了。
- Common mistake: Using を instead of に — unlike 卒業する which takes を, 入学する takes に: 小学校に入学する, not ❌ 小学校を入学する — this を／に contrast is a common trap.
- Memory tip: Shares the kanji 入 with this lesson's kanji list — remember 入る (to enter) as the base meaning.

**16. 成績（せいせき）— grades**
- Example: 今学期の成績はよかったです。
- Translation: 這學期的成績很好。
- Common mistake: Confusing 成績（せいせき, overall grades/academic performance）with 点数（てんすう, a specific numeric score) — 成績 is the broader evaluation, 点数 is the raw number.
- Memory tip: 成（become/achieve）+ 績（accomplishment）= "the record of what you've achieved."

**17. 点数（てんすう）— score, points**
- Example: 試験の点数は八十点でした。
- Translation: 考試分數是八十分。
- Common mistake: Using 成績 when a specific number is meant — if you can attach a number like 八十点, use 点数, not 成績.
- Memory tip: 点（point/dot）+ 数（number）= "the number of points."

**18. 合格する（ごうかくする）— to pass (an exam)**
- Example: 試験に合格しました。
- Translation: 我考試通過了。
- Common mistake: Using を instead of に — the exam passed takes に: 試験に合格する, not ❌ 試験を合格する (unlike English "pass the exam," which takes a direct object).
- Memory tip: 合（fit/match）+ 格（standard/qualification）= "meeting the standard," i.e., passing.

**19. 失敗する（しっぱいする）— to fail**
- Example: 試験に失敗しました。
- Translation: 我考試失敗了。
- Common mistake: Same particle trap as 合格する — use に, not を: 試験に失敗する.
- Memory tip: Learn 合格する／失敗する together as opposites, both taking に for the exam.

**20. 提出する（ていしゅつする）— to submit**
- Example: 宿題を明日提出してください。
- Translation: 請明天交作業。
- Common mistake: Using 出す (a more casual "to put out/hand in") interchangeably without realizing 提出する is the more formal, standard classroom/office term — both exist, but 提出する is safer in written or formal contexts.
- Memory tip: 提（carry/present）+ 出（out）= "presenting [something] out [to the recipient]."

**21. 質問する（しつもんする）— to ask a question**
- Example: 分からない所を質問してください。
- Translation: 不懂的地方請發問。
- Common mistake: Using 聞く (to ask/listen, Lesson 6) as if identical to 質問する — 聞く is a general everyday verb for asking, while 質問する emphasizes formally posing a question (e.g., in class).
- Memory tip: 質（substance/nature）+ 問（ask）= "asking about the substance [of something]," i.e., a real question, not small talk.

**22. 答える（こたえる）— to answer**
- Example: 先生の質問に答えました。
- Translation: 我回答了老師的問題。
- Common mistake: Using を instead of に with 答える — the question being answered takes に: 質問に答える, not ❌ 質問を答える.
- Memory tip: Pair directly with 質問する: 質問する→答える, "ask" and "answer" as a matched action pair.

**23. 説明する（せつめいする）— to explain**
- Example: 先生が文法を説明してくれました。
- Translation: 老師幫我說明了文法。
- Common mistake: Confusing 説明する (to explain, give an account of something) with 答える (to answer a specific question) — 説明する is broader and doesn't require a prior question.
- Memory tip: 説（speak/theory）+ 明（clear/bright）= "speaking [to make things] clear."

**24. 予習する（よしゅうする）— to prepare for class**
- Example: 明日の授業を予習しました。
- Translation: 我預習了明天的課。
- Common mistake: Mixing up 予習する (preparing BEFORE class) with 復習する (reviewing AFTER class) — the kanji 予 (beforehand) vs. 復 (again/repeat) is the key visual distinction.
- Memory tip: 予（in advance）+ 習（learn/practice）= "learning in advance."

**25. 復習する（ふくしゅうする）— to review**
- Example: 試験の前に復習します。
- Translation: 考試前我會複習。
- Common mistake: Same trap as above in reverse — don't say 予習する when you mean reviewing material already taught; 復習 always looks backward, 予習 always looks forward.
- Memory tip: 復 also appears in 復習 and in the theme of review lessons in this course (復習 = Review) — every fifth lesson is literally called 復習.

## ② Kanji (3): 会, 社, 入

**会（meet, meeting）**
- Meaning (Chinese): 會
- On-reading: カイ
- Kun-reading: あ.う
- Words: 会社（かいしゃ）, 会議（かいぎ）, 会う（あう）
- Example sentence: 明日、大切な会議があります。
- Stroke order tip: The top "人＋一" lid comes first, then the enclosed 云-like body below; keep the whole character compact and slightly wide, 6 strokes.
- Chinese difference: 会 is Japan's own simplified form of the traditional 會 used in Taiwan — it is not the same as mainland China's simplification either, so Taiwanese learners used to 會 may not recognize 会 at first glance; treat it as a "third" form to memorize alongside 會.

**社（company, shrine）**
- Meaning (Chinese): 社
- On-reading: シャ
- Kun-reading: やしろ
- Words: 会社（かいしゃ）, 会社員（かいしゃいん）, 社会（しゃかい, society）
- Example sentence: 父は大きい会社で働いています。
- Stroke order tip: Write the 礻 (altar/spirit) radical on the left first (top dot, then the curving vertical), then 土 on the right — 7 strokes.
- Chinese difference: Identical in shape and core meaning to Chinese 社; note that its kun-reading やしろ (a Shinto shrine) has no equivalent concept tied to this character in Chinese, where 社 rarely means "shrine" on its own.

**入（enter)**
- Meaning (Chinese): 入
- On-reading: ニュウ
- Kun-reading: はい.る、い.れる
- Words: 入学する（にゅうがくする）, 入院する（にゅういんする, Lesson 12）, 入る（はいる）
- Example sentence: 来年、大学に入学します。
- Stroke order tip: Just two strokes — a short stroke from upper-right to lower-left, crossed by a longer stroke from upper-left to lower-right; don't confuse the stroke order with the visually similar 人 (person).
- Chinese difference: Same shape and meaning as Chinese 入, but be careful not to confuse it with 人 (person) — the two characters differ only in which stroke is longer, a distinction Chinese speakers sometimes overlook since both are equally simple in Chinese.

**Kanji Review:** Can you read these? 会社／入学する／会議／入る

## ③ Grammar

### Grammar 1 — 形容詞＋名詞 (adjective modifying a noun)

**Meaning/Structure:** An adjective can sit directly in front of a noun to describe it. い-adjectives attach directly; な-adjectives insert な before the noun.
- い形容詞 + 名詞 → 難しい試験 (a difficult exam)
- な形容詞 + な + 名詞 → 簡単な宿題 (simple homework)

**Examples:**
- 今日は簡単な授業でした。(Today's class was a simple one.)
- 難しい試験がありました。(There was a difficult exam.)
- 有名な学校で勉強しています。(I study at a famous school.)

**Common mistakes (Chinese speakers):**
- ❌ 簡単授業 → ⭕ 簡単な授業 — Chinese 簡單的課 needs no extra particle before the noun, so learners often drop な after na-adjectives.
- ❌ 難しいな試験 → ⭕ 難しい試験 — the reverse error: adding な to an い-adjective, which never takes な before a noun.

**Exercises (translate into Japanese):**
1. a difficult test
2. simple homework
3. a famous school
4. a new classroom

### Grammar 2 — 〜くて／〜で（形容詞の接続）(connecting adjectives)

**Meaning/Structure:** To link two adjectives together ("A and B"), い-adjectives change 〜い to 〜くて; な-adjectives (and nouns) add 〜で before the next adjective/clause.

**Examples:**
- この授業は簡単で、面白いです。(This class is simple and interesting.)
- 今日の試験は難しくて、大変でした。(Today's exam was difficult and hard.)
- 教室は静かで、きれいです。(The classroom is quiet and clean.)

**Common mistakes (Chinese speakers):**
- ❌ 難しいくて → ⭕ 難しくて — forgetting to drop the い before adding くて (the い of an i-adjective is replaced, not kept).
- ❌ 簡単くて → ⭕ 簡単で — using くて with a na-adjective instead of で; only true い-adjectives take くて.
- ❌ 難しいと大変です → ⭕ 難しくて大変です — using と (which links nouns) instead of the adjective-connecting form.

**Exercises (translate into Japanese):**
1. This test is difficult and long (長い).
2. The classroom is quiet and nice (いい).
3. He is kind (親切な) and smart (頭がいい).
4. Today's homework is simple and fun.

## ④ Reading

私は大学生です。毎日、学校へ行きます。今日の授業は簡単で、宿題も少ないです。でも、来週は試験がありますから、今週末は図書館で復習します。授業の後で、友達が「今度の週末、一緒に映画を見ませんか」と聞きました。私は「試験がありますから、今は忙しいです。試験の後で行きましょう」と答えました。友達は「頑張ってください」と言いました。

**Comprehension Questions:**
1. 今日の授業はどうでしたか。
2. 来週、何がありますか。
3. 友達は何と聞きましたか。

**Answer Key:**
1. 簡単でした。宿題も少なかったです。(It was simple, and there was little homework.)
2. 試験があります。(There is an exam.)
3. 「今度の週末、一緒に映画を見ませんか」と聞きました。("Won't you watch a movie together this coming weekend?")

## ⑤ Listening

*Read once slowly, then once at natural speed.*

A: 田中さん、今週の週末、何か用事がありますか。
B: いいえ、特にありません。どうしてですか。
A: 来週の試験が難しくて、心配です。一緒に図書館で勉強しませんか。
B: いいですよ。分からない所があったら、質問してください。私が説明します。
A: ありがとうございます。頑張って復習します。
B: 頑張ってくださいね。

**Comprehension Questions:**
1. 田中さんは週末、用事がありますか。
2. 二人は週末、何をしますか。

**Answer Key:**
1. いいえ、ありません。(No, there is none.)
2. 一緒に図書館で勉強します。(They will study together at the library.)

## 🇹🇼 Chinese Notes

- **会 is not quite 會**: 会 is Japan's own historical simplification of 會, different from both Taiwan's traditional 會 and mainland China's simplified 会 (which happens to look the same here, but that's a coincidence of two independent simplification systems, not a shared reform).
- **頑張る has no literal Chinese meaning**: Reading the characters 頑 (stubborn) and 張 (stretch) separately gives no hint of "try your best" — treat 頑張る as a fixed Japanese expression to memorize as a whole, not a compound to decode kanji-by-kanji.
- **用事 is a set phrase, not a direct translation of 事情**: Taiwanese Mandarin speakers reach for 事情 or 有事, but Japanese uses the fixed phrase 用事があります for "I have something to attend to" — learn it as one chunk.

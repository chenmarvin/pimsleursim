# 🇯🇵 Japanese for Taiwanese Learners — JLPT N5, Lesson 27

**Theme:** 「何を着ますか」(What are you going to wear?)

**Objectives**
- Talk about clothing and appearance
- Ask polite questions with 〜でしょうか
- Say something is too much with 〜すぎます

*(Note: this lesson's actual grammar points, per the syllabus, are 〜たり〜たりします／〜あげます／〜もらいます — see ③ Grammar below. 〜でしょうか and 〜すぎます are covered in Lesson 32.)*

### Word-by-word details

**1. 服（ふく）— 衣服**
- 例文: 新しい服を買いました。
- 翻譯: 我買了新衣服。
- Common mistake: Expecting 服 to need a companion word like Chinese 衣服 — in Japanese, 服 alone is already the complete, everyday word for "clothes."
- Memory tip: One syllable, one meaning — just "clothes," no extra kanji needed.

**2. シャツ — 襯衫**
- 例文: 白いシャツが好きです。
- 翻譯: 我喜歡白色的襯衫。
- Common mistake: Using 服 (clothes in general) when specifically talking about a shirt — シャツ names the specific garment.
- Memory tip: Katakana loanword from English "shirt."

**3. ズボン — 褲子**
- 例文: 新しいズボンを履きました。
- 翻譯: 穿了新褲子。
- Common mistake: Using 着る (#9, for upper-body garments) with ズボン — pants take 履く (#10), not 着る.
- Memory tip: From French "jupon," borrowed early into Japanese; just memorize it as "pants."

**4. スカート — 裙子**
- 例文: 黒いスカートを買いました。
- 翻譯: 買了黑色的裙子。
- Common mistake: Using 着る with スカート — like pants, skirts use 履く, not 着る, since they're worn from the waist down.
- Memory tip: Katakana loanword from English "skirt" — easy to recognize.

**5. 靴（くつ）— 鞋子**
- 例文: 新しい靴を買いました。
- 翻譯: 買了新鞋子。
- Common mistake: Confusing 靴（くつ）with 靴下（くつした, #20, "socks"）— 靴下 literally means "under the shoe."
- Memory tip: Pairs with 履く (#10): 靴を履く = "to put on shoes."

**6. 帽子（ぼうし）— 帽子**
- 例文: 白い帽子をかぶりました。
- 翻譯: 戴了白色的帽子。
- Common mistake: Using 着る or 履く for 帽子 instead of the correct verb かぶる (#11), reserved for head-worn items.
- Memory tip: 帽（cap）＋子（noun suffix）＝ hat; matches the Chinese loanword shape closely.

**7. かばん — 包包**
- 例文: かばんの中に本があります。
- 翻譯: 包包裡有書。
- Common mistake: Confusing with 財布 (さいふ, "wallet," Lesson 2) — かばん is the larger bag/carrier, not a wallet.
- Memory tip: A very old native word, unrelated to any kanji — just memorize the sound.

**8. 眼鏡（めがね）— 眼鏡**
- 例文: 彼は眼鏡をしています。
- 翻譯: 他戴著眼鏡。
- Common mistake: Trying to use 着る／履く／かぶる for glasses — 眼鏡 typically pairs with かける or する, not the other wear-verbs.
- Memory tip: 眼（eye）＋鏡（mirror/lens）＝ "eye lens" = glasses; visually close to Chinese 眼鏡.

**9. 着る（きる）— 穿（上衣）**
- 例文: 新しいシャツを着ます。
- 翻譯: 穿新的襯衫。
- Common mistake: Using 着る for shoes, pants, or hats — 着る is reserved for garments worn on the torso/body as a whole (shirts, dresses, coats).
- Memory tip: Pair with 服, シャツ, コート, セーター — anything you "put your body into."

**10. 履く（はく）— 穿（鞋、褲）**
- 例文: 靴を履いて出かけます。
- 翻譯: 穿上鞋子出門。
- Common mistake: Confusing with 着る — 履く is specifically for anything worn on the legs/feet (pants, skirts, shoes, socks).
- Memory tip: Legs and feet go "through" (履) something — pants, skirts, shoes, socks.

**11. かぶる — 戴（帽子）**
- 例文: 帽子をかぶります。
- 翻譯: 戴帽子。
- Common mistake: Using 着る for hats — Japanese has a dedicated verb かぶる just for headwear.
- Memory tip: Think of "covering" the head — かぶる sounds like something being placed over/covering something.

**12. 脱ぐ（ぬぐ）— 脫下**
- 例文: 家に入る前に靴を脱ぎます。
- 翻譯: 進家門前先脫鞋。
- Common mistake: Forgetting that 脱ぐ is the single opposite verb for 着る／履く／かぶる *all at once* — Chinese uses different verbs (脫/摘) depending on the item, but Japanese just uses 脱ぐ for taking off any clothing item.
- Memory tip: One "take off" verb for everything you wear — simpler than Chinese in this respect.

**13. 手袋（てぶくろ）— 手套**
- 例文: 寒いので、手袋をします。
- 翻譯: 因為很冷，所以戴手套。
- Common mistake: Expecting a word built from 手 + 套 like Chinese 手套 — Japanese instead uses 袋 ("bag"), giving 手袋 = "hand bag."
- Memory tip: 手（hand）＋袋（bag）＝ "hand bag" = glove.

**14. ネクタイ — 領帶**
- 例文: 会社にネクタイをして行きます。
- 翻譯: 打著領帶去公司。
- Common mistake: Trying to use 着る with ネクタイ — a necktie pairs with する, not 着る.
- Memory tip: Katakana loanword from English "necktie."

**15. コート — 大衣**
- 例文: 冬はコートを着ます。
- 翻譯: 冬天穿大衣。
- Common mistake: Confusing with セーター (#16, "sweater") — コート is an outer layer worn over other clothes, while セーター is a knit garment worn underneath a coat.
- Memory tip: Katakana loanword from English "coat," worn with 着る since it covers the torso.

**16. セーター — 毛衣**
- 例文: 暖かいセーターがほしいです。
- 翻譯: 我想要一件溫暖的毛衣。
- Common mistake: See #15 — mixing up コート and セーター in terms of layering order.
- Memory tip: Katakana loanword from English "sweater."

**17. かっこいい — 帥氣的**
- 例文: そのシャツはかっこいいですね。
- 翻譯: 那件襯衫很帥氣呢。
- Common mistake: Using かっこいい to describe a woman's appearance the way Chinese speakers might use 帥 loosely — かっこいい leans toward describing men, cool style, or cool objects; きれい／かわいい are more typical compliments for women.
- Memory tip: From 格好（かっこう, appearance/shape）＋いい（good）＝ "good-looking."

**18. 派手な（はでな）— 花俏的**
- 例文: そのスカートは派手ですね。
- 翻譯: 那件裙子好花俏喔。
- Common mistake: Assuming 派手 is always a compliment — in Japanese it can sound like a mild criticism (too flashy/loud), unlike enthusiastically complimenting bright colors in some contexts.
- Memory tip: Pairs as an opposite with 地味な (#24, "plain") — learn them together.

**19. 似合う（にあう）— 適合、相配**
- 例文: この帽子は似合いますね。
- 翻譯: 這頂帽子很適合你呢。
- Common mistake: Using を instead of は/が with 似合う, or trying to translate "suit" word-for-word from English "fit" — the person is usually marked with に, and the clothing item is the subject: 帽子が（あなたに）似合います.
- Memory tip: 似る（にる, "resemble"）＋合う（あう, "match/fit"）＝ "matches well."

**20. 靴下（くつした）— 襪子**
- 例文: 靴下を履いてから、靴を履きます。
- 翻譯: 先穿襪子，再穿鞋子。
- Common mistake: Confusing with 靴 (#5, "shoes") since both start with 靴 — 靴下 literally means "under the shoe" = socks.
- Memory tip: 靴（shoe）＋下（under）＝ "under the shoe" = sock.

**21. 指輪（ゆびわ）— 戒指**
- 例文: 誕生日に指輪をもらいました。
- 翻譯: 生日收到了戒指。
- Common mistake: Confusing the reading ゆびわ with related words like 指 (ゆび, "finger") alone — 指輪 is the specific compound for "ring."
- Memory tip: 指（finger）＋輪（ring/circle）＝ "finger circle" = ring.

**22. ワンピース — 連身裙**
- 例文: 姉はきれいなワンピースを着ています。
- 翻譯: 姊姊穿著漂亮的連身裙。
- Common mistake: Confusing with スカート (#4, "skirt") — ワンピース is a one-piece dress covering the whole body, while スカート is only the lower half.
- Memory tip: From English "one-piece" — a single, one-piece garment.

**23. 下着（したぎ）— 內衣**
- 例文: 下着も旅行のかばんに入れます。
- 翻譯: 內衣也要放進旅行的包包裡。
- Common mistake: None phonetically; mainly a vocabulary gap — Chinese speakers sometimes reach for a loanword instead of remembering this native compound.
- Memory tip: 下（under）＋着（wear, from 着る）＝ "underwear."

**24. 地味な（じみな）— 樸素的**
- 例文: このシャツは地味ですね。
- 翻譯: 這件襯衫很樸素呢。
- Common mistake: Treating 地味 as a negative insult — it's a neutral-to-mildly-positive description of a plain, understated style, not necessarily "boring."
- Memory tip: Pairs as an opposite with 派手な (#18) — learn the pair together.

**25. サンダル — 涼鞋**
- 例文: 夏はサンダルを履きます。
- 翻譯: 夏天穿涼鞋。
- Common mistake: Using 靴 (#5, general "shoes") when specifically meaning sandals — サンダル is the more specific term.
- Memory tip: Katakana loanword from English "sandal."

## ② Kanji (4): 服, 着, 色, 白

**服 — Meaning: 服 (clothes)**
- On-reading: フク
- Kun-reading: —
- Words: 服, 洋服 (preview)
- Example sentence: 新しい服を買いました。
- Stroke order: 8 strokes — 月 (flesh/moon radical, 4 strokes) on the left, then the 𠬝-like component on the right (4 strokes).
- Chinese difference: Same meaning as Chinese 服, but Chinese rarely uses 服 alone to mean "clothes" (衣服 is the norm); in Japanese, 服 alone is the everyday standalone word.

**着 — Meaning: 著 (wear, arrive)**
- On-reading: チャク
- Kun-reading: き.る
- Words: 着る, 上着 (preview)
- Example sentence: 新しいシャツを着ます。
- Stroke order: 12 strokes — the 羊-like top component first, then 目 at the bottom.
- Chinese difference: Same character as Chinese 著/着, but Japanese narrows the everyday reading to two core meanings: き.る "to wear" and つ.く/ちゃく "to arrive." Chinese 着 (zhe/zhuó/zháo) additionally works as a grammatical aspect particle (as in 看着) — a function that doesn't exist in Japanese at all.

**色 — Meaning: 色 (color)**
- On-reading: ショク、シキ
- Kun-reading: いろ
- Words: 色, 黄色い (Lesson 11)
- Example sentence: この色が好きです。
- Stroke order: 6 strokes — the top 勹-shaped stroke wrapping down, then the 巴-like component below.
- Chinese difference: Identical meaning to Chinese 色 (color), but Japanese also uses it in idiomatic words like 景色 (けしき, scenery) with special readings that don't map directly onto Chinese usage.

**白 — Meaning: 白 (white)**
- On-reading: ハク、ビャク
- Kun-reading: しろ、しろ.い
- Words: 白い (Lesson 11), 白
- Example sentence: 白いシャツが好きです。
- Stroke order: 5 strokes — a small diagonal stroke at the top-left, then the rest forms 日 with an added top stroke.
- Chinese difference: Same character/meaning as Chinese 白 — one of the most transparent kanji for Chinese speakers. Note that 面白い (Lesson 8, "interesting") uses 白 in a purely idiomatic way, not literally "white-faced" — a common over-analysis trap for Chinese speakers.

## ③ Grammar

### Grammar 1 — 〜たり〜たりします (doing things such as X and Y)
**Meaning:** Lists a few representative actions out of a larger set, without implying an exhaustive or strictly ordered list.
**Structure:** V1(た-form) ＋ り ＋ V2(た-form) ＋ り ＋ します
**Examples:**
1. 週末は買い物したり、映画を見たりします。(On weekends, I do things like shopping and watching movies.)
2. 旅行中に、写真を撮ったり、お土産を買ったりしました。(During the trip, I did things like taking photos and buying souvenirs.)
3. 部屋で本を読んだり、音楽を聞いたりします。(In my room, I do things like reading and listening to music.)

**Common mistakes (Chinese speakers):**
- Using only one 〜たり without a second action and forgetting します at the end — the pattern normally needs at least two representative actions, closed with します.
- Conjugating the second verb into a different tense than the first, breaking the parallel structure.

**Exercises (translate into Japanese):**
1. On weekends, I do things like reading and swimming.
2. I did things like trying on clothes and buying shoes.

### Grammar 2 — 〜あげます (I give — to someone else)
**Meaning:** The speaker (or someone in the speaker's group) gives something to another person outside that group.
**Structure:** (私は) 人に物をあげます
**Examples:**
1. 私は友達にプレゼントをあげます。(I give a present to my friend.)
2. 母に手袋をあげました。(I gave gloves to my mother.)
3. 誰にネクタイをあげますか。(Who are you giving the necktie to?)

**Common mistakes (Chinese speakers):**
- Using あげます when the recipient is the speaker — あげます is never used for something given *to* "me"; that direction requires もらいます or くれます (Grammar 3 below, and Lesson 28).
- Forgetting that giving to a clearly senior/superior person often calls for the more polite 差し上げます in real-world speech — for N5 purposes, あげます is the standard general form.

**Exercises (translate into Japanese):**
1. I give a hat to my younger sister.
2. Did you give a bag to Tanaka-san?

### Grammar 3 — 〜もらいます (I receive — from someone)
**Meaning:** The speaker (or someone in the speaker's group) receives something from another person.
**Structure:** (私は) 人に／から物をもらいます
**Examples:**
1. 私は友達に手袋をもらいました。(I received gloves from my friend.)
2. 誕生日にプレゼントをもらいました。(I received a present for my birthday.)
3. 先生から本をもらいました。(I received a book from my teacher.)

**Common mistakes (Chinese speakers):**
- Confusing に and から for marking the giver — both are possible, but から feels more natural when the giver is more distant or formal (a teacher, an organization), while に works generally for people.
- Confusing もらう (I receive) with あげる (I give) — the subject of the sentence and the direction of the action flip between the two.

**Exercises (translate into Japanese):**
1. I received a ring from my mother.
2. I received a coat from my friend.

## ④ Reading

今日はデパートへ買い物に行きました。新しい靴と帽子を買いました。店で服を着たり脱いだりしました。かっこいいシャツを選んで、それを買いました。友達の誕生日プレゼントに、手袋もあげました。友達もとても嬉しいです。

**Comprehension questions:**
1. 今日どこへ行きましたか。
2. 何を買いましたか。
3. 友達に何をあげましたか。

**Answer key:**
1. デパートへ行きました。(They went to a department store.)
2. 靴と帽子とシャツを買いました。(They bought shoes, a hat, and a shirt.)
3. 手袋をあげました。(They gave gloves.)

## 🇹🇼 Chinese Notes

- **着る vs. Chinese 着/著**: In Chinese, 着/著 mostly functions as a grammatical aspect particle (as in 正在看著), with no clothing meaning at all — "to wear" is expressed by 穿. Japanese 着る ("to wear," for shirts/dresses) is a completely different, everyday verb; don't expect the particle usage to carry over.
- **手袋（てぶくろ）**: Chinese speakers might expect a word built around 套 (as in 手套); 手袋 (て + 袋, "hand bag") is a distinctly Japanese way of naming gloves, unrelated to the Chinese word for the same item.
- **服**: In Taiwanese Mandarin, 服 rarely appears alone (衣服 is the norm); in Japanese, 服 by itself is the everyday, standalone noun for "clothes."

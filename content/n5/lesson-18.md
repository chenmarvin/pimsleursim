### Vocabulary in Depth

**1. 家（いえ）— house**
- Example: これは私の家です。
- Translation: 這是我的家。
- Common mistake: Confusing 家（いえ, the physical house/building）with うち（also written 家, meaning "home" in a warmer, more personal sense) — both exist and overlap; いえ is safe as the default N5 word.
- Memory tip: Shares the kanji 家 with 家族 (family, Lesson 3) — "家" is both the building and, in compounds, the people in it.

**2. 部屋（へや）— room**
- Example: 私の部屋は狭いです。
- Translation: 我的房間很小。
- Common mistake: Trying to build 部屋's meaning from its parts (部 = department/section, 屋 = shop/roof) — the combination is a fixed word meaning "room," not decodable piece by piece; memorize it as a whole.
- Memory tip: Pair with 一番 (Grammar 1 below): 部屋の中で、ベッドが一番大きいです — practice 部屋 inside comparison sentences.

**3. 台所（だいどころ）— kitchen**
- Example: 台所で料理を作ります。
- Translation: 我在廚房做菜。
- Common mistake: Using キッチン (the katakana loanword) and 台所 as if always interchangeable — both are used in modern Japanese, but 台所 is the traditional word and appears more often in textbooks and older speech.
- Memory tip: 台（platform/stand）+ 所（place）= "the place with the [cooking] platform."

**4. お風呂（おふろ）— bath**
- Example: 毎日お風呂に入ります。
- Translation: 我每天洗澡。
- Common mistake: Using を with 入る instead of に — the fixed phrase is お風呂に入る (literally "enter the bath"), not ❌ お風呂を入る.
- Memory tip: Always keep the honorific お attached — お風呂 without お sounds unnaturally blunt in everyday speech.

**5. 玄関（げんかん）— entrance**
- Example: 玄関で靴を脱ぎます。
- Translation: 在玄關脫鞋子。
- Common mistake: Assuming 玄関 means "front door" (ドア) — it actually refers to the whole entryway/foyer area (where shoes are removed), not the door itself.
- Memory tip: A distinctly Japanese-home concept: picture the small step-up area just inside the front door where shoes come off.

**6. ベッド — bed**
- Example: ベッドで寝ます。
- Translation: 我在床上睡覺。
- Common mistake: Confusing ベッド (a Western-style bed) with 布団 (a Japanese-style futon laid on the floor, this lesson's #25) — many Japanese homes use 布団 instead of ベッド, so don't assume every bedroom has one.
- Memory tip: From English "bed" — straightforward loanword.

**7. 机（つくえ）— desk**
- Example: 机の上に本があります。
- Translation: 書桌上有書。
- Common mistake: Confusing 机（つくえ, a desk for study/work）with テーブル（a general table, e.g. for meals) — 机 implies a work surface, テーブル is more general.
- Memory tip: The kanji 机 contains 木 (wood) + 几 (a small stand) — a wooden stand to work at.

**8. 椅子（いす）— chair**
- Example: この椅子に座ってください。
- Translation: 請坐這張椅子。
- Common mistake: Using に with 座る correctly is fine, but learners sometimes use を instead — 椅子に座る (sit ON/IN a chair) always takes に, not を.
- Memory tip: 椅 (a rare kanji meaning "chair") + 子 (suffix for small objects/tools) — often written mostly in kanji even at beginner level since it's so common.

**9. テーブル — table**
- Example: テーブルの上に晩ご飯があります。
- Translation: 桌上有晚餐。
- Common mistake: Using 机 where テーブル is meant, e.g. for a dining table — remember 机 leans "desk/study," テーブル leans "table for meals or general use."
- Memory tip: From English "table" — te-e-bu-ru.

**10. 窓（まど）— window**
- Example: 窓が大きくて、部屋は明るいです。
- Translation: 窗戶很大，房間很明亮。
- Common mistake: Confusing 窓（まど, window）with ドア（door) — different kanji, different function, but both are common "openings" in a room, so beginners sometimes mix up which vocabulary word to use when describing a wall.
- Memory tip: The kanji 窓 contains 穴 (hole) at the top — a window is a "hole" for light and air.

**11. ドア — door**
- Example: ドアを閉めてください。
- Translation: 請關門。
- Common mistake: Using 戸（と, a traditional sliding door) interchangeably with ドア（a Western-style hinged door) — modern apartments mostly use ドア, but older/traditional rooms may use 戸 or 障子.
- Memory tip: From English "door" — do-a.

**12. 棚（たな）— shelf**
- Example: 本は棚にあります。
- Translation: 書在架子上。
- Common mistake: Confusing 棚（たな, a shelf/rack）with 机（つくえ, desk) or 冷蔵庫（refrigerator) as general "storage" — 棚 specifically refers to open shelving.
- Memory tip: The kanji 棚 has 木 (wood) on the left — most shelves are traditionally wooden.

**13. 冷蔵庫（れいぞうこ）— refrigerator**
- Example: 冷蔵庫に卵があります。
- Translation: 冰箱裡有雞蛋。
- Common mistake: Shortening it incorrectly or confusing it with 冷凍庫（れいとうこ, freezer) — the middle kanji 蔵 (storage) vs. 凍 (freeze) is the key distinguishing part.
- Memory tip: 冷（cold）+ 蔵（storage）+ 庫（warehouse/storeroom）= "a cold storage warehouse," i.e. fridge.

**14. エアコン — air conditioner**
- Example: 夏はエアコンをつけます。
- Translation: 夏天要開冷氣。
- Common mistake: Using 開ける（あける, to open) instead of つける（to turn on) for electronics — appliances are turned "on" with つける, opened doors/windows use 開ける.
- Memory tip: Shortened from English "air conditioner" — a common katakana abbreviation pattern (compare パソコン from "personal computer," Lesson 19).

**15. 電気（でんき）— electricity, light**
- Example: 電気をつけてください。
- Translation: 請開燈。
- Common mistake: Assuming 電気 always means "electricity" in the abstract — in everyday speech 電気をつける／消す almost always means "turn the light on/off," not "turn on electricity" in general.
- Memory tip: Shares the kanji 電 with 電車 (Lesson 17) — 電 is the "electricity" kanji across many compounds.

**16. 引っ越す（ひっこす）— to move (residence)**
- Example: 来年、新しい部屋に引っ越します。
- Translation: 明年我要搬到新房間。
- Common mistake: Using 動く（うごく, to move physically, general movement) where 引っ越す（specifically to relocate one's residence) is meant — different verbs for "move an object" vs. "move house."
- Memory tip: 引く（pull）+ 越す（cross over）= "pulling [your belongings] across [to a new place]."

**17. 住所（じゅうしょ）— address**
- Example: ここに住所を書いてください。
- Translation: 請在這裡寫地址。
- Common mistake: Confusing 住所（じゅうしょ, a mailing/home address）with 場所（ばしょ, Lesson 7, a general place/location) — 住所 specifically refers to a registered address.
- Memory tip: 住（reside, from 住む, Lesson 6）+ 所（place）= "the place where you live."

**18. アパート — apartment**
- Example: 駅の近くのアパートに住んでいます。
- Translation: 我住在車站附近的公寓。
- Common mistake: Assuming アパート and マンション are the same — in Japan, アパート usually refers to a smaller, often older, low-rise building, while マンション refers to a larger, modern, concrete apartment building (opposite of the English "mansion" association).
- Memory tip: From English "apartment" — a-pa-a-to.

**19. マンション — condominium**
- Example: あのマンションは新しくて、きれいです。
- Translation: 那棟大廈很新也很漂亮。
- Common mistake: Translating マンション as "mansion" (a large luxury house) based on the English word it comes from — in Japanese it just means a modern multi-story apartment/condo building, not necessarily luxurious.
- Memory tip: A classic "false friend" loanword — remember it means "condo building," not "mansion."

**20. 広い（ひろい）— spacious**
- Example: この部屋は広いです。
- Translation: 這個房間很寬敞。
- Common mistake: Confusing 広い（ひろい, spacious/wide）with 大きい（おおきい, Lesson 8, big in size/volume) — 広い emphasizes floor area/openness, 大きい is more general size.
- Memory tip: Opposite of 狭い below — always learn adjective pairs together.

**21. 狭い（せまい）— cramped, narrow**
- Example: 私の部屋は狭いです。
- Translation: 我的房間很狹窄。
- Common mistake: Confusing 狭い（せまい, cramped/narrow）with 小さい（ちいさい, Lesson 8, small) — 小さい describes overall size, 狭い specifically describes lack of space/area.
- Memory tip: Pair with 広い as opposites: 広い⇔狭い, just like 大きい⇔小さい.

**22. 庭（にわ）— garden, yard**
- Example: 家に小さい庭があります。
- Translation: 家裡有一個小庭院。
- Common mistake: Confusing 庭（にわ, an outdoor garden/yard attached to a house）with 公園（こうえん, Lesson 7, a public park) — 庭 is private, attached to a residence.
- Memory tip: Picture a small private yard behind a traditional Japanese house — 庭 is intimate and personal, unlike the public 公園.

**23. 鍵（かぎ）— key**
- Example: 鍵を忘れました。
- Translation: 我忘了帶鑰匙。
- Common mistake: Using ドア (door) where 鍵 (key) is meant when talking about locking up — the fixed phrase is 鍵をかける (to lock, literally "to hang/apply the key"), not ❌ ドアをかける.
- Memory tip: 鍵 also means "the key/crux [of a problem]" metaphorically, similar to English "the key to success" — a nice bridge to remember the word beyond just physical keys.

**24. 洗濯機（せんたくき）— washing machine**
- Example: 洗濯機で服を洗います。
- Translation: 用洗衣機洗衣服。
- Common mistake: Confusing 洗濯機（せんたくき, the washing machine appliance）with 洗濯する（せんたくする, Lesson 6, the verb "to do laundry") — 洗濯機 is the noun for the machine itself.
- Memory tip: Built from 洗濯 (Lesson 6: to do laundry) + 機 (machine) — once you know 洗濯する, this compound is transparent.

**25. 布団（ふとん）— futon**
- Example: 布団を敷いて寝ます。
- Translation: 鋪棉被睡覺。
- Common mistake: Assuming 布団 refers to a Western-style thick comforter/duvet only — in Japan, 布団 traditionally refers to the whole floor-bedding set (mattress-like base + cover) used instead of a bed frame.
- Memory tip: 布（cloth）+ 団（round/mass）= "a rounded mass of cloth [to sleep on]."

## ② Kanji (3): 家, 部, 屋

**家（house, home）**
- Meaning (Chinese): 家
- On-reading: カ、ケ
- Kun-reading: いえ、や
- Words: 家（いえ）, 家族（かぞく, Lesson 3）, 家庭（かてい, Lesson 3）
- Example sentence: 家族はみんな家にいます。
- Stroke order tip: The roof radical 宀 comes first (dot, then the two slanting strokes), then 豕 (pig) is built underneath stroke by stroke — 10 strokes; the classical idea is "a pig under a roof" representing a household.
- Chinese difference: Identical in shape and core meaning to Chinese 家; note the two readings いえ (a physical house) and うち/や (used in compounds like 家賃, やちん, rent) — Chinese doesn't split the reading this way.

**部（part, section)**
- Meaning (Chinese): 部
- On-reading: ブ
- Kun-reading: — (no common kun-reading at N5 level)
- Words: 部屋（へや）, 全部（ぜんぶ, Lesson 2）, 部分（ぶぶん, part/portion）
- Example sentence: この部屋は全部で三つあります。
- Stroke order tip: Write the left component 咅 first (its own multiple strokes), then the ⻏ (mound/city) radical on the right last — 11 strokes.
- Chinese difference: Same shape and meaning as Chinese 部 (department/section/part); but in the compound 部屋 the meaning shifts to simply "room," a usage that doesn't exist in Chinese, where 部 alone never means "room."

**屋（shop, roof)**
- Meaning (Chinese): 屋
- On-reading: オク
- Kun-reading: や
- Words: 部屋（へや）, パン屋（ぱんや, bakery, from パン Lesson 9）, 屋上（おくじょう, rooftop）
- Example sentence: 駅の前にパン屋があります。
- Stroke order tip: The 尸 (corpse/roof-like) radical goes on top, then 至 is built underneath — 9 strokes; treat 尸 as a simplified roof shape.
- Chinese difference: In Chinese, 屋 commonly means "house/room" on its own (屋子, 房屋); in Japanese, as a suffix 屋 usually means "shop" (パン屋 = bakery, 花屋 = flower shop) or "roof" (屋上), not a standalone word for "house" — a genuine false-friend trap worth remembering.

## ③ Grammar

### Grammar 1 — 〜の中で〜が一番〜です (of all X, Y is the most...)

**Meaning/Structure:** To say something is the "most ~" within a defined group, name the group with の中で ("among/within") and mark the winning item with が一番 ("is the most").
Structure: [Group] の中で、[Item] が一番 [adjective] です。

**Examples:**
- 部屋の中で、ベッドが一番大きいです。(Among the things in the room, the bed is the biggest.)
- 一年の中で、夏が一番暑いです。(Of the whole year, summer is the hottest.)
- 家族の中で、父が一番忙しいです。(Among my family, my father is the busiest.)

**Common mistakes (Chinese speakers):**
- ❌ 部屋の中、ベッドが一番大きいです → ⭕ 部屋の中で — で is required after 中 when describing the scope of a comparison; dropping it is a common omission.
- ❌ 部屋の中に、ベッドが一番大きいです → ⭕ 部屋の中で — に marks the location of physical existence (Lesson 12's があります／います pattern), but で marks the scope for a superlative comparison; these two easily get mixed up.
- ❌ ベッドは一番大きいです (dropping が) → ⭕ ベッドが一番大きいです — the item being singled out as "the most" takes が, not は.

**Exercises (translate into Japanese):**
1. Of all the rooms, the kitchen is the smallest.
2. Of all the seasons, winter is the coldest.
3. Of all my friends, Tanaka is the most famous.
4. Of all my family, my mother is the busiest.

### Grammar 2 — 〜がほしいです (I want [a thing])

**Meaning/Structure:** To express wanting a noun/object, mark the desired thing with が and use the i-adjective 欲しい ("wanted/desirable").
Structure: [Noun] が欲しいです。

**Examples:**
- 新しい机が欲しいです。(I want a new desk.)
- 広い部屋が欲しいです。(I want a spacious room.)
- 洗濯機が欲しいです。(I want a washing machine.)

**Common mistakes (Chinese speakers):**
- ❌ 机を欲しいです → ⭕ 机が欲しいです — 欲しい is an i-adjective describing a feeling ("desirable to me"), so its object takes が, not を, even though English "want A THING" suggests a direct object.
- Confusing 〜がほしいです (want a THING, a noun) with 〜たいです (want to DO something, an action verb, introduced in Lesson 19) — 欲しい only ever attaches to nouns, never directly to a verb.

**Exercises (translate into Japanese):**
1. I want a new refrigerator.
2. I want a bigger apartment.
3. I want an air conditioner.
4. I want my own room.

## ④ Reading

私は今、狭いアパートに住んでいます。部屋の中に、ベッドと机と椅子があります。台所は小さいですが、冷蔵庫と洗濯機があります。窓が大きくて、部屋はきれいです。でも、夏はエアコンがないので、とても暑いです。来年、新しいマンションに引っ越します。新しい家には、広い部屋と大きい台所がほしいです。庭があるうちが一番好きです。

**Comprehension Questions:**
1. 今のアパートはどうですか。
2. 来年、何をしますか。
3. 新しい家に何がほしいですか。

**Answer Key:**
1. 狭いです。台所は小さいですが、冷蔵庫と洗濯機があります。(It's cramped. The kitchen is small, but there's a refrigerator and a washing machine.)
2. 新しいマンションに引っ越します。(They will move to a new condominium.)
3. 広い部屋と大きい台所がほしいです。(They want a spacious room and a big kitchen.)

## ⑤ Listening

*Read once slowly, then once at natural speed.*

A: 新しい部屋はどうですか。
B: とても広いですよ。窓も大きくて、きれいです。
A: 家具は何がありますか。
B: ベッドと机と椅子と、冷蔵庫もあります。
A: いいですね。エアコンもありますか。
B: はい、ありますよ。でも、洗濯機はまだありません。
A: 洗濯機がほしいですか。
B: はい、早く買います。

**Comprehension Questions:**
1. 新しい部屋に何がありますか。
2. 洗濯機はありますか。

**Answer Key:**
1. ベッド、机、椅子、冷蔵庫、エアコンがあります。(A bed, desk, chair, refrigerator, and air conditioner.)
2. いいえ、まだありません。(No, not yet.)

## 🇹🇼 Chinese Notes

- **屋 is a false friend**: In Chinese, 屋 alone commonly means "house/room" (屋子, 房屋). In Japanese, as a suffix, 屋 usually means "shop" (パン屋 = bakery, 花屋 = flower shop) rather than "house" — don't assume 屋 always means a dwelling.
- **部屋 isn't the sum of its parts**: 部 (department/section) + 屋 (shop/roof) combine into a fixed word meaning simply "room" — memorize 部屋 as a whole vocabulary item rather than trying to derive its meaning from the individual kanji.
- **マンション ≠ "mansion"**: Despite coming from the English word "mansion," マンション just means an ordinary modern multi-story apartment/condo building in Japanese — it says nothing about size or luxury the way "mansion" does in English.

### Vocabulary in Depth

**1. 天気（てんき）— weather**
- Example: 今日は天気がいいです。
- Translation: 今天天氣很好。
- Common mistake: Saying ❌ 天気は暑いです instead of the correct pattern 今日は暑いです (weather-related adjectives usually attach to 今日/日 as the topic, not to 天気 itself, when describing hot/cold).
- Memory tip: 天（sky/heaven）+ 気（air/energy）= "the condition of the sky."

**2. 晴れ（はれ）— sunny**
- Example: 明日は晴れです。
- Translation: 明天是晴天。
- Common mistake: Confusing the noun 晴れ (sunny weather, used with です) with the verb 晴れる (to clear up, item 18 below) — 晴れです describes a state, 晴れます describes a change happening.
- Memory tip: Contains 日 (sun) inside the kanji 晴 — visually linked to sunshine.

**3. 雨（あめ）— rain**
- Example: 雨が降っています。
- Translation: 正在下雨。
- Common mistake: Using を with 降る instead of が — 雨が降る (rain falls) always uses が for the subject, not を.
- Memory tip: The kanji 雨 looks like raindrops falling from a cloud — one of the most pictographic kanji in this lesson.

**4. 雪（ゆき）— snow**
- Example: 冬に雪が降ります。
- Translation: 冬天會下雪。
- Common mistake: Confusing 雪（ゆき, snow）with 雲（くも, cloud) — visually different kanji but both relate to sky phenomena; keep them distinct by their radicals (雨 vs 雨-less 云 shape).
- Memory tip: 雨 (rain radical) on top + 彗-like shape below suggesting soft falling flakes.

**5. 曇り（くもり）— cloudy**
- Example: 今日は曇りです。
- Translation: 今天是陰天。
- Common mistake: Treating 曇り as an adjective and saying ❌ 曇りい — it's a plain noun, used with です like 晴れ and 雨.
- Memory tip: 曇 contains 雲 (cloud) hidden inside its structure — clouds covering the sun (日).

**6. 風（かぜ）— wind**
- Example: 今日は風が強いです。
- Translation: 今天風很大。
- Common mistake: Confusing 風（かぜ, wind）with 風邪（かぜ, a cold, Lesson 12）— identical pronunciation, different kanji and meaning; context always disambiguates (weather vs. illness).
- Memory tip: Associate 風 with 台風 (typhoon) — a huge, memorable weather word using the same character.

**7. 台風（たいふう）— typhoon**
- Example: 今週、台風が来ます。
- Translation: 這週會有颱風。
- Common mistake: Using 来ます literally as "come" without realizing 台風が来る is the standard fixed way to say "a typhoon is approaching," even though English might say "hit" or "arrive."
- Memory tip: 台 (platform/pedestal, also appears in 台湾) + 風 (wind) — a very relevant word for Taiwanese learners, since Taiwan gets many typhoons too.

**8. 春（はる）— spring**
- Example: 春は暖かいです。
- Translation: 春天很溫暖。
- Common mistake: Using に unnecessarily after season names in general statements — 春は暖かいです (topic) is more natural than 春に暖かいです when making a general claim about the season itself.
- Memory tip: 春 contains 日 at the bottom, suggesting sunlight warming the new year.

**9. 夏（なつ）— summer**
- Example: 夏は暑いです。
- Translation: 夏天很熱。
- Common mistake: Confusing 夏（なつ, summer）with 何（なに/なん, what, Lesson 1) due to similar vowel sounds — different kanji, no real overlap once written.
- Memory tip: Pair with 暑い (Lesson 8: hot) — 夏は暑いです is a natural, easy-to-remember sentence.

**10. 秋（あき）— autumn**
- Example: 秋は涼しいです。
- Translation: 秋天很涼爽。
- Common mistake: Confusing 秋（あき, autumn）with 開き／空き（あき, an opening/vacancy, unrelated homophones) — context (season vs. availability) disambiguates.
- Memory tip: 秋 contains 禾 (grain) + 火 (fire), evoking the harvest season and autumn colors.

**11. 冬（ふゆ）— winter**
- Example: 冬は寒いです。
- Translation: 冬天很冷。
- Common mistake: Forgetting winter pairs with 寒い (さむい, cold weather, Lesson 8), not 冷たい (つめたい, cold to the touch, used for objects/drinks) — 冬は寒いです, not ❌ 冬は冷たいです.
- Memory tip: Group all four seasons together as a set: 春・夏・秋・冬 — memorize them in this fixed order, as Japanese always lists them this way.

**12. 涼しい（すずしい）— cool (weather)**
- Example: 秋の朝は涼しいです。
- Translation: 秋天的早晨很涼爽。
- Common mistake: Confusing 涼しい（すずしい, pleasantly cool, positive nuance）with 寒い（さむい, cold, often uncomfortable）— 涼しい carries a refreshing, pleasant connotation, unlike 寒い.
- Memory tip: Contains 水 (water) radical on the left — coolness associated with water/breeze.

**13. 暖かい（あたたかい）— warm (weather)**
- Example: 春は暖かくて、気持ちがいいです。
- Translation: 春天很溫暖，很舒服。
- Common mistake: Confusing 暖かい（あたたかい, warm weather/air）with 温かい（あたたかい, warm to the touch, e.g. food/drinks）— same reading, different kanji and typical usage context.
- Memory tip: Contains 日 (sun) on the left — sunlight bringing warmth.

**14. 気温（きおん）— temperature**
- Example: 今日の気温は二十五度です。
- Translation: 今天氣溫是25度。
- Common mistake: Confusing 気温（きおん, air temperature specifically）with 体温（たいおん, body temperature) — same 温 character, different context word in front.
- Memory tip: 気（air）+ 温（warmth）= "the warmth of the air," i.e. temperature.

**15. 予報（よほう）— forecast**
- Example: 天気予報を見ました。
- Translation: 我看了天氣預報。
- Common mistake: Dropping 天気 and using 予報 alone to mean "weather forecast" in contexts where it could be ambiguous with other kinds of forecasts (地震予報, etc.) — 天気予報 is the fixed, complete term.
- Memory tip: 予（in advance）+ 報（report）= "an advance report," i.e. a forecast.

**16. 傘（かさ）— umbrella**
- Example: 傘を持っていますか。
- Translation: 你有帶雨傘嗎？
- Common mistake: Using 持ちます alone without を, saying ❌ 傘持ちます — object particle を is still required: 傘を持つ.
- Memory tip: The kanji 傘 visually resembles an open umbrella with its ribs — a rare pictographic kanji for a modern object.

**17. 濡れる（ぬれる）— to get wet**
- Example: 傘がないので、濡れました。
- Translation: 因為沒帶傘，我淋濕了。
- Common mistake: Using を with 濡れる, but it's an intransitive verb describing a state change — the subject that gets wet takes が: 服が濡れました, not ❌ 服を濡れました.
- Memory tip: Contains 水 (water) radical on the left — directly tied to water/wetness.

**18. 晴れる（はれる）— to clear up**
- Example: 午後には晴れるでしょう。
- Translation: 下午應該會放晴。
- Common mistake: Confusing the verb 晴れる (weather changing to become sunny) with the noun 晴れ (item 2 above, describing the state) — 晴れます describes the transition, 晴れです describes the current condition.
- Memory tip: Same kanji as 晴れ (noun) — る added turns it into a verb, a common noun→verb pattern to notice.

**19. 降る（ふる）— to fall (rain/snow)**
- Example: 明日は雪が降るでしょう。
- Translation: 明天大概會下雪。
- Common mistake: Using 落ちる (to fall/drop, for objects) instead of 降る specifically for precipitation — 雨/雪が降る is the fixed verb for weather falling from the sky.
- Memory tip: Pair fixed phrases: 雨が降る, 雪が降る — always が, never を or に.

**20. 吹く（ふく）— to blow (wind)**
- Example: 強い風が吹いています。
- Translation: 現在正吹著強風。
- Common mistake: Using が with 吹く but then adding an unnecessary を, saying ❌ 風を吹きます when talking about the wind blowing naturally (intransitive) — 風が吹く is correct for natural wind; を吹く is only used for actively blowing something like a whistle.
- Memory tip: Contains 口 (mouth) radical — blowing air out of a mouth-like shape.

**21. 虹（にじ）— rainbow**
- Example: 雨の後に虹が出ました。
- Translation: 雨後出現了彩虹。
- Common mistake: Using 見えます (can be seen) interchangeably with 出ます (appears/comes out) without noticing 虹が出る is the more standard fixed collocation for a rainbow appearing.
- Memory tip: Contains 虫 (insect radical, historically used for creatures/curved shapes) + 工 — an old association between rainbows and mythical creatures.

**22. 雷（かみなり）— thunder**
- Example: 雷が鳴っています。
- Translation: 正在打雷。
- Common mistake: Using 音がします (a sound is made) generically instead of the specific fixed verb 鳴る (to resound/ring) that's typically paired with 雷: 雷が鳴る.
- Memory tip: Contains 雨 (rain radical) on top + 田 (field) below — an old depiction of lightning striking fields.

**23. 湿度（しつど）— humidity**
- Example: 夏は湿度が高いです。
- Translation: 夏天濕度很高。
- Common mistake: Using 高い for humidity correctly (湿度が高い) but mistakenly using 多い (many, for countable things) instead — humidity, like temperature, uses 高い/低い, not 多い/少ない.
- Memory tip: 湿（damp）+ 度（degree）= "degree of dampness."

**24. 四季（しき）— four seasons**
- Example: 台湾より、日本のほうが四季がはっきりしています。
- Translation: 比起台灣，日本的四季更分明。
- Common mistake: Confusing 四季（しき, the four seasons as a set concept）with simply listing 春夏秋冬 — 四季 is the collective abstract noun, useful for talking about seasonal variety as a topic.
- Memory tip: 四（four）+ 季（season）= literally "four seasons," a transparent compound.

**25. 気候（きこう）— climate**
- Example: 台湾の気候は暖かいです。
- Translation: 台灣的氣候很溫暖。
- Common mistake: Confusing 気候（きこう, climate, long-term pattern）with 天気（てんき, weather, day-to-day condition）— 気候 describes general/regional patterns, 天気 describes today's specific conditions.
- Memory tip: 気（air）+ 候（season/time）— related to 天候 (てんこう, weather conditions), a near-synonym pair worth noting.

## ② Kanji (3): 天, 気, 雨

**天（heaven, sky）**
- Meaning (Chinese): 天
- On-reading: テン
- Kun-reading: あめ
- Words: 天気, 天気予報, 雨天（うてん, rainy weather）
- Example sentence: 今日は天気がいいです。
- Stroke order tip: Two horizontal strokes at top (shorter then longer), then the crossing diagonal strokes below forming a person-like shape — 4 strokes total.
- Chinese difference: Identical to Chinese 天 in shape and core meaning ("sky/heaven"); the reading あめ (as in 雨天) is unusual to Chinese speakers, who would expect 天 alone to never sound like "rain."

**気（spirit, air)**
- Meaning (Chinese): 氣
- On-reading: キ、ケ
- Kun-reading: —
- Words: 天気, 元気, 気温
- Example sentence: 今日はいい天気ですね。
- Stroke order tip: The 米-like internal strokes are drawn after the enclosing 气-shaped frame — practice this one slowly, as the simplified Japanese form (気) differs visually from traditional Chinese 氣.
- Chinese difference: This is the simplified Japanese form of traditional Chinese 氣 (simplified Chinese uses 气, also different). Meaning "energy/spirit/air" carries over, but the written form is distinct from both traditional and simplified Chinese — don't try to write it as 氣 or 气.

**雨（rain）**
- Meaning (Chinese): 雨
- On-reading: ウ
- Kun-reading: あめ
- Words: 雨, 大雨（おおあめ, heavy rain）, 雨天
- Example sentence: 雨が降っています。
- Stroke order tip: Draw the top horizontal line and outer frame first, then the four dots inside representing raindrops, added last.
- Chinese difference: Identical to Chinese 雨 in shape and meaning; note that as a radical inside other kanji (雪, 雲, 電, 雷), it always keeps this same raindrop-frame shape, which is a big reading shortcut for weather-related kanji.

**Kanji Review:** Can you read these? 天気／雨／気温／天気予報

## ③ Grammar

### Grammar 1 — い形容詞＋です (i-adjective + です)

**Meaning/Structure:** I-adjectives (ending in い) can attach directly to です to make a polite present-tense description, without needing な or を. Structure: [Noun] は [i-adjective]です.

**Examples:**
- 今日は暑いです。(Today is hot.)
- この傘は新しいです。(This umbrella is new.)
- 冬は寒いです。(Winter is cold.)

**Common mistakes (Chinese speakers):**
- ❌ 今日は暑いなです → ⭕ 今日は暑いです — learners familiar with na-adjectives (Lesson 8/14) sometimes mistakenly add な before です for i-adjectives; i-adjectives never take な before です.
- ❌ 今日は暑くです → ⭕ 今日は暑いです — the plain present form already ends in い; don't conjugate to くです for a simple affirmative statement (くて/くない are for connecting or negating, covered elsewhere).

**Exercises (translate into Japanese):**
1. This umbrella is expensive.
2. Summer is hot in Taiwan.
3. That mountain (山) is high.
4. Today's weather is good.

### Grammar 2 — い形容詞の否定形（〜くないです）(i-adjective negative form)

**Meaning/Structure:** To negate an i-adjective politely, drop the final い and add くないです. Structure: [i-adjective stem] く ない です. (Exception: いい → よくないです.)

**Examples:**
- 今日は寒くないです。(Today is not cold.)
- この傘は高くないです。(This umbrella is not expensive.)
- 天気はよくないです。(The weather isn't good.)

**Common mistakes (Chinese speakers):**
- ❌ 寒いないです → ⭕ 寒くないです — learners forget to change い to く before adding ない; the い must drop first.
- ❌ いいくないです → ⭕ よくないです — いい is irregular: for negation, revert to the older stem よい before conjugating.

**Exercises (translate into Japanese):**
1. Today is not hot.
2. This bag is not small.
3. The exam was not difficult (careful: use present tense here for practice).
4. This coffee is not good.

## ④ Reading

台湾の夏はとても暑いです。気温は三十五度ぐらいで、湿度も高いです。でも、冬はあまり寒くないです。日本の冬のほうが台湾より寒いです。私は日本の四季の中で、秋が一番好きです。秋は涼しくて、天気もいいです。今日は曇りですが、天気予報によると、明日は晴れるでしょう。傘は要らないと思います。

**Comprehension Questions:**
1. 台湾の夏はどんな天気ですか。
2. 台湾と日本と、どちらの冬が寒いですか。
3. 明日の天気はどうですか。

**Answer Key:**
1. とても暑くて、湿度も高いです。(Very hot, and the humidity is also high.)
2. 日本のほうが寒いです。(Japan's winter is colder.)
3. 明日は晴れるでしょう。(It will probably clear up tomorrow.)

## ⑤ Listening

*Read once slowly, then once at natural speed.*

A: 今日の天気予報を見ましたか。
B: いいえ、まだです。どうでしたか。
A: 午前中は曇りで、午後から雨が降るそうです。
B: え、本当ですか。傘を持っていないです。
A: 大丈夫ですよ。私の傘、二つありますから、貸しますね。
B: ありがとうございます。助かります。

**Comprehension Questions:**
1. 午後の天気はどうなりますか。
2. Bさんは傘を持っていますか。

**Answer Key:**
1. 雨が降ります。(It will rain.)
2. いいえ、持っていません。(No, they don't have one.)

## 🇹🇼 Chinese Notes

- **気 is not written like 氣 or 气**: Japanese uses its own simplified form 気, distinct from both traditional Chinese 氣 and simplified Chinese 气 — a common writing mistake for Chinese speakers who default to one of their native forms.
- **暖かい vs 温かい**: Both are read あたたかい, but 暖かい is for weather/air warmth while 温かい is for the warmth of objects like food or water — Chinese 暖/溫 map similarly but aren't always distinguished as strictly in casual Chinese speech.
- **風邪 vs 風**: Sharing the character 風, 風邪 (a cold) and 風 (wind) are pronounced identically as かぜ but mean completely different things — always check the second kanji or the context.
